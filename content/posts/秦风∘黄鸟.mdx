苛政恶俗，人不敢违。哀哉良善，其何以堪。
《读诗臆补》评此诗：恻怆悲号，哀辞之祖


## 🌼 《秦风∘黄鸟》原文：

```
交交黄鸟，止于棘（ki?k）。
谁从穆公？子车奄息（si?k）。
维此奄息（si?k），
百夫之特（d?k）。
临其穴（kyuet），
惴惴其慄（liet）。
彼苍者天（thyen），
歼我良人（njien）！
如可赎兮，人百其身（sjien）！
```

*书香君👉🏻：*

> 首章三韵。“棘、息、息、特”，职部，押韵；“穴、慄”，质部，押韵；“天、人、身”，真部，押韵

```
交交黄鸟，止于桑（sang）。
谁从穆公？子车仲行（hang）。
维此仲行（hang），
百夫之防（biuang）。
临其穴（hyuet），
惴惴其慄（liet）。
彼苍者天（thyen），
歼我良人（njien）！
如可赎兮，人百其身（sjien）！
```

*书香君👉🏻：*

> 二章三韵。“桑、行、行、防”，阳部，押韵；以下五句，两韵，并分别与首章叠句且遥韵

```
交交黄鸟，止于楚（tshia）。
谁从穆公？子车鍼虎（xa）。
维此鍼虎（xa），
百夫之御（ngia）。
临其穴（hyuet），
惴惴其慄（liet）。
彼苍者天（thyen），
歼我良人（njien）！
如可赎兮，人百其身（sjien）！
```

*书香君👉🏻：*

> 尾章三韵。“楚、虎、虎、御”，鱼部，押韵；以下五句分别与首章、二章叠句且遥韵

```
鍼，有些版本作“针”，我以为不妥；慄，有些版本作“栗”，与《毛诗》原本不同。
```

## 🍁️ 骏注：

1、秦人痛哀殉葬三良之诗。秦穆公死，秦人一百七十余人为之殉葬（自杀陪葬或被活埋陪葬），中有秦之良臣“子车氏”（又称“子舆氏”）兄弟三人。秦人哀三良、刺穆公而作此诗。诗旨古今无异议。

2、封建国君，专制一方。生杀予夺，惟意所欲。苛政恶俗，人不敢违。哀哉良善，其何以堪。书香君每读此篇，不寒而栗，悲郁难舒。

3、《毛诗序》：“《黄鸟》，哀三良也。国人刺穆公以人从死，而作是诗也。三良，三善臣也，谓奄息、仲行、鍼虎也。从死，自杀以从死。”

书香君：奄息、仲行、鍼虎，子车氏兄弟三人的名字。都是秦国良臣，为秦穆公辟土开疆、四方征伐出了良计也立了大功，秦穆公很喜欢他们仨，据说临死前指名要他们三人陪葬（有说三人是自愿陪葬的；有说是秦康公逼他们陪葬的）。子车氏，秦国初期一个大夫叫“子车”，他的后人以他名字“子车”为氏。这个“车”，宜读作ju。春秋时，“车”是读如“居”的。车，声如居，言行所居人也。就是说车里是居着人的。

而车、舆，两字又互通。

《广雅.释诂》：“舆，举也。”可推测“舆”字的古读也近似ju。

《潜夫论.相列》：“（木）曲者宜为轮，直者宜为舆”，就是说木材弯的做轮，直的做车厢。舆，就有“车厢”的意思，后来引申为“车”。

4、《郑笺》：“黄鸟止于棘，以求安己也，此棘若不安则移。喻臣之事君亦然。今穆公使臣从死，刺其不得黄鸟止于棘之本意。”意思是，黄鸟尚且可以止于棘、得其所，而国君却令臣子殉葬从死，使臣子不能安己而得其所。

今人程俊英从马瑞辰《毛诗传笺通释》对“止于棘”的解释，恰好与《郑笺》相反，其言：“黄鸟落在棘、桑、楚等小树上是不得其所，以此兴三良的殉葬是不得其死”。书香君不能认同马瑞辰、程俊英的说法。因为，黄鸟是体小且轻的小鸟，这些小树都是可以栖息的，小树枝也能够承受黄鸟的重量（可以止于棘、桑、楚的）。

5、《诗集传》：“秦穆公卒，以子车氏之三子为殉，皆秦之良也。国人哀之，为之赋《黄鸟》。事见《春秋传》，即此诗也。言交交黄鸟，则止于棘矣；谁从穆公？则子车奄息也。盖以所见起兴也。临穴而惴栗，盖生纳之圹中也。三子皆国之良而一旦杀之。若可贸以它人，则人皆愿百其身以易之矣。穆公于此，其罪不可逃矣。”

6、《诗经原始》：“

《黄鸟》，哀三良也。此诗事见《左传》，凿凿有据，自不必言。或以三良从死，命出穆公，或以为康公迫死，或又以为秦俗如此，非关君之贤否。总之……封建固良法，封建亦虐政，秦、汉后竟不能复，虽曰时势，亦人心为之也。圣人存此，岂独为三良悼乎？亦将作万世戒耳！”

7、书香君：方玉润“将为万世戒”的感叹，人同此心。三良殉葬，有说是穆公遗命，有说是穆公之子康公强迫的。据说殉葬之俗源于戎狄，而秦国居然拿过来用，还习以为常了。《史记》有提到：秦武公死，陪葬者六十六人；秦穆公死，包括子车氏三兄弟在内，有一百七十七人陪葬；秦始皇死，更登峰造极，后宫皆令从死、陵墓工匠活闭于墓中。惨绝人寰、无耻狠恶至极！

8、本诗在写作上运用了双关语和夸张手法。比如：棘，暗指紧“急”；桑，暗指“丧”；楚，暗指“痛楚”。而“人百其身”则是夸张手法，正如后世《离骚》之“虽九死其犹未悔。”有人认为，《离骚》此句即来自于对“人百其身”的化用。

9、交交，鸟鸣声。

《毛传》：“交交，小貌。”

马瑞辰《毛诗传笺通释》：“交交”，通作“咬咬”，谓鸟声也。古歌有“黄鸟鸣相追，咬咬弄好音”。

《玉篇》、《广韵》皆释：咬，鸟声。

《毛诗》作“交交”，应为“咬咬”的省借字。

朱熹和方玉润都将“交交”释为“飞而往来之貌”，书香君认为不妥，我猜也许是他们错误地理解了《毛传》，因为《毛传》在“交交，小貌”之后，紧接着有一句“黄鸟以时往来得其所，人以寿命终，亦得其所。”

我愿意把“交交”理解成“小而轻巧”或“鸣声柔婉”。即“小貌”或“鸟声”。

10、黄鸟，即黄雀。

11、棘，酸枣树。棘，言“急”也，是双关语。

马瑞辰《毛诗传笺通释》：“棘之言急也，桑之言丧也，楚之言痛楚也。”

12、从，从死，即殉葬。

13、穆公，秦穆公。春秋时秦国国君，姓嬴，名任好。

14、子车，复姓，又称子舆。奄息，“奄”乃其字，“息”是其名。下文仲行、鍼虎同此，都是前字后名、姓子车。这三人为兄弟，都是当时秦国有名的贤臣。

《毛传》：“子车，氏。奄息，名。”

《孔疏》：《左传》作子舆。舆、车，字异义同。

15、特，匹配。

《毛传》：“特，匹也”。匹之言敌也，当也。

“百夫之特”犹言“乃当百夫之德”。

《郑笺》：“百夫之中最雄俊也。”

《诗集传》：“特，杰出之称。”

16、“临其穴，惴惴其慄”，

《毛传》：“惴惴，惧也。”

《郑笺》：“穴，谓塚圹中也。秦人哀伤此奄息之死，临视其圹，皆为之悼慄。”

书香君：圹，读音kuang去声，意思是墓穴、坟茔。

《诗集传》：“临穴而惴慄，盖生纳之于圹中也。”（生纳，指把活人放到墓穴中陪葬。真正天良丧尽、惨不忍睹）

17、“彼苍者天”，哀恸至极的呼泣，犹今语“那青天啊”、“苍天啊”、“我的老天爷呀”等等。

《郑笺》：“言彼苍者天，愬之。”

书香君：愬，两个读音shuo或su，都是去声。字义分别是“恐惧貌”、“诉告/控诉/诉说”。我觉得这两种意思都可以有。

18、歼，灭、灭尽。因三兄弟同死，故曰“歼”。

《毛传》：“歼，尽。”

19、良人，好人、善人贤者。有些人认为“良人”就只是妇人对自己丈夫的称谓（比如《绸缪》里正是），其实不然。古时妇人称丈夫“良人、夫子、君子、伯”都有，是美称，而非专称。

20、“人百其身”，情愿自己死一百次来换三良的命。这是一种夸张说法，极言痛惜三良之死。

有些版本解为“用一百人赎其一命”，书香君认为是错误的。作者最多表达自己愿意以死换三良的强烈愿望，怎能拿其他人的命（还是一百条人命！若是虚指，那也是很多条人命）换自己敬重的人？屈子“虽九死其犹未悔”也是说自己即使死很多次也不悔，跟别人的性命无关。

《郑笺》：“百其身，谓一身百死犹为之，惜善人之甚。”

《诗集传》：“若可贸以它人，则人皆愿百其身以易之矣。”

21、防，比，当，相当。

《郑笺》：“防，犹当也。言此一人当百夫。”

《诗集传》：“防，当也。言一人可当百夫也。”

22、御，与上文“防”义近。当，比。

《毛传》：“御，当也”

《诗集传》：“御，犹当也。”

23、清陈继揆《读诗臆补》评此诗：恻怆悲号，哀辞之祖。
