钱钟书《管锥编》曾评价《子衿》，称其开启了后世文学言情心理描绘的先河。

《毛传》释“青衿”为“学子之所服”，似乎是最早的关于“青衿”是周朝学子服的说法，也可能是误读。但不管是不是误读，反正后来的读书人衣襟就多为青色了。从北齐、北周到隋唐乃至两宋，学子皆着青色交领的深色衣。“青衿”也就代指学子，特别是年轻学子了。

## 🌼 《郑风∘子衿》原文：
```

青青子衿（kiəm），
悠悠我心（siəm）。
纵我不往，子宁不嗣音（iəm）？

```
*书香君👉🏻：* 

> “衿、心、音”，侵部，押韵

```
青青子佩（buə），
悠悠我思（siə）。
纵我不往，子宁不来（le）？

```
*书香君👉🏻：* 

> “佩、思、来”，之部，押韵

```
挑兮达（that）兮，
在城阙（khiuat）兮。
一日不见，如三月（ngiuat）兮！

```
*书香君👉🏻：* 

> “达、阙、月”，月部，押韵

```
本诗三章，每章一韵。

```
## 🍁️ 骏注：

1、对于普通读者来说，《郑风》里最脍炙人口的篇章首推《子衿》应该没有疑问，即便在《国风》160篇甚至整个《诗经》的305篇中，《子衿》的流行度或受众熟悉度大概率可以排进前三十。

本诗，今人多解为女子对所爱之人唱的恋歌。

后世“青衿”一词即源出于首句“青青子衿”；本诗且与《采葛》一起，同为美句“一日不见如三月”的发端。

2、“子”，对男子的美称、雅称、敬称。至于后世将本诗中的“子”解为“学子”，将“青衿”解为读书人的代指，源于《毛诗序》，但，这是一个美丽的“误会”。后面再细说。

3、“衿”这个字，有名词和动词两种词性。

名词用法很多。
“衿”的字面是“衣”和“今”的组合。“今”这个字甲骨文里就有了，殷商时期，“今”的字义逐步演变成“此时”、“当日”或更宽泛的“现时”。
（《说文》：“今，是时也。”
《苍颉篇》：“今，时辞也。”
就是说，“今”是表达时间概念的词。——书香君注）

“衣”、“今”组合的“衿”就是“现时、当面的服装”，表“正装”。因为正装和便装的区别主要在“衣领”，也由于“古者斜领下连于衿 ”，故，“衿”就演变成“正装交领”（谓“领”为“衿”），这是之后“衿”的所有字义的基础来源。了解了这个，“衿”的具体字义就可在阅读古文时自己意会。
作名词用时，有：

a、“衣领”（如“青衿”）、“衣襟”（如《庄子.让王》有“正冠而缨绝，捉衿而肘见，纳履而踵决”句）；

b、衣的前幅（衣下两旁掩裳际处）；

c、胸怀（同“襟”，如“衿抱”）；

d、衣（领）上的系带等几种字义。

作动词用时，就是“结带”、“以带束之”、“系”、“穿（如“衿甲”）”等字义。

4、再说“青衿”。《毛传》云：“青衿，青领也，学子之所服。”毛公这一解读似乎是最早的关于“青衿”是周朝学子服的说法，但也几乎是孤证（我查文献没有其他例证，在此也求教方家）。但不管是不是一个美丽的误读，反正后来的读书人衣襟就为青色了，从北齐、北周到隋唐乃至两宋，学子皆着青色交领的深色衣。“青衿”也就代指学子、读书人。
明清两朝“衿士”、“青衿”或“衿”都可代指“秀才”或泛指“读书人”，如“绅衿”。当然，更广义地，“青衿”也可以指年轻人。古时按礼，父母健在，则“衣纯以青”。

旧说跟今天的理解不同。我们来看看历代解读。

《毛诗序》：“《子衿》，刺学校废也。乱世则学校不修焉。郑国谓学为校，言可以校正道艺。青衿，青领也，学子之所服。”

*书香君👉🏻：* 

> 有意思喔，毛序说，郑国把“学”称作“校”，可以校正道艺。“校”是“校正”的意思，那就要读作jiao去声。“学校”的意思并不是现在所谓的“学习的地方、场所”，而是“学习（知识）并以之校正道艺”。并且，把“青衿”解为“学子之所服”。

毛序说本诗是“刺学校不修而废”的。今天能看出来吗？匪夷所思哎。

《郑笺》：“学子而俱在学校之中，己留彼去，故随而思之耳。君子之学，以文会友，以友辅仁，读学而无友，则孤陋而寡闻，故思之甚。”

*书香君👉🏻：* 

> 《郑笺》的大意是，“我”留在学校而别人（同学）离开了，所以我想念他（他们），一个人独自学习而没有文友、学友可交流，必然孤陋寡闻，所以“我”殷切思念学友。

我们今天来看序笺的解读，是不是觉得太会发挥、太过附会了？但是，自唐至宋元明，主流解读却皆承序笺，奇怪不？

还是朱熹，他弃序笺之附会，看出是男女词。
其《诗集传》曰：“子，男子也。衿，领也。悠悠，思之长也。我，女子自我也。嗣音，继续其声问也。此亦淫奔之诗。”

*书香君👉🏻：* 

> 朱熹的解读是生动活泼的。但他习惯性地把爱情诗看成“淫奔词”，戴着卫道士、理学家的有色眼镜。

姚际恒认为本诗“刺学校无据，疑亦思友之诗。”

方玉润认为，毛序说得不算错，只不过并不是“刺学校”的，是“老师感伤学校久废不修，学者散处四方”而作的诗。方氏从“纵我不往，子宁不来”这句看出是“师之于弟子”的口气（有意思，怎么看出来的呢？我是没看出——书香君注）。其《诗经原始》曰：“《子衿》，伤学校废也。（其师）曰：学问之道未可孤陋自安也，今学校废久矣，予不能再赴讲席而广教，思彼青青子衿者，相从有素，能无系于心哉？”

*书香君👉🏻：* 

> 我们今天来品读“一日不见如三月”，这般浓烈、刻骨，恐怕不大可能是老师想念学生那种情感。所以，我认为“男女相悦之词”，也即“爱情诗”，是本诗最合适的解读。

5、“悠悠”，忧思绵绵的样子。《集传》：“悠悠，思之长也”。

6、“嗣音”，传送音讯。“嗣”在这里不读si，要读yi阳平，通“诒”或“贻”，是“寄”或“给”的意思。韩诗本中，“嗣”就作“诒”。“不嗣音”就是“不给我音讯”，“不把音讯寄送、传送给我”。

《毛传》释：“嗣，习也。”将“嗣音”释为“教以诗乐、诵之歌之、弦之舞之。”与今天的解释不同。

《郑笺》释：“嗣，续也。”释“嗣音”为“传声问我”。朱熹亦持此解，与今天的解释相近。
7、“佩”，佩玉，古时男子常佩玉在身。此处引申或代指“系佩玉的带子”、“佩绳”。以与上文的“衿”相对应。

8、“挑兮达兮”，就是“挑达兮”。“挑兮” 走来走去，往来貌。就是“溜达”，等人时的来回踱步（暗含焦急）。“挑达”，读音为 “tao（阳平） ta（去声）”。今天的潮州话，“出去逛街、出去溜达”之类，还说成“tao ta”呢。

9、“城阙”，城门两边的关楼、观楼，也是彼时男女约会之所。古代，城墙大门两边筑台，台上设楼，称“观”、“阙”。

闻一多先生《风诗类钞》：“城阙，是青年们常幽会的地方”。

10、钱钟书《管锥编》曾评价《子衿》，称其开启了后世文学言情心理描绘的先河。

11、曹操著名的《短歌行》也直接引用了《子衿》和《鹿鸣》的句子：
“青青子衿，悠悠我心。
但为君故，沉吟至今。
呦呦鹿鸣，食野之苹。
我有嘉宾，鼓瑟吹笙。”

当然，曹操是求贤而不是谈恋爱。他诗尾还用了“周公吐哺”的典故（周公旦“一食而三吐哺，一沐而三握发”）。《诗经》句子和典故信手拈来用得自然贴切，堪称完美，真的要佩服曹操。
