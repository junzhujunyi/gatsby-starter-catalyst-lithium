成语“孔武有力” “舍命不渝”或出于此诗。

## 🌼 《郑风∘羔裘】原文：
```

羔裘如濡（njio），
洵直且侯（ho）。
彼其之子，舍命不渝（jio）。

```
*书香君👉🏻：* 

> “濡、侯、渝”，侯部，押韵

```
羔裘豹饰（sjiək），
孔武有力（liək）。
彼其之子，邦之司直（diək）。

```
*书香君👉🏻：* 

> “饰、力、直”，职部，押韵

```
羔裘晏（ean）兮，
三英粲（tsan）兮。
彼其之子，邦之彦（ngian）兮。

```
*书香君👉🏻：* 

> “晏、粲、彦”，元部，押韵

```
本诗三章，每章一韵。末章各句的句尾语助词“兮”不入韵。

```
## 🍁️ 骏注：

1、本诗是《诗经》三篇同名诗《羔裘》之一，另外两篇《羔裘》分别收在《唐风》和《桧风》中，以后会按顺序分享。

本篇，旧说有认为是刺诗的，通过赞美古之君子以讽刺郑国无贤臣，如毛传郑笺。也有认为是赞美诗的，如方玉润《诗经原始》题解就是“美郑大夫也”。我们从字面并看不出有什么嘲讽意味。当代多当做赞美贤者仪容和品行的诗。

2、《毛诗序》：“《羔裘》，刺朝也。言古之君子，以风其朝也。”

3、《郑笺》解释《毛序》中“言古之君子”，就是“称道古之君子”——“言犹道也。郑自庄公而贤者陵迟，朝无忠正之臣，故刺之。淄衣羔裘，诸侯之朝服也。言古朝廷之臣，皆忠直且君也。君者，言其正衣冠，尊其瞻视，俨然人望而畏之。守死善道，见危授命。”

4、朱熹承认是赞美大夫的诗，但不知具体赞美何人。其《诗集传》云：“羔裘润泽，毛顺而美，豹甚武而有力，故服其所饰之裘者如之，当生死之际，能以身居其所受之礼，而不可夺。盖美其大夫之词，然不知其所指矣。”

5、方玉润认为刺说不成立。他认为郑国当时并非无贤臣良士，且引《辩说》言：当时郑之大夫，如子皮、子产之徒，岂无可以当此诗者？方玉润认为此诗不仅是赞美诗，而且赞美的不是某一个人而是郑国当时的一批贤臣。所谓“盈廷硕彦济美一时，或则顺命以持躬，或则忠鲠以事上，或则儒雅以声称，皆能正己以正人。故诗人即其服饰之盛，以想其德谊经济文章之美，而咏叹之如此。舍命不渝者，君子安命虽临利害而不变；邦之司直者，大臣刚毅有力，独能主持国是而不摇；邦之彦兮者，学士文采高标，足以黼黻猷为，而极一时之选也。有此数臣，国势难孱，故可特立晋楚大国之间而不致败，此郑之所以为郑也。

6、本诗《左传》也有提到——“郑六卿饯韩宣子于郊，子产赋郑之《羔裘》。宣子曰：起不堪也。”说的是郑国六卿在城郊为韩宣子（韩起）饯行，席间子产吟诵《羔裘》，韩宣子称谢，说我韩起实不敢当啊。韩宣子，就是韩起，春秋时晋国六卿之一。三家（韩赵魏）分晋之“韩”氏就是他家族的后代。有记载，那次饯别，当时送他的郑国六卿分别吟诵了《野有蔓草》、《羔裘》、《褰裳》、《风雨》、《有女同车》和《萚兮》。

这也从侧面说明了，《羔裘》那时就被当做赞美贤臣的诗歌了。

7、成语“孔武有力”、“舍命不渝”就出于此诗。

8、羔裘，羊皮大衣，古大夫的朝服。

9、濡，柔润，湿润，柔而有光泽。“相濡以沫”之“濡”也是此意。

10、洵，读音xún，信，诚然，的确。

11、侯，美。“侯”的这个词意“美”，就取自对诸侯的称赞 。姚际恒释“侯”：侯，即诸侯之侯，当时称诸侯亦取美意也”。

12、渝，改变。

13、豹饰，用豹皮装饰皮袄的袖口。是卿大夫的官服装饰。饰，“缘袖也”，就是羔裘的袖口滚一圈豹皮。古时按礼，“君用纯物，臣下之”。什么意思呢？就是说，国君穿的是纯羊皮的裘，大夫的羔裘朝服则降低一个等级，不是纯羊皮的，是杂以豹皮为装饰的。

14、孔，表程度的副词。甚、很。

15、司直，官名。古时负责矫正人民过失的官员，一说劝谏君主过失的官员。估计类似于现在的检察院或惩戒署官员。

16、晏，鲜艳、鲜盛貌。

17、三英，羔裘上三条装饰的缨丝饰条，装饰袖口的三道豹皮镶边。类似现在制服上的装饰标志。旧说如《郑笺》，则释“三英”为“三德”：刚克、柔克、正直。

18、粲，色彩艳丽。旧说释为“多，众意”。
“三英粲兮”，旧释：充溢着美好的品德，即今之“满满的正能量”之谓。

19、彦，美男子，对男子的美称。也代指才德出众之人，如今之“才俊、俊杰”

《毛传》：“彦，士之美称”。

闻一多则解为：“彦”，古通“宪”，法则、仪表。可备一说。

> 羊皮大衣光泽又柔润，他的为人善良又刚正！
> 
> 宁死也不愿苟且偷生，他就是这样美好的人！

> 羊皮大衣袖口饰豹皮，
> 
> 他是那么的孔武有力。
> 
> 百姓的案讼交他断论，他就是执法公道的人！

> 羊皮大衣光洁又鲜艳，三条锦丝灿烂真耀眼。
> 
> 治理国家要靠栋梁才，他就是这样贤良的人！
