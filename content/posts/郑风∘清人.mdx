《左传》记载：郑人恶高克，使帅师次于河上，久而弗召。师溃而归，高克奔陈。郑人为之赋《清人》。

有评曰：诗人之意微婉如此！

## 🌼 《郑风∘清人》原文：
```

清人在彭（beang），
驷介旁旁（peang）。
二矛重英（yang），
河上乎翱翔（ziang）。

```

> “彭、旁、英、翔”，阳部，押韵

```
清人在消（sio），
驷介麃麃（pio）。
二矛重乔（gio），
河上乎逍遥（jio）。

```
*书香君👉🏻：* 

> “消、麃、乔、遥”，宵部，押韵

```
清人在轴（diuk），
驷介陶陶（du）。
左旋右抽（thiu），
中军作好（xu）。

```
*书香君👉🏻：* 

> “轴、陶、抽、好”，觉幽通韵

```
本诗三章，每章一韵。

```
## 🍁️ 骏注：

1、旧说是讽刺郑国将领高克、进而讽刺郑文公的诗作，作者是公子素（一说乃公孙素之误，一说乃公子士）。用的是反衬的手法，一边写战马强壮武器精良，一边写军士闲散嬉戏逍遥。当代也有人认为，从字面看就是描写驻军训练场景的诗作。当然从尾句“中军作好”可以看出讽刺意味。

2、《毛诗序》：“《清人》，刺文公也。高克好利，而不顾其君，文公恶而欲远之，不能，使高克将兵，而御狄于竟，陈其师旅，翱翔河上，久而不召，众散而归，高克奔陈。公子素，恶高克进之不以礼，文公退之不以道，危国亡师之本，故作是诗也。”

毛序此说有根据——《左传》记载：郑人恶高克，使帅师次于河上，久而弗召。师溃而归，高克奔陈。郑人为之赋《清人》。

《诗集传》直接说是郑文公厌恶高克：“郑文公恶高克，使将清邑之兵御狄于河上。久而不召，师散而归，郑人为之赋此诗。言其师出之久，无事而不得归，但相与游戏如此，其势必至于溃败而后已尔。”

*书香君👉🏻：* 

> 高客，郑国将领。恶高克者，左传谓“郑人”，曲笔也。普通郑人能下令高克率军驻扎在边境河岸（也可能是越境屯兵，郑在卫之河对岸，狄侵卫，郑越河在卫边境戍兵）？恶高克者，郑文公也，朱子之说是也。

这就好玩了，一国之君因为讨厌一个本国将领，用这么一个孩子气的玩家家手段，调这个将领去戍边，又久不征用，拖得这只军队无所事事、军纪涣散、玩乐遨游，操练时不从实战出发只是装样子作秀，最终溃散回国，而主将逃到陈国。国君这种因个人情绪的好恶置国家利益于不顾的荒唐任性做法真的匪夷所思。所以，著有《春秋传》的宋人胡安国（胡文定公）说：“人君擅一国之名宠，生杀予夺，惟我所制尔。使高克不臣之罪已著，按而诛之可也；情状未明，黜而退之可也；爱惜其才，以礼驭之，亦可也。乌可假以兵权，委诸竟上，坐视其离散，而莫之恤乎？《春秋》书曰：郑弃其师。其责之深矣。”朱熹评本诗时也引用了胡安国的这番话。胡文定公所说的“春秋书曰郑弃其师”，此事见于《春秋.闵公二年》：“冬，十有二月 ，狄入卫，郑弃其师。”

3、所以方玉润解此诗曰：“刺郑文公弃其师也”。《诗经原始》：“高克弃师奔陈，郑人为赋此诗，事有明文，固勿庸议。即彭、消、轴，或以为地名，或以为非地名，皆不可考。惟郑文公恶高克，而使之拥兵在外，此招乱之本也。幸而师散将逃，国得无恙；使其反戈相向，何以御之？由斯以观，高克亦无能辈耳，何以见恶于文公耶？诗曰翱翔，曰逍遥，曰左旋右抽、中军作好，所谓霸上诸军直同儿戏，即使作乱亦易制服。诗人固早有以知其必不然也。若文公者则不能无所议焉，故刺之。”

方氏解诗，说得很到位了。

4、前已述及，本诗末句点出了军中闲散嬉戏的情状，虽未直接嘲讽但咀嚼之下自含刺意。所以姚际恒《诗经通论》评此篇有言：“诗人之意微婉如此”。

5、清人，指郑国大臣高克带领的清邑的士兵。清，郑国之邑。据王先谦《诗三家义集疏》说，高克也是清邑人，《易林》即有“清人高子”语。有些版本将“清邑”说成是卫国地名，恐不妥。

6、彭，郑国地名，在黄河边。一说“彭”乃“清”之误写。

7、驷介，一车驾四匹披挂铠甲的马。介，甲。

8、旁旁，同“彭彭”，马强壮有力貌。一说行走、奔跑貌。

9、二矛，酋矛、夷矛。插在车子两边。

10、重（chóng）英，以朱羽为矛饰，二矛树车上，遥遥相对，重叠相见。重，重叠。英，矛上的缨饰。一说二矛靠在一起，其缨重绕。

11、翱翔，游戏貌，与今意不同。

12、消，黄河边上的郑国地名。

13、麃（biāo）麃，英勇威武貌。有些版本作“镳镳”。

14、乔，“鷮（jiāo）”的借字。长尾野鸡，此指矛上装饰的野鸡羽毛。

15、逍遥，闲散游逛貌。

16、轴，黄河边上的郑国地名。

彭，消，轴，到底是郑国还是卫国地名，各种版本说法不一，现在也没有准确考证。也许是郑国边境的三个地名，也许是卫国境内靠近黄河岸、靠近郑国的三个地方，也许只是虚指，代指抗狄（防狄）的边境，换字也是为了押韵。郑、卫两国隔着黄河相望，那么，防守戎狄（其时赤狄侵扰甚至压境卫国），可以在两国的边境，三个地方都是边境，这一点是确实的。

17、陶陶，和乐貌。一说疾驰貌。

18、左旋右抽，古时战车标配是驾车者在车左，负责执辔御马；将居鼓下。“左旋右抽”，就是指高克令御手和兵士们练习旋车之技和从车右侧抽刀之法，这种练习可能对实战无宜，而只是观赏起来好看，属于我们今天所说的“花架子”。我们也看到过电视报道，军人打靶，靶子倒下后就有一行标语口号冒出来；还有军被叠得象刀切。就不知道真打起仗来咋样。

我们要知道，高克的部队为啥散漫、嬉戏？根子在郑文公啊。

旋，转车。抽，拔刀。
“左旋右抽”，左边的御者旋转车子，右边的军士拔刀操练。一说：身体左转、右手拔刀。结合上下文，我觉得分解成左边的御手、右边的刀剑手更好些。

19、中军，古时军队一般分成三军（左中右，或前中后）。一说即“军中”。

20、作好，《毛传》释：“居军中为容好”。拿今天的话说，就是看上去很好看但没什么实用价值。指表面工作、装样子，不是实战训练。与今之“作秀”、“花架子”意近。
