傍晚怀人，
真情真境，
描写如画。

晋唐人田家诸诗，
恐无此真实自然。

鸡栖于桀下牛羊，
饥渴萦怀对夕阳。
已启唐人闺怨句，
最难消遣是昏黄。

## 🌼 《君子于役》原文：
```

君子于役，不知其期（giə）。
曷其至哉（tzə）？
鸡栖于埘（zjiə）。
日之夕矣，羊牛下来（lə）。
君子于役，如之何勿思（siə）！

```
*书香君👉🏻：* 

> “期、哉、埘、来、思”，之部，押韵

```
君子于役，不日不月（ngiuat）。
曷其有佸（kuat）？
鸡栖于桀（giat）。
日之夕矣，羊牛下括（kuat）。
君子于役，苟无饥渴（khat）。

```
*书香君👉🏻：* 

> “月、佸、桀、括、渴”，月部，押韵

```
本诗两章，每章一韵。

首章第二句“曷其至哉”，有些版本作“曷至哉”，少一个“其”字，不影响诗意。

又，二章尾句最后的句号有些版本用的是问号。

再又，有些版本把每章分为八句，即把“君子于役，不知其期（不日不月）”和“日之夕矣，羊牛下来（羊牛下括）”各分为两句。我本人倾向于每章分为五句。

```
## 🍁️ 骏注：

1、“王风”的“王”，指洛邑的王城，有说就指洛邑（今洛阳一带）。有说洛邑地域内另有一个更小的“王城”，有考古证据支持。我们不是专门的研究者，只是《诗经》的欣赏者，有大致了解即可。洛邑，事实上在成王、周公时期平叛三监后就开始建设，当时只作为周的宗庙区（西周都于丰镐，即丰京和镐京），平王东迁才都于洛邑。我们看文献中，洛邑也曾称“土中”、“洛师”、“东国洛”。《王风》就是洛邑一带周王所直接掌控地区的“风”，相当于京畿之地的民歌民谣。《王风》十篇，多悲怨之作，我们昨天分享过的《黍离》就是其中的杰出代表。李白也说：“王风何怨怒”。有南季北孔之称的大儒季札评《王风》曰：“美哉，思而不惧”。季札是孔子推崇的人，我在《浅谈诗经之古远丰美》中，详细介绍过他对风雅颂的评点（季札听诗），此文就不细说了。

本篇是一首妻子思念远役中的丈夫的诗作。

来看看旧说。

《毛诗序》：“《君子于役》，刺平王也。君子行役无期度，大夫思其危难以风焉。”

毛序以为是“大夫思君子”。
现在看就太穿凿附会了。

其实，汉代对本诗就有了“怨女旷夫”的解读，班彪在西汉末年的动乱中离长安至天水避难，行程中写下著名的《北征赋》，其中就有这样的句子：

日晻晻其将暮兮，
睹牛羊之下来。
寤旷怨之伤情兮，
哀诗人之叹时。

应该是化用了《君子于役》，并且当做妇人思夫之作的。

朱熹也明确说是妇人思夫。解“君子”为“妇人目其夫之辞”。其《诗集传》谓：“大夫久役于外，其室家思而赋之。”
这是更贴近生活的解读。

方玉润《诗经原始》：“《君子于役》，妇人思夫远行无定也。此诗言情写景，可谓真实朴至。夫妇远离，怀思不已，用情而得其正。”

方氏眉评盛赞此诗“傍晚怀人，真情真境，描写如画。晋、唐人田家诸诗，恐无此真实自然。”

2、君子，妻子对丈夫的尊称、敬称、美称。一如《伯兮》中的“伯”。

3、于役，服役中。役，兵役或徭役。

4、曷，读音he阳平。通“何”。“曷其至哉”这句，有些版本作“曷至哉”，没有“其”这个字，诗意都不影响。但是“曷其至哉”与下文的“曷其有佸”字数相同，有“其”字更对称些。我倾向于采用“曷其至哉”，这句意思是“何时回到（家）呀”。

5、埘，读音shi阳平。鸡舍。古时墙壁上掏洞做鸡舍，就是“埘”。下文的“桀”，意思是“横木”，供鸡夜栖，也是简易鸡舍。

6、“羊牛下来”，这个“下来”，很多版本就译成今天的“下来”，有点轻率了。《诗经》时代何时这样用词？“下”就已经是“下来”、“下山”了，为啥还要再加一个“来”字？这有点小看《诗经》了。事实上，“来”古读如 li 阳平，我认为或可
通“离”或“罹”，可解为“藩篱”、“篱笆”，引申为牛圈、羊圈。这跟前文的“埘”、“桀”就相互呼应了。

7、“如之何勿思”，就是“如何不思”？

8、不日不月，就是“不知哪月哪日”，不知具体归期也。有说言其时间长，我个人不太认同。丈夫哪怕较长时间才回来也不算最难熬，只要有准信。因为毕竟有盼头，知道一个时间段后就能见面（居家14天就能解除隔离的感觉有没有？）。而“不知归期”就很要命了，无有定期、没有指望，最是难熬。

9、“曷其有佸”，这句，“有”读去声，就是“又”。比如《出师表》“尔来三十有一年矣”，这句中“有”就读去声，读上声是不妥的。“佸”，聚会、相聚、相会。整句意思就是“什么时候又能聚在一起”，指丈夫服役完毕回家相聚。

10、羊牛下括。括，集束，喻牛羊的圈栏。一说通“佸”，就是牛羊聚集在一起。我采前说。顺便说一下，《诗经》时代，叙述事物，一般地，小者在前，大者在后，如“羊牛下括”。羊在牛前，不似今日习惯写成“牛羊”。又如，“燕燕于飞，下上其音”（《燕燕》）也是“下”在前而“上”在后，不似今日习惯写成“上下”。

11、苟无饥渴，这句表面是推测，事实上是祈使。苟，或许，表达的却是“但愿”；“苟无”，就是“但愿没有”。不能简单理解成“假如没有、或许没有”。饱含着担忧，同时祈祷这种担忧不会成真，“希望没有、但愿没有”。正如《二子乘舟》“不瑕有害”句（可参我2021年1月27号朋友圈）。饥渴，除字面意思之外，暗喻“情事饥渴”。壮年的丈夫在外服役，妻子除了担心他日常饮食温饱之外，也心疼丈夫煎熬着单身。这是本诗令人心动感慨处之一。

12、日复一日，日之升兮、日之夕矣，鸡栖了，牛羊也入圈了而丈夫不得归且无归期，那份惆怅哀婉，又如之何？！

13、本诗在文学上开创了“日暮怀人”的典型环境，后世文人骚客借鉴之下，创作出很多“读来柔肠寸断、思之刻骨铭心”的作品，正所谓“鸡栖于桀下牛羊，饥渴萦怀对夕阳。已启唐人闺怨句，最难消遣是昏黄。”

骏译：（我尝试五言译之）

丈夫在服役，
遥遥无归期。
鸡儿暮时栖 ，
羊牛宿圈篱。
郎君何时返？
日头又落西。
君已远行久，
妻也忧思密。

丈夫去服役，
归家未有期。
鸡儿上架栖，
牛羊下山聚。
何日再团圆？
太阳又渐西。
郎君在役中，
宁无腹中饥？
