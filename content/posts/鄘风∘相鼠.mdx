开口直骂、骂得还狠。

意在笔先、几近裂眦。
嫌于虐且俚。

这样一篇诗作，为何官家能把它收入《诗三百》？

孔子曰：为后世留的。

## 🌼 《鄘风∘相鼠》原文：
```

相鼠有皮（biai），
人而无仪（ngiai）。
人而无仪（ngiai），
不死何为（hiuai）？

```
*书香君👉🏻：* 

> “皮、仪、仪、为”，歌部，押韵

```
相鼠有齿（thjiə），
人而无止（tjiə）。
人而无止（tjiə），
不死何俟（zhiə）？

```
*书香君👉🏻：* 

> “齿、止、止、俟”，之部，押韵

```
相鼠有体（thyei），
人而无礼（lyei）。
人而无礼（lyei），
胡不遄死（siei）？

```
*书香君👉🏻：* 

> “体、礼、礼、死”，脂部，押韵

```
本诗三章，每章一韵。
每章二三句仅换一字，几成重文，叠进诗旨。

```
## 🍁️ 骏注：

1、《相鼠》应该是《诗经》里骂人最直接最尖锐的一首。历代儒生评价其“嫌于虐且俚矣！”但此诗所骂何人？旧说大致有二：《毛诗序》（“刺无礼也”）、《郑笺》认为是刺在位者无礼仪的。《鲁诗》则认为是妻谏夫的（“此妻谏夫之诗也”），班固《白虎通义》承鲁诗说，且一众大家如明进士何楷、清学者魏源都在此说基础上再度发挥，但后世多数学者认为诗中语气严厉至极，绝不类妻谏夫口吻。

我们看看各家如何说。

《毛诗序》：“《相鼠》，刺无礼也。卫文公能正其群臣，而刺在位承先君之化，无礼仪也。相，视也。无礼仪者，虽居尊位，犹为暗昧之行。”

《郑笺》：“仪，威仪也。视鼠有皮，虽处高显之处，偷食苟得，不知廉耻，亦与人无威仪者同。人以有威仪为贵，今反无之，伤化败俗，不如其死，无所害也。”

《诗集传》：“言视彼鼠，而犹必有皮，可以人而无仪乎？人而无仪，则其不死亦何为哉！”

书香君以为方玉润之解最为出色。其《诗经原始》言：“序谓刺无礼，诸家皆然。以人之仪喻鼠之皮，则未免轻视礼仪。夫麟凤尚有威仪，龙马必多精神，人之所以异于禽兽者，礼义以制心，威仪以饬躬也。倘去此威仪礼义而不之检，则是卑污贱恶不过如鼠之徒有其皮与齿，以成其体而已矣。虽欲求为禽兽之长而不可得，况人乎？夫人也而禽兽之不若，则何以自立于天地之间？固不如速死之为愈耳。若此解诗，语意方能圆到。或又谓此卫文公训诰臣民之辞，亦颇近是。”

2、《诗经》中关于“鼠”的篇章尚有《召南.行露》、《魏风.硕鼠》等。似乎自古以来，鼠之形象就和丑陋、狡黠、偷窃、毁物联在一起。本诗斥骂当权者之丑恶超过人人厌恶的老鼠，疾呼为何还不赶快去死！有力量、很过瘾！至于所刺的“在位者”具体是谁，所刺具体何事，我们现在已无法考证。不过，好在有史料记录了卫国在位者的诸多不堪事，如州吁弑兄自立为君、宣公强娶子妇（宣姜）、宣公谋杀亲子（太子伋）、惠公与黔牟兄弟相杀、懿公好鹤淫乐终致国灭，等等。父子反目，兄弟相残，父霸子妇，子淫父妾，桩桩件件无耻至极。骂他们禽兽不如，咒他们赶快去死，似是卫人普遍心愿，《相鼠》一篇只不过喊出国人心底的呼声。

3、此篇三章叠唱互济，构成完整诗意。以鼠为兴，用“无仪”、“无止”、“无礼”，指斥坏人的外表、内心和行为都很丑陋。末句反诘进逼，既加强了讽刺的力度又增强了诗文的妙趣。

4、相，看。《毛传》：“相，视也”。
鼠，根据陆佃（陆游祖父，号陶山，著有《陶山集》、《埤雅》、《春秋后传》，曾师王安石）的说法，有一种鼠见到人时，会站立以两只前脚作揖，谓礼鼠、拱鼠。故，韩愈有句曰：“礼鼠拱而立”。

5、仪，威仪。可畏为威，可象为仪。一说礼仪。

6、何为，倒文（为了押韵），即今之倒装。为何，为什么。和，鲁诗作“胡”

7、止，容止、节止、守礼法。无止，就是无礼节。一说是“耻”的借字。

8、俟，等。“何俟”，“俟何”，倒装。何，三家诗作“胡”。

9、体，身体、肢体。

10、礼，礼仪。此处指知礼仪，或指有教养。

11、胡，何，为何，为什么，怎么。胡，三家诗作“何”。很好玩，三家诗和毛诗，胡、何，是互换的。

12、遄，读音chuán，快，速速，赶快。

13、请教孔老夫子：“这样一篇意在笔先、几近裂眦、嫌于虐且俚的诗作，为何官家能把它收入《诗三百》呢？”

孔子曰：为后世留的。

## 🍁️ 骏译：

> 你看老鼠都有皮，
> 
> 做人哪能没威仪。
> 
> 假如做人没威仪，
> 
> 你说为啥不去死？

> 老鼠个个有牙齿，
> 
> 做人反倒无节止。
> 
> 假若做人不知耻，
> 
> 还等什么不去死？

> 看那老鼠有肢体，
> 
> 为人反而不守礼。
> 
> 人若处事不守礼，
> 
> 为何还不赶快死？
