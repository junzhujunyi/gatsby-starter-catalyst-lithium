## 🌼 《关雎》原文：
```

关关雎鸠（kiu），
在河之洲（tjiu）。
窈窕淑女，君子好逑（qiu）。

```
*书香君👉🏻：* 

> 首章一韵。鸠、洲、逑，幽部押韵

```
参差荇菜，左右流（liu）之；
窈窕淑女，寤寐求（giu）之。
求之不得（tək），
寤寐思服（biuək）。
悠哉悠哉，辗转反侧（tzhiək）。

```
*书香君👉🏻：* 

> 本章两韵。虚词不入韵。但有些字表面看是虚词但取其所通假的实词之意时又是可以入韵的，这个要根据文意判断，以后会碰到，再解释。本章的“之”就是虚词，不做韵脚，韵脚落在“流”、“求”，属韵读的幽部，押韵；得、服、侧，职部押韵。请注意，很多人认为，“服”似乎与上下文的“得”、“侧”太不押韵了。是的，以今之汉语拼音的读音，“服”字显然与“得”、“侧”的发音不搭，但是古读它们是押韵的，押的是“ək”。“服”的古读近于今之“别”或“pe”。

```
参差荇菜，左右采（tsə）之；
窈窕淑女，琴瑟友（hiuə）之。
参差荇菜，左右芼（mo）之；
窈窕淑女，钟鼓乐（lok）之。

```
*书香君👉🏻：* 

> 尾章两韵。采、友，之部押韵；芼、乐，宵药通韵。同样地，本章各句尾字“之”是虚词，不做韵脚。“采”和“友”若按今音，差得千远万远，按韵读，就押“ə”这个韵。“芼”和“乐”也存在这个问题，读今音押韵不太好，只是听起来勉强过得去，大家可以读读看，按韵读就好很多。所以很多韵读和今音差得太多的句子并不适合朗诵。我之所以列出古读的音标，就是说明，《诗经》作品，尤其风诗，在它们被创作的时代是押韵的。小雅也比较押韵，大雅和颂就稍差。

## 🍁️ 骏注：

1、《关雎》，是周南的首篇，当然也是《诗经》的首篇，应该是《诗经》中最为读者熟悉的篇章之一。它的具体解读经过了几千年的流变。是歌颂太姒（也写作大姒，大、太通。——书香君注）之德、“风天下而正夫妇”也好，是“哀窈窕思贤才”也罢，或者就是直接咏唱青年男女纯美爱情的（也许是失恋的情歌），本文就不多谈。

2、左右流之的“流”，到底什么意思？
有些诗经解读著作解释为“择取”，有些把“流”通“捞”来解，有些解释为“顺着水流（采摘）”，我个人觉得这些解读不是不可以，也都通。但都还差点意思。流沙河先生做了一个有趣的解读（当然也未必对），我跟大家分享一下：他说，“流”就是“扭”（他的四川口音 L和 N不分的），窈窕淑女们采摘荇菜时，岸上可是有很多男孩子在看呢。彼时，采摘荇菜或食用或祭祀用，渐渐地，采荇菜成为了一种青年男女的社交活动（类似今天的集体相亲）。那么采荇菜的美女们要向岸边的帅哥们展示自己，特别是要展示身材，所以就边采荇菜边秀身段，左扭一下摆个pose，右扭一下拗个造型，是谓“左右流（扭）之”。这样解读是不是活泼一些？

3、关于“服”的音和义。前已述及，“服”字，读今音肯定不妥，完全不押韵。一般不接触《诗经》的人，摸不着头脑，以为如此厉害的《诗经》不过尔尔，怎么这么不押韵？当然不是！古音的问题我们读古代文献肯定要考虑，当然不必（也不能）所有的字都去读古音，太难读也难听。但韵脚的字还是要讲究古音。“服”，古音biuək，读音近于今之“别”，与上句的尾字“得”是押韵的。（参见王力先生《诗经韵读 楚辞韵读》中华书局，2014年3月第一版）。我听过流沙河先生读作pe。一般地，都把“思服”直接解为“思念”，或者把“服”解为无实意的虚词，也说得过去。但“服”解为虚词，跟上文的“得”不对称，“求之不得”的“得”并非虚词。我个人倾向于认可把“服”解为“迫切”的“迫”。“迫”，现在很多方言中还是读作pe而不是标准的po或pai。整句的意思和韵脚也都解决了。

4、“乐”，在现代汉语里是个多音字，读音有六个之多，分别是liao，le，luo，yue，yao，lao。“钟鼓乐之”这句中的“乐”字，意思在各种解读书籍中基本趋同，就是“使…快乐”。（我也看到有解读成“撩”的，就是现代“撩妹”的撩，意思与“使之快乐”相近，但貌似更有情趣。）那么我们现在读le或者lao我个人觉得都是可以的。当然，le这个音，古时读音近于luo，就是我在篇首原文中标注的lok。

5、这篇《关雎》，大家相对比较熟悉，所以一上来我先讲我个人认为比较关键的几个地方。那接下来还是要说说历代学者和注疏大家的解读，所谓“旧说”。

6、《毛诗大序》：“《关雎》，后妃之德也。风之始也。所以风天下而正夫妇也，故用之乡人焉，用之邦国焉。风，风也，教也。风以动之，教以化之。诗者，志之所之也。在心为志，发言为诗。情动于中而形于言，言之不足，故嗟叹之，嗟叹之不足，故永（永，咏也。——书香君注）歌之，永歌之不足，不知手之舞之、足之蹈之也。情发于声，声成文，谓之音。治世之音，安以乐，其政和。乱世之音，怨以怒，其政乖。亡国之音，哀以思，其民困。故正得失，动天地，感鬼神，莫近于诗。先王以是经夫妇，成孝敬，厚人伦，美教化，移风俗。故诗有六义焉：一曰风，二曰赋，三曰比，四曰兴，五曰雅，六曰颂。上以风化下，下以风刺上。主文而谲谏，言之者无罪，闻之者足以戒，故曰风。至于王道衰，礼仪废，政教失，国异政，家殊俗，而变风、变雅作矣。国史明乎得失之迹，伤人伦之废，哀刑政之苛，吟詠情性，以风其上，达于事变，而怀其旧俗者也 故变风，发乎情，止乎礼义。发乎情，民之性也，止乎礼义，先王之泽也。是以一国之事，系一人之本，谓之《风》。言天下之事，形四方之风，谓之《雅》。雅者，正也，言王政之所由废兴也。政有小大，故有《小雅》焉，有《大雅》焉。《颂》者，美盛德之形容，以其成功，告于神明者也。是谓四始，诗之至也。然则《关雎》《麟趾》之化，王者之风，故系之周公。南，言化自北而南也。《鹊巢》《驺虞》之德，诸侯之风也，先王之所以教，故系之召公。《周南》《召南》，正始之道，王化之基。是以《关雎》乐得淑女，以配君子，尤在进贤，不淫其色。哀窈窕，思贤才，而无伤善之心焉，是《关雎》之义也。”

*书香君👉🏻：* 

> 这篇《毛诗大序》可谓中国历史上最早的文学评论，它在《毛传》的开篇，既总揽《诗经》，又评论了《关雎》。或者从“是以《关雎》乐得淑女…”开始作为《关雎》的小序。今天是“骏注骏译”第四轮分享的第一篇，我稍微介绍一下：《毛传》在诗经每篇前都有一小段概括介绍文字，称小序，或毛诗序，或毛序。《毛传》就是大毛公毛亨和小毛公毛苌合编的《诗经》读本。诗经读本除毛传外，尚有三家诗——齐诗、鲁诗和韩诗，以及根据传习者回忆而编成的《韩诗外传》，三家诗都已失传，现在只能根据散见于典籍的一鳞半爪来引用、分析。我们当代所看到的诗经版本，其母本都是《毛传》。汉代大儒郑玄为《毛传》作注，附在诗经内文中，称《毛诗传笺》、《郑笺》，或称《郑氏笺》、《笺》。唐代学者孔颖达又为《毛诗传笺》作疏，称《毛诗正义》，又简称《孔疏》。

7、孔子评曰：“《关雎》，乐而不淫，哀而不伤。”

8、朱熹《诗集传》盛赞孔子之言：“得其性情之正，声气之和也。盖德如雎鸠，挚而有别，则后妃性情之正，固可以见其一端矣。至于寤寐反侧，琴瑟鼓钟，极其哀乐而皆不过其则焉，则诗人性情之正，又可以见全体也。”

9、方玉润认为序所谓的“后妃之德”、诗集传所谓“宫人之咏大姒、文王”，皆无确证。《诗经原始》：“此诗盖周邑之咏初昏者，故以为房中乐，用之乡人，用之邦国，而无不宜焉”。方氏认为也可以解读为咏文王、大姒的作品。圣人取之，以冠三百篇首，非独以其为夫妇之始，可以风天下而厚人伦也，盖将见周家发祥之兆，未尝不自宫闱始耳。

方玉润并盛赞《关雎》——此诗佳处全在首四句，多少和平中正之者，细咏自见。取冠《三百》，真绝唱也。”

10、闻一多《风诗类钞》评曰：“《关雎》，女子采荇菜于河滨，君子见而悦之。”

*书香君👉🏻：* 

> 当代，一般地，都把君子解为贵族男子，其爱上了采摘荇菜的窈窕淑女，但又求之不得，只能把恋爱甚至结婚的愿景寄托于想象，而深情又哀怨地咏唱。我觉得，君子也未必就是贵族男子，可以是对任何阶层男子的美称。

本诗，一般读者都比较了解，所以我只挑一些需要注意的字词来解释。

11、“窈窕”，这个词在诗经时代的意思跟今天是不同的，我们今天主要地解为外形之美，身材之美，婀娜曼妙之类，但古时有解为“安闲”的，有解为“美好”的，不仅外形美而且内心美，所谓“美心为窈，美状为窕”、“善心为窈，美容为窕”。

12、“关关”，鸟相和鸣之声。白居易《琵琶行》即有“间关莺语花底滑”之语。

13、雎鸠，一种水鸟，都是雌雄成双成对地栖息。

14、逑，匹配，配偶。“好逑”就是好的、合适的配偶，所以“好”不能读去声，要读上声。

15、芼，与“采”意近。

16、友之，友爱而亲近之，与“乐之”意近，前文已有解释。

显然，好的古诗是无需也不宜译成白话的，尤其大家都懂的诗篇更不宜译出。故此，骏译略。
