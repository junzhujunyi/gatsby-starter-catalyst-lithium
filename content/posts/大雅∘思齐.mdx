这篇的题目是你理解的意思不？林徽因的名字跟它也有关呢。



## 🌼 《大雅∘思齐》原文：

```
思齐大任，文王之母（mə）。
思媚周姜，京室之妇（biuə）。
大姒嗣徽音（iəm），
则百斯男（nəm）。
```

*书香君👉🏻：*

> “母，妇”，之部ə，押韵；“音，男”，侵部əm，押韵

```
惠于宗公（kong）。
神罔时怨，神罔时恫（thong）。
刑于寡妻（tsyei），
至于兄弟（dyei），
以御于家邦（peong）。
```

*书香君👉🏻：*

> 注意，本章是抱韵的一种衍生方式。一二五句“公，恫，邦”，东部ong，押韵；三四句“妻，弟”，脂部ei，押韵。

```
雍雍在宫（kiuəm），
肃肃在庙（miô）；
不显亦临（liəm），
无射亦保（pu）。
```

*书香君👉🏻：*

> 本章一三句“宫，临”，侵部əm，押韵；二四句“庙，保”，宵（ô）幽（u）合韵，押韵

```
肆戎疾不殄，
烈假不瑕。（无韵）
不闻亦式（sjiək），
不谏亦入（njiəp）。
```

*书香君👉🏻：*

> 本章一二句无韵，三四句“式，入”，职（ək）缉（əp）合韵，押韵

```
肆成人有德，小子有造（dzu）。
古之人无斁，誉髦斯士（dzhiə）。
```

*书香君👉🏻：*

> “造，士”，幽（u）之（ə）合韵，押韵


## 🍁️ 骏注：

1、我们之前介绍《小雅》时提到，古时雅颂排列时是分“什”的。以十篇为一组叫“什”，以每组篇首的篇名为组名。《大雅》从《文王》到《文王有声》10篇，就称之为“文王之什”。接下来有“生民之什”10篇和“荡之什”11篇。它们共同组成了《大雅》的31篇。

古代这种以十篇为一组的编排方式，或许为了装订、翻检、阅读的便利，或许觉得同一组十篇中，篇与篇之间有某些相似相近之处。当代，《诗经》出书已不大采用这种编组方式了。

2、今天要分享的这篇《思齐》，是《大雅.文王之什》的第六篇。名字就有讲究。“思”只是语助词，无实意。而“齐”通“斋”（或“斋”的借字），也要读zhai阴平，“庄敬”的意思。可不要把“思齐”理解成“想着要看齐”或者“思念齐国”之类，那就贻笑大方了。

3、本篇有一句“大姒嗣徽音”，是民国一代才女“林徽因”名字的出处。林徽因最早叫“林徽音”。

4、本诗是一首对周文王的赞美诗。通过赞美三太（太姜，太任，太姒）来歌颂周文王的敬祖、礼妻、友弟、自省、纳谏、善教、育才、选贤之美德。

历代有代表性的点评：

《毛诗序》因评曰：“《思齐》，文王所以圣也”。

《毛传》：“言非但天性，德有所由成。”

《郑笺》：“常思庄敬者大任也，乃为文王之母，又常思爱大姜之配大王之礼，故能为京室之妇。言其德行纯备，故生圣子也。”

朱熹《诗集传》曰：“此诗亦歌文王之徳”。

《诗经原始》：“《思齐》，刑于化洽也。文王治家，不独以身为率，又得圣母以为之倡，故其宫闱寝庙间，肃肃雝雝，太和翔洽，莫可言喻。此盖其母大任氏德性齐庄，而又能上媚先姑，以尽子妇之职。故其子妇亦有所式，化而成内助之功。此文王刑于之化至神且远，而独有异乎人者也。故此诗当以刑于数语为主。”

5、“思”：发语词，无实义。“齐”，“斋”的借字，端庄、肃静貌。

6、“大任”，即太任，王季（古公亶父之子，又叫姬季、季历）之妻，文王之母。

7、“媚”，美好。一说“爱”。《诗集传》：“媚，爱也。”

8、“周姜” 即太姜。古公亶父之妻，王季之母，文王之祖母。

9、“京室”，周室，王室。

10、“大姒”，即太姒，文王之妻。这里说一下“大”这个字。“大”，古时读音只有tai和dai，没有da。（当然，我们说的都是古读，只是近似地用汉语拼音来标注而已，其读音与汉语拼音并不完全相同）。“士大夫”宜读成“士dai夫”。先秦文献中的“大王”根据语境，要读成“dai王”或“tai王”，不宜读“da王”。现在描述土匪头子的“山大王”也应读“山dai王”。

11、“嗣”，继承，继续。

12、“徽音”，美誉。《郑笺》：“徽，美也。嗣大任之美音，谓续行其善教令。”

13、“百斯男”，众多男儿。百，虚指，泛言其多。斯，语助词，无实义。男，男丁，指儿子。不是指“子孙”。这句“百斯男”造成歧义，后世附会周文王有百子，九十九个亲生一个养子（雷震子）。事实上周文王有据可查的儿子只有十七个。长子伯邑考死于纣王之手，次子姬发就是武王。其他儿子有周公旦、召公奭、霍叔管叔蔡叔康叔振铎叔等等一堆叔（“叔”，指“武王之弟”也），周族立朝时没死的叔们都封国封侯，从尧舜禹的禅让制变成了妥妥的家天下，贻害两千多年。

14、“惠”，恭顺、孝敬。《郑笺》：“惠，顺也。”

15、“宗公”，宗庙里的历代先祖。“神”，此指祖先之神。“罔”，无。“时”，所。“恫”，读音tong，阴平。哀痛。《毛传》：“恫，痛也。”

16、“刑”，通“型”，典型，典范。“寡妻”，嫡妻。《毛传》：“刑，法也。寡妻，适妻也”。“御”，治理。

17、“雍雍”：和洽貌。有些版本作“雝雝”。“宫”，家，宫室。

“肃肃”，恭敬貌。“庙”，宗庙。“雍雍在宫，肃肃在庙”二句，言“文王在闺门之内则极其和，在宗庙之中则极其敬”。

18、“不显亦临，无射亦保”这两句是本诗难点之一。书香君也头疼。主要有两种解读方式，都通，但难在对字的理解不仅各有所本而且大相径庭。我来分述如下供大家参考。

第一种：“不显”的“不”，是虚词，无实意，要读作pi阴平。不能当做否定词“不”来解。（“不”的这种虚词用法在雅颂里常见，以后碰到我们还会介绍。另外，“丕显”一词在商周、先秦时代的文献尤其铭文里也经常写成“不显”，是“英明”的意思——丕，大也；显，光也。但，本诗中，结合上下文，我觉得“不显”不能当做“丕显”来理解。而应把“不”当做虚词。）

“不显”就是“显”，意思就是“明处，明显处”。“临”，临视，临察，省察。“不显亦临”就是“明显处要省察”。

“无射”的“无”对应上句，也是虚词，无实意。“射”，与“夜，夕”因叠韵而通用，就是“晦暗，不明”。“保”，保持，保守。“无射亦保”意思是“晦暗处也谨慎自持、自矜警醒”

两句总体意思是“明处既临视自省，暗处也谨慎警醒”这是夸周文王始终兢兢业业，明显处和细微处都能审察，对形势保持警醒。

第二种。“临”和“保”的解读与第一种相同。

“不显”，就按字面理解为“不明显之处”。“无射”就是“无斁”，不厌倦也。“射”为古“斁”字，读音也是yi而不是she。两句的总体意思就是“细微之处也临察审视，修身不倦且谨慎自持”。不过，我对“无射”通“无斁”仍有疑惑，请参“骏注25”。

19、“肆”，于是。“戎疾”，西戎之患。一说，戎，大也。，“戎疾”，就是很大的灾难。“殄”，断绝、灭绝。“肆，戎，殄”这三个字我们上篇《绵》里也解释过，请参。

20、“烈假”，指害人的疾病。“瑕”，远也，远去，指灾害离开，解除，与上文的“殄”义近。都是指灾患祸害灭除。

21、“不闻亦式”的“不”，是虚词，无实意。“闻”，听到。“式”，适合，适用。“不谏亦入”的“不”同上，也是虚词。“谏”，进谏。“入”，接受，采纳。

22、“不闻亦式，不谏亦入”这两句意思是“听善言则用之，有进谏则纳之”。

23、“小子”，青年人。造：造就。此指可资造就。

24、“古之人”，指周之历代先王，尤指文王。

25、无斁（yì）：无厌，无倦。对照18的第二种解读，我有困惑了，本诗既然已经有“无斁”这个词出现，为什么还要用“无射”来代替呢？《诗经》对词的重复出现并不忌讳呀。

26、“誉”，美名，声誉。

27、“髦”，俊，俊杰。“斯”，语助词。“士”，泛指周朝士人。一说“斯士”指上文的“小子”。“古之人无斁，誉髦斯士”意思是“古代先贤不遗余力地教导，使得士人们（或青年们）都能成为英才”。

分享《大雅》，主要是研读字词句式和了解相应的历史，“骏译”就不发了。
