《诗经》305篇中，《萚兮》是最短篇之一，文辞极为简单。这种简朴单纯的歌谣，虽然古老，却又是常读常新的，今天亦然。

## 🌼 《郑风∘萚兮》原文：
```

萚（thak）兮，
萚（thak）兮！
风其吹（thjiuai）女。
叔兮伯（peak）兮，
倡予和（huai）女。

```
*书香君👉🏻：* 

> 首章两韵。第一二四句“萚、萚、伯”，铎部，押韵；第三、五句“吹、和”，歌部，押韵

```
萚（thak）兮，
萚（thak）兮！
风其漂（phio）女。
叔兮伯（peak）兮，
倡予要（io）女。

```
*书香君👉🏻：* 

> 尾章两韵。第一二四句与首章叠句。第三五句“漂、要”，宵部，押韵

```
我们之前也介绍过这种押韵方式，可以看做两个声部轮流唱。一二句也可以合并成一句，有些版本也是这样安排的，即“萚兮萚兮”连在一句处理，那就是一三句、二四句分别押韵（同时也换韵，我自己把这种押韵方式称为“跳韵”），典型的男女对唱或者两个声部轮唱的模式。一个唱一三句，另一个唱二四句。我们如果以唱歌的方式来理解《诗经》的韵脚，很多地方为什么那样处理就豁然开朗了。关于《诗经》的韵脚，这可能是书香君首次提出“跳韵”或轮唱这个说法（迄今为止我还没有看到其他文字这样说过，也许我孤陋寡闻，在这里请大家指正，谢谢）。

```
## 🍁️ 骏注：

1、当代一般认为这是一首男女聚会（或约会）时对唱（女子先唱）的歌曲，或者是感伤季节变换、时光流逝的诗作。

2、旧说是“刺诗”（大多）或“淫女之辞”（朱熹）。即便持“刺诗”论者，在细节上各家说法也有差别。

3、《毛诗序》：“《萚兮》，刺忽也。君弱臣强，不倡而和也。”

什么意思呢？古时君臣之礼是人臣待君倡而后和。就是说国君先唱（“倡”就是“唱”——书香君注）了，做臣子的才和歌，否则就是失礼。朝堂上，国君未唱臣子就开唱了，或者大臣之间互相唱和了，这就表明了君弱臣强。

4、《郑笺》：“不倡而和，君臣各失其礼，不相倡和。萚，槁也。木叶槁，待风乃落。风喻号令也，喻君有政教，臣乃行之。言此者，刺今不然。”

郑玄这个解读很奇妙哎，他认为这首诗是反话正说。

5、朱熹还是有眼力的，他不认为是刺诗。虽然他脱不了理学家的窠臼，解本诗曰：“此淫女之词”。但他毕竟看出了诗中的男女之情，其《诗集传》谓：“言萚兮萚兮，则风将吹女矣。叔兮伯兮，则盍倡予，而予将和女矣。”

6、但是方玉润不干了。他严厉地驳斥了朱子《诗集传》的“淫女之词”观点，他说：“天下行淫之女，岂有呼叔而又呼伯者？且叔伯何所倡，而女又何所和？言之不徒污人齿颊，讵可以之释经？”（“讵”，读音ju去声，字义是“难道”——书香君注）。

方氏认为本诗是“讽朝臣共扶危也”，并赞赏严粲的以下解读与诗旨最为接近——“此小臣有忧国之心，呼诸大夫而告之。言槁叶风吹不能久矣，岂可坐视，叔伯诸大夫其亟图之，患无其倡，不患无和之者。”

方氏认为，联系到姬忽（郑桓公）之世，权臣专擅而国君微弱，序笺说本诗“刺忽”也说得过去。

7、当代程俊英《诗经注析》：这首诗可能是当仲春“会男女”的集体歌舞曲，称叔称伯，显然是女子带头唱起来，男子跟着应和的。而且不止两个人，而是一群男女的合唱。

8、事实上，早在《诗经》成书不久的春秋中晚期（《诗》是春秋中叶定编成书的，其时名《诗》，之后才兼称“诗三百”，西汉始名《诗经》），就有人把本篇《萚兮》解为“昵燕好”之词了。

事见《左传.昭公十六年》：

“…夏四月，郑六卿饯宣子于郊。宣子曰：二三君子请皆赋，起亦以知郑志。…子游赋《风雨》，子旗赋《有女同车》，子柳赋《萚兮》。宣子喜曰：郑其庶乎！二三君子以君命贶起，赋不出郑志，皆昵燕好也。二三君子数世之主也，可以无惧矣。”宣子把《风雨》《有女同车》和《萚兮》都当作“昵燕好”之词，虽然宣子没有单独说《萚兮》，但是，这三篇以今人眼光看，都是有关夫妻或男女情感的诗作，可见，那时候就有人解为“昵燕好”之“男女词”了。

9、我个人的观点之前也说过——《诗经》中的篇章刚创作出来时，也许确实有旧说中解读的诗旨，比较含蓄（所谓“谲谏”者），毕竟三家诗和毛诗的时代（战国和西汉）离春秋更近。如果我们今天一味地讽刺旧说“附会”也未必正确。但是时代在进步，《诗经》的赏析更多地落到它的史学和文学价值上来。尤其，欣赏它的文学价值，才是我们今天读《诗经》最紧要的事。正是在这个意义上，我倾向于当代的解读，但不代表我们完全否定前人的所谓“旧说”。

10、我们还可以把本诗的“风吹其女”的“风”比作男子，把“萚”比作女子，诗中女子的口吻，希望男子能象风吹萚一样，把感情倾投到女子的身上，把情爱的歌儿与她相唱和。我个人觉得，“叔兮伯兮”，未必非得是几个人，当作女子喜欢的某一个男子就好，泛指代替专指，更加符合诗意。向着众人说话，其实是说给其中的某一个人听的。不是吗？其中的况味我们或可心领。

11、萚，读音tuo阴平，树木枯槁即将凋零的枯枝残叶的统称。

12、女，汝、你。注意了，诗中有两处出现“女”这个字，我认为，“风其吹女”之“女”，就是“萚”，枯枝残叶。
“倡予和女”之“女”就指的是与女子唱和的某个或某些男子。

13、叔、伯，女子呼爱人为伯或叔或叔伯。“叔兮伯兮”语气像对两人说话，实际是对一人说话。旧说把“叔伯”解为群臣按长幼互相称谓之语。《毛传》：“叔伯，言群臣长幼也。”《郑笺》：“叔伯，群臣相谓也，兄弟之称。”

14、倡，同“唱”。

15、予，我。

16、“倡予和女”，倒文，即今之倒装句。就是“予倡女和”，我唱你和。

17、漂，通“飘”，吹动。有些版本也作“飘”。

18、要，读音yào，会合。指唱歌的人以声音相会和，也就是我们今天说的“和声”。一说通“邀”（那“要”字就要读阴平），指女子希望男子邀自己相会。一说“成、和”。

19、《诗经》305篇中，《萚兮》是最短篇之一，文辞极为简单。这种简朴单纯的歌谣，虽然古老，却又是常读常新的，今天读来亦然。
