一去乡里远其兄弟
举目无亲谁可因依

## 🌼 《王风∘葛藟》原文：
```

绵绵葛藟，在河之浒（xa）。
终远兄弟，谓他人父（biua）。
谓他人父（biua），
亦莫我顾（ka）。

```
*书香君👉🏻：* 

> “浒、父、父、顾”，鱼部，押韵

```
绵绵葛藟，在河之涘（zhiə）。
终远兄弟，谓他人母（mə）。
谓他人母（mə），
亦莫我有（hiuə）。

```
*书香君👉🏻：* 

> “涘、母、母、有”，之部，押韵

```
绵绵葛藟，在河之漘（djiuən）。
终远兄弟，谓他人昆（kuən）。
谓他人昆（kuən），
亦莫我闻（miuən）。

```
*书香君👉🏻：* 

> “漘、昆、昆、闻”，文部，押韵

```
本诗三章，每章一韵。

```
## 🍁️ 骏注：

1、这是一首流亡者（流浪者）失去家族依靠、求助社会而不得的自叹诗。

还有人认为本诗是一个入赘者在他人家生活，抒发孤独寂寞的悲歌。或可备一说。

2、《毛诗序》：“《葛藟》，王族刺平王也。周室道衰，弃其九族焉。”

*书香君👉🏻：* 

> 毛序说本诗是周平王的同族所作，刺平王寡于恩施、远弃族亲，不顾眷我、不与我相闻命。

今天看来，失之泛泛。诗中“亦”字，语气与“刺平王弃其九族”似有不洽，诗意也索然。

3、朱熹《诗集传》：“世衰民散，有去其乡里家族而流离失所者，作此诗以自叹。言绵绵葛藟，则在河之浒、之涘、之漘矣。今乃终远兄弟，而谓他人为己父、己母、己兄，己虽谓彼为父为母为兄，而彼亦不我顾，则其穷也甚矣。”

朱熹说得就具体些了。

4、方玉润对本诗的题解和眉评就是“民穷无所依也。沉痛语，不忍卒读。”他认为朱熹的解读是合适的（“斯得之矣”），而毛传之“刺平王弃九族”之诗旨解读是不妥的（“反无谓也”）。方氏又言：葛藟本蔓生，必有所依而后附，今乃在河之浒与涘与漘，无乔木高枝以引其条叶，虽足自庇本根，而本根已失，奈之何哉？故人一去乡里远其兄弟，则举目无亲谁可因依？虽欲谓他人之父以为父、他人之母以为母、他人之兄以为兄，而其所欲谓之人反愕然而不之顾；民情如此，世道可知。谁则使之然哉？当必有任其咎者。

书香君深以为然。

5、今人程俊英/蒋见元《诗经注析》：“这首诗的作者可能是从洛阳附近的邻国逃亡到王都的，他到处乞求，甚至称别人为父母兄弟，希望得到一点同情和救济，但人们给他的却只有白眼。诗深刻地反映了当时炎凉无情的世态。”

6、绵绵，有些版本作“緜緜”，连绵不绝貌。

7、葛藟，藤类蔓生植物，野葡萄。

8、浒，水边。

9、终，既、已。终、既、已，三字同义。《毛传》释“终远兄弟”曰：“兄弟之道，已相远矣。”

10、远，此处要读去声，远离、离弃。

11、顾、有、闻：皆亲爱之意也，可解作“眷顾、照顾”。

“有”通“友”。《郑笺》：“有，识有也。”马瑞辰：“有当读为亲有”。杜预：“有，相亲有也。”
一说通“佑”，帮助。我采前说“友”。

闻（此处要读去声），问、恤问、慰问。王引之《经义述闻》：“谓相恤问也。古字闻与问通”。亦莫我闻，犹“亦莫我顾/有”也。

12、涘（si去声）：水边。《说文》：“涘，水厓也。”厓，即“涯”。

13、漘（chun阳平），河岸，深水边。厓上平坦而下水深者为漘。就是水边较高的平土岸。

14、昆，兄。周人谓兄为“昆”。《尔雅.释亲》作“晜”。《礼经》：大功以上为昆弟，小功以下为兄弟，以别亲疏。大功小功指五服之内的某种血缘范畴。大功比小功血缘关系更近。

## 🍁️ 骏译：

> 绵绵长长葛藟藤，
> 
> 在那水涯岸边生。
> 
> 远离兄弟与亲友，
> 
> 求人帮忙喊父亲。
> 
> 即便叫爹含悲声，
> 
> 一点眷顾也不肯。

> 绵延葛藤长又长，
> 
> 缠绕丛生河岸旁。
> 
> 骨肉兄弟都离散，
> 
> 对着外人喊亲娘。
> 
> 即便喊娘声悲怆，
> 
> 不曾把我当儿郎。

> 葛藟藤蔓生绵延，
> 
> 长在小河水岸边。
> 
> 兄弟亲人已离散，
> 
> 面对他人喊兄长。
> 
> 叫人哥哥心悲伤，
> 
> 不闻不问情何堪。
