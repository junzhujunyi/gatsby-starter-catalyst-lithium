姚际恒赞此诗：先从绿衣言黄里，又从绿衣言丝，又从丝言絺綌，似乎无头无绪，却又若断若连，最足令人寻绎。

## 🌼 《邶风∘绿衣》原文：
```

绿兮衣兮，绿衣黄里（liə）；
心之忧矣，曷维其已（jiə）！

```
*书香君👉🏻：* 

> 首章一韵。“里、已”，之部，押韵

```
绿兮衣兮，绿衣黄裳（zjiang）；
心之忧矣，曷维其亡（miuang）！

```
*书香君👉🏻：* 

> 二章一韵。“裳、亡”，阳部，押韵

```
绿兮丝（siə）兮，
女所治（diə）兮；
我思古人，俾无訧（hiuə）兮。

```
*书香君👉🏻：* 

> 三章一韵。“丝、治、訧”，之部，押韵。本章句尾“兮”字，是语助词，不做韵脚

```
絺兮綌兮，凄其以风（piuəm）；
我思古人，实获我心（siəm）。

```
*书香君👉🏻：* 

> 尾章一韵。“风、心”，侵部，押韵

## 🍁️ 骏注：

1、今天这首《绿衣》比较简单，不过学界仍有两种看法。

其一，以序笺、孔颖达和朱熹观点为代表，认为是庄公妻子卫姜（也叫庄姜、卫庄姜）向庄公表达忧思哀怨的诗。

《毛诗序》：“《绿衣》，卫庄姜伤己也。妾上僭，夫人失位，而作是诗也。妾上僭者，谓公子州吁之母，母嬖而州吁骄。”

郑玄认为，标题“绿衣”之“绿”，应为“褖”字之误。褖，汉语拼音读音是tuan去声，古时“礼服”。《郑笺》释：“褖衣自有礼制也。诸侯夫人祭服之下，鞠衣为上，展衣次之，褖衣次之，次之者，众妾亦以贵贱之等服之。鞠衣黄，展衣白，褖衣黑，皆以素纱为里。今褖衣反以黄为里，非其礼制也，故以喻妾上僭。”

孔颖达《毛诗正义》：“作《绿衣》诗者，言卫庄姜伤己也。由贱妾为君所嬖而上僣，夫人失位而幽微，伤己不被宠遇，是故而作是诗也。”

朱熹《诗集传》：“绿，苍胜黄之间色。黄，中央土之正色。间色贱而以为衣，正色贵而以为里，言皆失其所也。庄公惑于嬖妾，夫人庄姜贤而失位，故作此诗，言绿衣黄里，以比贱妾尊显．正嫡幽微，使我忧之不能自已也。”

什么意思呢，庄公被嬖妾迷惑，使得贤惠的正夫人庄姜失去地位。衣服颜色外绿而里黄——普通的绿色（绿色是青色、黄色之间的过渡色，为间色，相对为贱）做成了彰显的外衣而尊贵的黄色反倒被做成衬里或内衣，喻指地位低的妾取代了正妻的地位，表达自己作为正嫡反而不受重视的郁闷。后世严粲、姚际恒、方玉润等基本从序笺、《孔疏》、《诗集传》之说。

方玉润还特别肯定是庄姜所作。方氏提到第三章“绿兮丝兮”，谓此绿也，本丝也，素洁之时汝之所制，何为而染成此绿也？！犹墨子悲丝，谓其“可以黄，可以黑”之意。

*书香君👉🏻：* 

> 这里，方玉润引用了“墨子悲丝”的典故——墨子见染丝者而叹曰：染于苍则苍，染于黄则黄。五入为五色，不可不慎也。非独染丝，治国亦然。

其二、是庄公自己思念亡妻的诗作。

庄公这位亡妻就是庄姜，就是《硕人》里“手如柔荑，肤如凝脂，螓首蛾眉，巧笑倩兮，美目盼兮”的那位绝色新娘子。庄姜是齐国公主（春秋时期，齐国出美女），虽貌美但似乎并不受卫庄公待见，她婚后郁郁寡欢，很不幸福。

现、当代学者基本倾向于《绿衣》是思念亡妻的怀人佳作，并且应该是中国最早的悼亡诗。

闻一多《风诗类钞》：“《绿衣》，感旧也。妇人无过被出，非其夫所愿。他日，夫因衣妇旧所制衣，感而思之，遂作此诗。”

程俊英/蒋见元《诗经注析》：“这是诗人睹物怀人思念故妻的诗。”

王秀梅《诗经》：“这是一首思念亡故妻子的诗。诗人睹物怀人，看到妻子亲手缝制的衣裳，想到妻子对自己各方面的关心照顾，现在已物是人非，因此内心充满忧伤。”

骆玉明《诗经.风》：“绿为间色而为衣，黄为正色而为裳，选色倒错，譬行为违礼，故思古人之义。一说见衣而思人，怀念亡妻。”

至于作者是不是卫庄公，不能确证。我们就当作通常意义的怀念亡妻的诗篇就行了。

3、“衣”，上衣。

4、“裳”，读音chang阳平。下衣。

5、“里”，内衣。外衣曰衣，内衣曰里。一说，“里”指衣的衬里。

6、“已”，止、止息。

7、“维”，语助词无实意。可不译或根据上下文语境灵活翻译。

8、“亡”，通“忘”。

9、“治”，理也。拾掇、打理、修补等等。

10、“古人”，故人。此处应指诗者的亡妻。

11、“俾”，使。

12、“訧”，读音you阳平。同“尤”，过失，过错、罪过。

13、“絺”，读音chi阴平。细葛布。

14、“绤”，读音xi去声。粗葛布。

15、“凄”，凉而有寒意。“凄其”，同“凄凄”。就是风吹身上寒凉。有些版本解读成“凉爽”，我以为不妥。粗布细布功用不同，穿的季节也不同。书香君认为应该是作者感慨妻子不在身边，无论粗布细布衣服都挡不住寒凉（主要是内心的感觉）。

16、“以”，因。

17、“获”，得。
