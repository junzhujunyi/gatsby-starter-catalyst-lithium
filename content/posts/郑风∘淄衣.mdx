## 🌼 《郑风∘缁衣》原文：
```

缁衣之宜（ngiai）兮，
敝予又改为（hiuai）兮。
适子之馆（kuan）兮，
还予授子之粲（tsan）兮。

```
*书香君👉🏻：* 

> “宜、为”，歌部，押韵；“馆、粲”，元部，押韵

```
缁衣之好（xu）兮，
敝予又改造（dzu）兮。
适子之馆（kuan）兮，
还予授子之粲（tsan）兮。

```
*书香君👉🏻：* 

> “好、造”，幽部，押韵；后两句与首尾两章的后两句完全一样，叠句、遥韵

```
缁衣之席（zyak）兮，
敝予又改作（tzak）兮。
适子之馆（kuan）兮，
还予授子之粲（tsan）兮。

```
*书香君👉🏻：* 

> “席、作”，铎部，押韵；后两句与前两章的后两句叠句、遥韵。

```
本诗三章，每章两韵。

```
## 🍁️ 骏注：

昨天分享了《王风》的最后一篇（《王风》十篇就全部完成了）。今天开始《郑风》的分享。

郑guo，事实上有两个阶段，这两个阶段的地域、都城都不相同。现在河南的“新郑”，名字就有来头，与郑guo的新阶段有关。我们来捋一下。最早的郑guo是周宣王姬静封给同父异母弟弟姬友（郑guo第一任guo君郑桓公，他与姬静都是周厉王的儿子）的采邑，在今陕西华州一带（以前叫华县，古时称棫林、咸林）。周宣王的儿子周幽王即位后，任命叔叔郑桓公为周王室司徒，分管土地户籍类事务，受周人爱戴。郑桓公有远见，看到周幽王昏聩无能，想迁走郑guo，在太史伯的建议下，选中了虢、郐两guo附近区域，并派儿子掘突（郑武公）向虢、郐两guoguo君行贿，共借了虢、郐十个州的土地，逐步迁徙郑guo的财物、人口安置于此，筹备为新的郑guo（都城就是现在河南省新郑市一带。“新郑”之名即由此来。新郑，新的郑guo也，或者新的郑都也）。事实上是强借不还，郑武公时代还灭了郐guo（就是桧guo，诗经中的《桧风》即是桧地民歌）。“桓公寄孥”或“虢郐寄孥”说的就是这事，以后涉及到再细说。

周幽王时期，猃狁之难致西周灭，周幽王、郑桓公同死。平王东迁，郑武公也正式将郑guo由今陕西西安一带东迁到现在的河南开封一带。

《郑风》就是东周时期、郑武公时代的郑guo民歌，收录21篇，今天分享的《淄衣》，是《郑风》的首篇。

1、这首《缁衣》，主要有四种有代表性的解读。

其一，赞美guo君厚待投奔的贤士的作品。旧说多认为赞美郑武公（郑桓公之长子掘突），也有认为是赞美郑桓公的。

《毛诗序》：“《淄衣》，美武公也。父子并为周司徒（郑桓公、郑武公父子二人都被周王室任命为司徒——书香君注），善于其职，guo人宜之，故美其德，以明有guo善善之功焉。”

说是guo人赞美郑武公有美德，对guo家有功（事实上，平王东迁时，郑武公确实有大功相助，且郑武公是周平王的叔叔——书香君注）。

其二、guo人表达对郑武公的爱戴。

朱熹《诗集传》：“旧说郑桓公、武公相继为周司徒，善于其职。周人爱之，故作是诗。言子之服淄衣也甚宜，敝则我将为子更为之。且将适子之馆，既还而又授子以粲。言好之无已也。”

说的是周人爱戴郑武公（或郑桓公），愿意为他修补衣服、做美食，这是一种爱戴之情的表达。

其三、赞美郑武公好贤。
贤士官服破旧了及时为其缝制新衣（“改衣”）、去贤士居所探望（“适馆”）、为其提供美食（“授粲”，粲，此种语境就解作“餐”）来体现guo君/大夫对贤才无微不至的关怀。古时图谋大业的君主对可用之才，建立馆舍，出有车，食有鱼肉。 历史记载很多。

方玉润《诗经原始》：“…武公居则虚衷以前席，出则凭轼而过门。罗贤以礼不以貌，亲贤以道尤以心。贤所以乐为用、而共成辅guo宏猷。guo人好之，形诸歌咏，写其好贤无倦之心，殆将与握发吐哺后先相映，为万世美谈，此《淄衣》之诗所由作也。”

其四，妻（或妾）对身为官员的丈夫体贴照顾（如补旧衣、赠新衣或做好精美餐食以待——“粲”指新衣之鲜亮、或指“餐”、或指粟之精凿者。），表达恩爱情意的作品。

闻一多先生就解为“赠衣诗”。

以上几种似皆通，但细细品味诗意，“家人（或妻子、妻妾）赠衣诗”说似更切合。

2、缁，音zī，黑色。缁衣，春秋时期，官员上朝面见君王要穿皮裘类朝服，回自己办公的馆舍（古称“私朝”）则要穿官服。《郑笺》：“淄衣者，居私朝之服也。天子之朝服，皮弁服也。”
当时官服是黑色的。故“缁衣”在本诗中喻指官服（缁衣视语境还有囚服、底层劳作之人所穿工服的意思）。

3、敝，坏，指衣服破旧。

4、席，有些版本作“蓆”，宽大、舒适貌。《毛传》：“蓆，大也”。

5、宜，合适、合身；好，美好、漂亮；席，宽大舒适。
此三字都是形容衣服质量之好看、合身、舒适等等，换字是从多角度来描述衣服，也是为了押韵和避免重复。

6、“改为、改造、改作”，也都是重新制作服装的意思。一说这是随着衣服的破烂程度不同而加以不同程度的缝制、加工。
《郑笺》：“改，更也。造，为也。作，为也。”

7、适，去，往。

8、馆，官员办公场所。 《郑笺》：“卿士所之之馆，在天子之宫，如今之诸庐也。”

9、还，通“旋”，读音也是xuan阳平。回来，回家。

10、粲，漂亮、崭新。指所做的新衣服好看、鲜亮。旧说“粲”通“餐”，或指“粟之精凿者”，代指精细的食品，《郑笺》：“设餐以授之，爱之欲饮食之。”也通。
