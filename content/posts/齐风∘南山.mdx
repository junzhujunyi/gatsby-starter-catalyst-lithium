姜老太公名字多，
子孙后代故事多。

春秋五霸之一的齐桓公、本诗男主人公之一齐襄公、本诗女主人公文姜，以及我们前面其他篇章多次涉及到的宣姜、公子纠，他们之间都是兄弟姐妹关系（大多同父异母）。

对照历史文献中明确存在的“齐襄公与文姜兄妹乱伦”的记述，这篇《南山》，说它刺齐襄、鲁桓，当非空穴来风。

## 🌼 《齐风∘南山》原文：
```

南山崔崔（tsuəi），
雄狐绥绥（siuəi）；
鲁道有荡，齐子由归（kiuəi）。
既曰归（kiuəi）止，
曷又怀（hoəi）止？

```
*书香君👉🏻：* 

> 首章一韵：“崔、绥、归、归、怀”，微部，押韵

```
葛屦五两（liang），
冠緌双（sheong）止。
鲁道有荡（dang），
齐子庸（jiong）止。
既曰庸（jiong）止，
曷又从（dziong）止？

```
*书香君👉🏻：* 

> 二章两韵：一三句“两、荡”，阳部，押韵；二四五六句“双、庸、庸、从”，东部，押韵

```
蓺麻如之何？衡从其亩（mə）。
取妻如之何？必告父母（mə）。
既曰告（kuk）止，
曷又鞠（kiuk）止？

```
*书香君👉🏻：* 

> 三章两韵：一二句“亩、母”，之部，押韵；三四句“告、鞠”，觉部，押韵

```
析薪如之何？匪斧不克（khək）。
取妻如之何？匪媒不得（tək）。
既曰得（tək）止，
曷又极（giək）止？

```
*书香君👉🏻：* 

> 尾章一韵：“克、得、得、极”，职部，押韵

```
本诗四章。首尾两章各一韵，中间两章各两韵。

```
## 🍁️ 骏注：

1、今天分享《南山》，为了让朋友们了解旧说的渊源，就顺便再介绍下背景。

武王姬发灭商后大封兄弟和有功之臣，封首功者姜子牙于营丘，即土著的东夷人之所，建齐国。所以姜太公是齐国的开国国君。姜子牙姜姓，字子牙。“姜”是上古华夏族源于母系社会的八大姓（姬，姜，姒，嬴，妘，妫，姚，姞/妊）之一。传说姜姓最早的来源是居于姜水之畔的炎帝神农氏。姜子牙先祖助大禹治水有功被舜封于吕地，姜姓之族遂以地称氏，姜姓而吕氏。太公名尚，又，名望。故，姜尚、姜望、吕尚、吕望都是姜子牙的名字（是不是太多了啊？此外他还别号“飞熊”）。齐国民风“阔达多匿智，其天性也”（《史记•齐太公世家》）。“其俗宽缓阔达，而足智，好议论” （《史记•货殖列传》）。太公尊重齐地土著居民这样的民风民俗，封齐伊始就“因其俗，简其礼”。齐国因此不断发展壮大，到他的第十二代孙，著名的齐桓公（就是跟公子纠争王位、咬舌吐血蒙骗管仲的那个公子小白，又叫吕小白、姜小白）终于成为春秋五霸之一。齐人阔达足智、观念自由开放，加之齐国君主“因民俗”的政策，齐文化得以博大宏富。《齐风》就产生于这样的社会环境。所以公子季札听诗时，赞《齐风》曰：“美哉，泱泱乎大风也哉！表东海者，其大公乎？国未可量也。”（《左传•襄公二十九年》）。师乙与子贡论乐：“温良而能断者，宣歌《齐》（《齐》即《齐风》）”。“《齐》者，三代之遗声也；齐人识之，故谓之《齐》。……明乎齐之音者，见利而让。临事而屡断，勇也。见利而让，义也。有勇有义，非歌能保此？”（《礼记•乐记》）。司马迁在《史记•齐太公世家》中评论说：“吾适齐，自泰山属之琅邪，北被于海，膏壤二千里，其民阔达多匿知，其天性也。以太公之圣，建国本，桓公之盛，修善政，以为诸侯会盟，称伯，不亦宜乎？洋洋哉，固大国之风。”
诚然。

2、齐风共11篇。其中，《南山》、《敝笱》、《载驱》这三篇古时多认为与文姜与其兄齐襄公不伦之恋有关。现代则多认为是有关送女出嫁的诗作。《猗嗟》古时认为是赞婿之歌，今人多谓女子夸赞男友之诗（一如《叔于田》）。二论差别不大。《还》、《卢令》这两篇乃歌颂猎人勇武英俊之诗，古今皆然其说。《鸡鸣》、《著》、《东方之日》、《东方未明》、《甫田》这五篇多解读为男女相恋之歌。当然，《东方未明》另有解读成刺诗的也有相当道理。而对历史文献中明确存在的“齐襄公与文姜兄妹乱伦”的记述，这篇《南山》，说它刺齐襄、鲁桓，当非空穴来风。我采此说。

3、刚才说的春秋五霸之一的齐桓公、本诗男主人公之一齐襄公、本诗女主人公文姜，以及我们前面多次介绍过的宣姜、公子纠都是兄弟姐妹关系。文姜嫁的是鲁桓公，又与同父异母哥哥齐襄公有不伦之恋导致丈夫被杀（齐襄公指使公子彭生杀了鲁桓公）。上述一干人等波澜起伏的故事在《诗经》中占有不少篇幅。

《毛诗序》：“《南山》，刺襄公也。鸟兽之行，淫乎其妹，大夫遇是恶，作诗而去之。”

《毛传》：“…齐大夫见襄公行恶如是，作诗以刺之，又非鲁桓公不能禁制夫人而去之。”

《郑笺》：“雄狐行求匹耦于南山之上，行貌绥绥然，喻襄公居人君之尊，而为淫泆之行，其威仪可耻恶如狐。”

郑玄认为本诗前两章刺齐襄公和文姜，后两章“非鲁桓”（非，指责也。鲁桓，鲁桓公——书香君注）。

《诗集传》：“南山有狐，以比襄公居高位而行邪行。且文姜既从此道归乎鲁矣，襄公何为而复思之乎？欲取妻者必先告其父母。今鲁桓公既告父母而取妻矣，又曷为使之得穷其欲而至此哉？此诗前二章刺齐襄，后二章刺鲁桓也。”

《诗经原始》：“《南山》，刺襄公淫其妹，而鲁不能禁也。”

人们对于此诗究竟是刺其齐襄公的还是刺文姜的还是刺鲁桓公的，说法不一、其依据的理由也各异。
《毛诗序》笼统地说刺襄公的；
《毛传》和朱熹《诗集传》都认为是刺齐襄公和鲁桓公的；
《郑笺》及方玉润《诗经原始》都认为刺齐襄、鲁桓和文姜三人的；
明代儒家学者季明德认为“通篇刺文姜”；
严粲《诗辑》、姚际恒《诗经通论》则认为“通篇刺鲁桓”。

4、南山，非泛指，齐国实有其山，又称牛山。非“寿比南山”中的终南山。

5、“崔崔”，通“崔嵬”。《毛传》：“崔崔，高大也。国君尊严，如南山崔崔然。”

“雄狐”，喻齐襄公。

“绥绥”，行走缓慢之貌。《集传》言：“狐，邪魅之首；绥绥，求匹之貌”。本诗以南山雄狐喻荒淫的齐襄公。我们前已说过，诗经时代，常以狐，黑鸦喻男子。比如《卫风.有狐》就有“有狐绥绥”句。“有狐绥绥”本义就是“雄狐慢慢走”。在本诗中，暗喻齐襄公跟随文姜狎昵撩骚之态。陈奂《诗毛氏传疏》言：“绥绥然相随之貌，以喻襄公之随文姜”。

6、“鲁道”，齐国通往鲁国的官道、大路。

“有荡”，就是“荡荡”，平坦。《毛传》释“荡，平易也。”。

7、“齐子”，齐侯之子、之女。子，在诗经时代既可指男也可指女，似乎指女还稍多。本诗专指文姜。《毛传》：“齐子，文姜也。”

8、“由归”，从这里，从这条路上嫁出去，指文姜走这条路由齐国嫁到鲁国。《郑笺》：“妇人谓嫁曰归。”

9、“怀”，《毛传》释“思也”；《郑笺》释“来也。”我采前说。

“止”，语助词无实意。下文所有的“止”皆是。

10、屦，读音ju阴平。鞋子。葛屦，葛麻编的鞋，多为底层百姓穿着。《毛传》：“葛屦，服之贱者”。

11、“五两”，“五”通“伍”，配伍、配对。“两”，成双。都是数词当动词用。“五两”就是“五而成两”或“五而两之”。

12、“緌”，读音rui阳平，帽上的缨带。“顺颐而下结之谓缨，缨之垂者谓緌”。“双”，打成双结。

《毛传》：“冠緌，服之尊者”
“葛屦五两、冠緌双止”，喻人们不论贵贱都各有配偶。

但《郑笺》又是另一种解读，把“五”就当作数词，“两”就是量词，双。“葛屦五两”，就指五双鞋，喻文姜和她的姪、娣、傅母、保姆五个人同处。而冠緌与葛屦不宜同处，喻襄公、文姜不宜为夫妇之道。

这也是一种解读，可参考。

13、“庸”，由，或通“用”。

14、“艺”，种植。

15、“衡从”，通“衡纵”，就是“纵横”。“亩”，田亩，田垄。
“东西耕曰横，南北耕曰由”。“从”，《韩诗》正作“由”。

《郑笺》：种麻要先耕治其田，喻人君取妻，必先跟父母商议。

“必告父母”的“告”，《毛传》释：“告父母庙”。什么意思呢？《郑笺》释：取妻之礼，议于生者，卜于死者，此之谓告。取妻这样的大事，父母在世，就要跟父母商量；不在世，就要问卜于父母庙，祷告父母在天之灵的认可。

16、“鞠”，放纵、放任无约束。跟下文中的“极”义近。
《毛传》言：“鞠，穷也”。我理解应该是“穷尽其欲”的意思。《集传》也说：“又曷为使之得穷其欲而至此哉？”拿今天的话来说，就是“放任他的欲望而不加约束”。史载：鲁桓公对文姜放任，终招杀身之祸。清代诗经大家方玉润《诗经原始》亦言：“（鲁桓公）又曷从其入齐，至今得穷其所欲而无止极，自取杀身祸乎？”。书香君亦然之。

17、“析”，分开、劈开。“析薪”就是劈开柴禾。“析薪”，古时有婚姻意象。

18、“克”，能够。
“得”，得到，办成事，此处指“娶到家”。
“极”，穷，穷极，或“至”。
