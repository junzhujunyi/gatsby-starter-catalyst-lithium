这首诗历代有多种解读。我愿意把它看成一首动人而奔放的情歌。周时代中国人的爱情跟古代西方人的爱情一样，绚烂多姿且开放。

## 🌼 《鄘风∘桑中》原文：
```

爰采唐（dang）矣，
沬之乡（xiang）矣。
云谁之思？美孟姜（kiang）矣。
期我乎桑中（tiuəm），
要我乎上宫（kiuəm），
送我乎淇之上（zjiang）矣。

```
*书香君👉🏻：* 

> 首章两韵。本章是一种新的押韵方式。其中，一二三六句“唐、乡、姜、上”，阳部押韵；四五句“中、宫”，侵部，押韵。正如我前面已经介绍过的，我们或可猜测，阳部是一个人或一个声部在唱，侵部是另一个人或另一个声部在唱。也许是当时演出时的一种对歌或和歌的演唱方式。

```
爰采麦（muək）矣，
沬之北（pək）矣。
云谁之思？美孟弋（jiək）矣。
期我乎桑中（tiuəm），
要我乎上宫（kiuəm），
送我乎淇之上（zjiang）矣。

```
*书香君👉🏻：* 

> 本章三韵。一二三句“麦、北、弋”，职部押韵；四五句“中、宫”，侵部，押韵；末句“上”，阳部，与首章末句遥韵。并且，四五六句与首章四五六句重章。

```
爰采葑（piong）矣，
沬之东（dong）矣。
云谁之思？美孟庸（jiong）矣。
期我乎桑中，
要我乎上宫，
送我乎淇之上矣。

```
*书香君👉🏻：* 

> 末章与二章一样，也是三韵。只不过一二三句“葑、东、庸”，是东部。

```
另外，需要说明的是，古时，本诗每章是被分为七句的——也就是第三句分为三、四两句（“云谁之思”和“美孟姜矣”各作一句），那么第三句就有一个韵脚问题不好解决。王力先生分为每章六句，较好地安排了韵脚，我个人觉得是合理的。

```
## 🍁️ 骏注：

1、这首诗很有意思。旧说最早认为是刺诗，直到清代才开始有人把它当做爱情诗。另外，诗中的女子究竟是一个人还是三个人，各有不同理解。书香君个人意见：若理解成刺诗，则说三人更合理；若当做爱情诗，则应是一人。我倾向于后者。且让我一一道来。

2、《毛诗序》：“《桑中》，刺奔也。卫之公室淫乱，男女相奔。至于世族在位，相窃妻妾，期于幽远。政散民流而不可止。”说是“刺奔”诗，是讽刺贵族男女偷情的。

3、朱熹《诗集传》亦主毛序说。但是朱熹认为既为刺人之诗，不应该用“期我、要我、送我”这样的词，所以应该是偷情男子自作此诗。关于作者这一点，他与毛序有异。

4、方玉润是认同毛序的，他的《桑中》篇的诗题解就是“刺淫也”。在作者的身份上，他认为朱熹的“作者自述说”是误读了本诗。方氏以为该诗是他人而非诗中之男子所作。诗中“唐、麦、葑”非一其采；“沬乡、沬北、沬东”非一其地；“孟姜、孟弋、孟庸”非一其人。而“期、要、送”则都在桑中（桑间）之“上宫”与“淇上”，不可能一人分身三时三地与三人期。所以赋诗者并非诗中之人，所赋之事也非赋者自为。作者不过代诗中人为辞。即便诗中之人事也未必确有完全吻合的例证，只是作者以此来刺当世确有的一种现象。刺淫之诗亦谓亡国之音，“桑间”与“濮上”古时经常并称于“亡国之音”中，《乐记》有言：“郑卫之音，乱世之音也，比于慢矣。桑间濮上之音，亡国之音也。其政散，其民流，诬上行私，而不可止也”。本诗中的“桑中”即有此意象。当时的卫国已经社会沦丧、国之将亡。圣人取以为戒故选之入《诗》（他当然以为是孔子删定《诗经》的，其实未必。我曾在小文《我眼中的诗经——浅谈诗经之古远丰美》中提到《诗经》非孔子定编的依据）。

5、清代崔述《读风偶识》明确指出本诗并非刺诗。崔述说：《桑中》一篇，但有叹美之意，绝无规戒之言。如果这样的诗都叫讽刺诗，那曹植《洛神赋》、李商隐《无题》
（“相见时难别亦难”那首）、韩偓《香奁集》不就更是刺淫诗了？

6、闻一多认为是情诗，他说：“《桑中》，思会时也。”

7、今人多认为诗中所谓“孟姜、孟弋、孟庸”，可能就是诗中所美之人的代指，即如《郑风.有女同车》中的“孟姜”、《陈风.衡门》里的“齐姜、宋子”，以及后世以“西施”代称美女。诗中女子应该是同一个人。

不过，当代仍然有学者认为是作者想象和三个（或多个）贵族女子幽会的诗，如骆玉明。也有学者据此（三个女子与诗中之“我”约会）以为本诗有“男子炫耀魅力”之意味。

8、爰，于何处，在哪里。闻一多解“爰”乃“于、焉”之合音。于焉，就是“在哪”。

9、唐，女萝，俗称菟丝子。一说是“棠”的通假，一种梨。皆可。

10、沬，mei去声。不要看成“沫”喔。卫国的都城“朝歌”，也就是商朝时著名的“妹邦”、“牧野”（“沬”，《说文》作“坶”）。 牧野这名字以后还会多次出现在《诗经》的其他篇章中。沬乡，就是朝歌，就是邶鄘卫合并之前之“卫”地。下文的沬北就是沬之北，就是邶国之地。沬东，就是沬之东，就是鄘国之地。

11、谁之思，思谁，思念谁，思念的是谁。

12、孟，排行老大。孟姜，姜家的大姑娘、大小姐。孟姜这一结构词也多次出现于《诗经》。《有女同车》篇中就有“彼美孟姜”，“孟姜”后世逐渐成为美女的代指。（孟姜女哭倒长城中的“孟姜”是不是也是这种用法呢？我把它作为一个题目放在这。）本诗中的姜，弋，庸，皆是古代贵族姓氏，代指、泛指大家闺秀、美女。

“弋”，《春秋》作“姒”，《公羊传》、《穀梁传》作“弋”，杞女夏后氏之后人，贵族。（周朝开朝之初曾为尧、舜后人及夏、商两朝王室后裔封国以继其嗣。夏朝后裔封于杞国——书香君注）

《说文》无“姒”字，作“以”，而“弋”与“以”一声之转（因音同而借用），故有时又作“弋”。

“庸”，即“鄘”，是鄘地之人以旧邑之名为族姓而来。

王先谦《诗三家义集疏》：《地理志》鄘作庸，孟庸即孟鄘。鄘在沬东，居此之人，取旧邑之名以为族。

但朱熹认为“庸”到底是什么意思没听说过（“未闻”），猜测“庸”也是贵族。

姜，卫国本无姜姓，本诗，孟姜指姜姓列国（如齐、许、申、吕）之长女。抑或泛指贵族之长女。《郑笺》：“思美孟姜，思与淫乱，疾世族在位有是恶行也。”

13、桑中，地名，一说桑林之中，同“桑间”。

14、要，通“邀”。一说犹“迎”（朱熹、方玉润均解为“迎”）。吾意解为“迎”更贴切，“邀”本身暗含有“迎”的意味。

15、上宫，宫室，宫楼，祠堂，祠庙。有说就是古时宫室的通称，就是“楼”。一说地名。马瑞辰《毛诗传笺通释》：“桑中为地名，则上宫宜为室名。孟子之滕，馆于上宫。上宫，楼也。”

朱熹：“桑中、上宫、淇上，又妹乡之中小地名也。”也通。

16、淇，淇河。淇上，应为地名或淇河某一渡口。陈奂《传疏》释：“淇上，即淇水口也，黎阳淇口（今河南浚县东北）”。

17、葑，feng阴平，蔓菁。即现在的萝卜。

18、如前所述，这首诗历代有多种解读。我个人愿意把它看成一首动人而奔放的情歌！我无意更无权评判古人的爱情观，我只能说，周时代中国人的爱情跟古代西方人的爱情一样的绚烂多姿且开放。

也因为，诗中女子“姜、弋、庸”可能只是代指或泛指，本诗被后世文人尊为“无题诗”之祖。

方玉润之言诸君或可一阅，以窥前人如何说诗，书香君摘录如下：

诗中人亦非真有其人，真有其事，特赋诗人虚想。所采之物，不外乎唐、麦、葑；所游之地，不外乎沬乡、沬北、沬东；即所思之人亦不外此姜、弋、庸。而此姜、弋、庸，则尚在神灵恍惚、梦想依稀之际。即所谓期我、要我、送我，又岂真姗姗
其来，冉冉而逝乎？此后世所谓无题诗也。音由心生，诗随时变。诗与风为转移，时因心为隆替。闻其音而知政治之得失，读其诗而知卫国之将亡。古来亡国之音，桑间与濮上动辄并称，虽未必专指此诗，而此诗亦其类也。

> 我的心里想着谁？
> 
> 她的名字叫孟姜。
> 
> 约我桑林诉柔肠，
> 
> 邀我欢会祠庙上，
> 
> 依依别我淇河旁。

> 收割麦子在哪里？
> 
> 就在牧野北边地。
> 
> 我的心里念着谁？
> 
> 她的名字叫孟弋。
> 
> 约我桑林言爱意，
> 
> 与我欢笑在庙宇，
> 
> 别我在那淇河堤。

> 采摘芜菁哪田垄？
> 
> 就在牧野城之东。
> 
> 我的心里掂着谁？
> 
> 她的名字叫孟庸。
> 
> 约我桑林表衷情，
> 
> 与我欢娱在上宫，
> 
> 淇河之畔两相送！
