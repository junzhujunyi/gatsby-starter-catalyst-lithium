本诗关键词：发夕、岂弟。

本诗较多地采用了叠字，如：薄薄，济济，濔濔，汤汤，彭彭，滔滔，儦儦。使状物摹声，愈发生动。

刘勰《文心雕龙》也曾评论《诗经》中叠字的功效，我们不妨体会。

“灼灼”状桃花之鲜，
“依依”尽杨柳之貌，
“杲杲”为出日之容，
“瀌瀌”拟雨雪之状，
“喈喈”逐黄鸟之声，
“喓喓”学草虫之韵。

## 🌼 《载驱》原文：
```

载驱薄薄（bak），
簟茀朱鞹（khuak）；
鲁道有荡，齐子发夕（zyak）。

```
*书香君👉🏻：* 

> “薄、鞹、夕”，铎部，押韵

```
四骊济济（tzyei），
垂辔濔濔（nyei）；
鲁道有荡，齐子岂弟（dyei）。

```
*书香君👉🏻：* 

> “济、濔、弟”，脂部，押韵

```
汶水汤汤（sjiang），
行人彭彭（beang）；
鲁道有荡（dang），
齐子翱翔（ziang）。

```
*书香君👉🏻：* 

> “汤、彭、荡、翔”，阳部，押韵

```
汶水滔滔（thu），
行人儦儦（pio）；
鲁道有荡，齐子游敖（ngo）。

```
*书香君👉🏻：* 

> “滔、儦，敖”，幽宵合韵

> 本诗四章，每章一韵。

> “濔”，当代有些版本作“瀰”，我觉得似乎不妥，后面会说。

> “敖”，当代有些版本作“遨”。

## 🍁️ 骏注：

1、本诗诗旨旧说有二：
其一，认为是讽刺文姜和齐襄公的。他们华车招摇于途，行其不伦之恋。只不过，是主刺文姜还是主刺襄公？“簟茀朱鞹”这样的华车是襄公之车从齐到鲁，还是文姜之车从鲁到齐？这两点说法不一，但不影响诗旨解读。认同此解者历代为多。

其二，认为是写齐襄公嫁小女儿哀姜给鲁庄公的诗。哀姜一路上跟鲁庄公谈条件，迟迟不入鲁境。这种解读也有相当道理，《齐诗》、王先谦《诗三家义集疏》及今人程俊英/蒋见元《诗经注析》就持此说，我后面也会再细讲。

2、《毛诗序》：“《载驱》，齐人刺襄公也。无礼义故，盛其车服，疾驱于通道大都，与文姜淫，播其恶于万民焉。”（说的是刺襄公）

3、《郑笺》：诸侯之路车，有朱革之质而羽饰，襄公乘之而来与文姜会。襄公既无礼义，乃疾驱其乘车以入鲁竟（竟，境也。——书香君注），徒为淫乱之行。鲁之道路平易，文姜发夕由之往会焉，曾无慙（慙，读音can阳平，同“惭”。——书香君注）耻之色。汶水之上，盖有都焉，襄公与文姜时所会。（说的是襄公之车。因革质羽饰之车是诸侯路车，所以旧说认为是襄公所乘，是刺襄公的。其实诸侯夫人也是可以乘坐此种车的。——书香君注）

3、《诗集传》：“齐人刺文姜乘此车（簟茀朱鞹）而来会襄公也。岂弟，乐易也。言无忌惮羞耻之意也。彭彭，多貌。言行人之多，亦以见其无耻也。”（说的是刺文姜，是文姜之车）

4、《诗经原始》：“《载驱》，刺文姜如齐无忌也。此诗以专刺文姜为主，不必牵涉襄公，而襄公之恶自不可掩。夫人之疾驱夕发以如齐者，果谁为乎？为襄公也。夫人为襄公而如齐，则刺夫人即以刺襄公。”

方氏列举了《春秋》所载的文姜与齐襄公私会记录——鲁庄公二年，夫人姜氏会齐侯于禚（禚，读音zhuo阳平，鲁国地名——书香君注）；四年，夫人姜氏享齐侯于祝丘；五年，夫人姜氏如齐师；七年，夫人姜氏会齐侯于防；又，会齐侯于穀。——来证明文姜到齐国（会襄公）已到肆无忌惮的程度。他认为诗中的“发夕、岂弟、翱翔、游敖”就是描述文姜、襄公二人在鲁庄公时期，偷情已经乐而忘返、翱翔远游，宣淫于通道大都，不顾行人讪笑，岂尚知人间有羞耻事哉？他说汶水上的文姜台与卫之新台可以并臭千古。《邶风.新台》可参我2021年1月26号朋友圈。

5、《齐诗》（三家诗之齐说）认为本诗说的是齐公主哀姜（齐襄公小女儿）嫁鲁庄公的事。

王先谦认同齐说。其《诗三家义集疏》言：“襄齐嫁女，至于荡道。齐子旦夕，留连久处。《毛序》以为刺襄公，非也。”
王先谦分析了齐说、《春秋经》（今称《春秋》）、《公羊传》（《春秋公羊传》）以及何休的注，得出他的上述结论。

*书香君👉🏻：* 

> 分析史料，哀姜应是齐襄公即位后所生，当是齐襄公的小女儿。她嫁鲁庄公时，齐襄公已死，齐襄公弟弟齐桓公（就是公子小白）在位。哀姜是齐桓公的侄女，那时不到二十岁。而鲁庄公已经三十六岁左右，在娶哀姜之前曾与党氏之女孟任生了一个儿子公子般。鲁庄公去齐国迎娶哀姜时，哀姜迟迟不肯入鲁境，就是在跟鲁庄公谈条件，主要一点就是要求鲁庄公婚后要“远媵妾”，以确保自己以及将来儿子的接班地位，也许是孟任及公子般的存在让她不安吧。果不其然，鲁庄公的接班问题后来搞出很大动静。哀姜婚后不育，陪嫁的媵妾叔姜给鲁庄公生了一个儿子叫启发，哀姜收为己子（后来即位为鲁闵公）。庆父、哀姜要立启发，鲁庄公、叔牙要立公子般。庆父，大家都知道，鲁庄公的弟弟之一，成语“庆父不死，鲁难未已”让他后世留名。哀姜还跟庆父这个小叔子有染。我们之前说过卫宣公、宣姜、文姜的乱伦事，说句玩笑话，乱伦这事，春秋王室是认真的。
这首诗如果解读为哀姜与鲁庄公的婚事，也是有历史背景支撑的。

6、载，发语词，可理解为“乃”，如载笑载言，载奔载驰等等，经常的用法。

7、薄薄，象声词，马蹄声，车轮转动声。
《毛传》《诗集传》《诗经原始》皆释：“薄薄，疾驱声也。”

8、“簟茀朱鞹”，这句难一点。
簟，读音dian，去声。方形条纹的竹席。可做车门处的挡风帘。
《毛传》：“簟，方文席也。”
茀，读音fu，阳平。野鸡羽毛编织的帘或画饰之类，可做车两侧或后面的帘。
《毛传》：“车之蔽曰茀。”
《诗集传》《诗经原始》：“茀，车后户也。”
鞹，读音kuo，去声。光滑的兽皮。
朱鞹，就是红漆染的兽皮，蒙在车前门处，还配上雉羽，做车饰。此种车也称“路”、“辂”、“路车”、“辂车”，是君王、诸侯和少数贵族所乘用的车，诸侯夫人称“小君”，也是可以乘坐的。

9、发夕，这个词，历代解读各种不同。《毛传》解为：“自夕发至旦”。有的解为“发——旦，夕——暮，发夕就是旦暮，“犹言朝见暮见，即久处之意”（王先谦）；有的解为“天将明而日未出”（马瑞辰）。
《诗集传》释：“夕，犹宿也。发夕，谓离于所宿之舍。”

我倾向于“发夕”就是“夕发（骆玉明）”。跟现在的用法一样，如夕发朝至。

当然，古代诗作中的某个词，在作品被创作出的那一刻，一定是有明确的词意的。后世解读诗旨时，不同的理解则会选择该词的不同义项，这也是很自然的事。有些字词的释意，真的不好说对错。

10、骊，黑马。

11、济济，美好。（我们是否要重新考虑“人才济济”的原始意思？）
一说济济即“齐齐”，齐整，喻四马步调一致。

12、濔，读音ni，上声，柔软的样子。一说“众”
《毛传》：“濔濔，众也。”
《诗集传》：“濔濔，柔貌。”
濔是“”
当代有些版本作“瀰”，我认为不妥，因为濔、瀰字意并不重叠。

13、齐子，齐国公主。这里指齐姜（另一种解读就指哀姜）。瞧，齐姜这个争议美女又出现了。

15、岂弟，读音要注意，岂，读音kai上声。弟，读音ti去声。这个词是本诗第二大难点，也有各种解读。
有说“（天亮）才动身”的意思（陈乔枞、王先谦、《尔雅》）；
有说“欢乐”（骆玉明）；
有说“乐易也”（朱熹）；
有说“跟发夕是相近的意思”（郑玄）；
有说《鲁诗》为“齐子闿圛”，闿圛（圛，读音yi去声），发也。

一个词为什么会有多种毫不搭界的词义呢？因为历代学者把字词作了不同的假借，又参考不同的古籍作不同的理解推演，从而得出五花八门的各种解读，你还不能说谁一定是对的谁又一定是错的，总不能找原作者来问吧？所以，对于某些太难搞明白的，我们有个大概了解就好，可以根据自己对诗意的把握来选择。

我采纳“欢乐”说。

16、彭彭，众多的样子。行人，路人。有说是指文姜的随从。

17、“翱翔”，遨游，游逛，跟今天的意思不同。以前解释过。
《毛传》：“翱翔犹彷徉”。近似今天“徜徉”的意思。

18、儦儦，biao阴平，行人来来往往的样子。
《毛传》：“儦儦，众貌。”

19、敖，当代有些版本作“遨”。“敖”是“遨”的古写。
游敖，就是游遨、遨游，与上文“翱翔”意近。

20、我按照刺文姜和襄公的解读来译。当然大家可以有其他的理解。
