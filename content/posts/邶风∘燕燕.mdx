万古送别诗之祖，
足可以泣鬼神也。

诗中“瞻望弗及，泣涕如雨”，已成千古名句，被后世文豪反复化用。

## 🌼 《邶风∘燕燕》原文：
```

燕燕于飞（piuəi），
差池其羽（hiua）；
之子于归（kiuəi），
远送于野（jya）。
瞻望弗及，泣涕如雨（hiua）。

```
*书香君👉🏻：* 

> 本章两韵。一三句“飞、归”，微部；二四五句“羽、野、雨”，鱼部，押韵。

```
燕燕于飞（piuəi），
颉之颃（hang）之；
之子于归（kiuəi），
远于将（tziang）之。
瞻望弗及（giəp），
伫立以泣（khiəp）。

```
*书香君👉🏻：* 

> 本章三韵。一三句“飞、归”，微部；二四句“颃、将”，阳部；五六句“及、泣”，缉部，押韵。

```
燕燕于飞（piuəi），
下上其音（iəm）；
之子于归（kiuəi），
远送于南（nəm）。
瞻望弗及，实劳我心（siəm）。

（本章两韵。一三句“飞、归”微部；二四五句“音、南、心”侵部，押韵。

大家应该也看出来了，本诗前三章的一三句完全相同，是谓重章。）

仲氏任只，其心塞渊（yien）。
终温且惠，淑慎其身（sjien）。
先君之思，以勖寡人（njien）。

```
*书香君👉🏻：* 

> 末章一韵。“渊、身、人”，真部，押韵。

## 🍁️ 骏注：

1、关于本诗的诗旨以及诗中的送人者和被送者究竟是什么人、什么关系，历代众说纷纭、莫衷一是。主要有三种观点。

其一、庄姜送别戴妫（妫，读音gui阴平）。

其二、定姜送别儿媳。

其三、某一任卫君送别其二妹出嫁。

前两种观点的提出，源于历史上有对应的故事。第三种观点则更多地考虑了诗的文字本身，并在推理判断的基础上得出较为合理的结论。让我一一介绍。目前第一、三两种观点认同者较多，书香君个人从文学角度出发，倾向于第三种观点。

导致诗旨分歧的其中一个关键问题就是，“于归”一词，究竟解为“回娘家”还是“出嫁”？所谓“大归”是否可称“于归”？

2、《毛诗序》：“《燕燕》，卫庄姜送归妾也。庄姜无子，陈女戴妫生子名完，庄姜以为己子。庄公薨，完立，而州吁杀之，戴妫于是大归。庄姜远送之于野，作诗见己志。”

《毛序》此一说法采信者众。《郑笺》、《孔疏》、《诗集传》、《诗经原始》、骆玉明等均从之。

据《左传·隐公》记载，卫庄公夫人庄姜（卫姜）无子，卫庄公又娶了陈国的一对姐妹厉妫、戴妫为妾。戴妫生子名“完”（就是公子完），过继给庄姜并被立为太子。卫庄公死，“公子完”即位称卫桓公，后来被卫庄公另一个庶子“州吁”所杀。戴妫因儿子被杀无嗣又非正妻，被遣送回娘家陈国。这种带有“休妻”性质的遣送称为“大归”，即“归而不再回”（以上学者把诗中的“于归”解为“大归”，而另一部分持不同观点的学者则认为“未闻有称大归为于归者”）。庄姜因与戴妫同养儿子，相处感情深厚，送别戴妫时作此诗。那诗中的“先君”就是“卫庄公”。

还有学者认为庄姜因为长期失宠，情绪寥落，与戴妫逐渐发展出了同性恋情。

3、清代诗经大家王先谦《诗三家义集疏》援引《鲁诗》说，认为写的是卫定公的夫人定姜。定姜的儿子去世，儿媳没有子女，服丧三年后，定姜把她送回娘家。婆媳感情深厚，临别时定姜写了这首诗。

《列女传.母仪篇》也持此说。

这种说法就把“仲氏”解为“仲家女子”，媳妇的娘家姓“仲”。但是，即便这种解释能通，我仍怀疑，既然婆媳感情深厚，是可以不送媳妇走的呀，定姜夫人并没有失位，养这个媳妇是很容易做到的事，并不需要送走。不象第一种说法，因为戴妫的儿子（庄姜的养子）卫桓公被篡位的“州吁”所杀，两者是政治死敌，死敌的生母被赶，那是不得不走的。

4、另有说法认为此诗写卫国国君送其二妹远嫁。因为字面看，“于归”更多地解为“出嫁”。“仲氏”就是老二，女性排行第二，那就是二妹（或二姐）。文中的“先君之思”，是兄妹二人的父亲（先君）已经不在人世了，两人常一起回忆或者相互提醒父亲的治国教诲。而“寡人”，是作者的口吻，当然古时“寡人”可以是诸侯自称，也可以是诸侯夫人（所谓小君）自称。但是如果作者是庄姜，她失宠很多年也不会有什么权力了，自称“寡人”可能性并不大。而如果是卫君自称则非常合适。设想一下，同胞的二妹（也许嫡出的就他们二人，感情深厚）就要嫁到外国，一路相送，一路叮咛。哥哥嘱咐妹妹在他国要多保重；妹妹回忆父亲（诗中的“先君”）的教诲勉励哥哥好好治国。两人从此天各一方又各担大任（春秋时各诸侯国公主选嫁某国多有政治考量），离别无靠，加上政治压力，心情的悲伤寥落可以想见。

崔述《读风偶识》、程俊英《诗经注析》即持此说。

5、我倾向于认可兄妹送别。哥哥是卫国某一任国君，其二妹将要远嫁，哥哥送别妹妹。

6、这首诗是古代最早的送别之作。被历代《诗经》大家评说为万古送别之祖。其艺术感染力可以泣鬼神也。前三节用了《诗经》中常用的复沓形式，重章叠唱，循序渐进，把惜别之深情表现得哀婉沉痛。后世许多大诗人如李白、苏轼都化用了此诗的用句或意境，创作了美好深情的送别诗。

诗中“瞻望弗及，泣涕如雨”一句被历代反复化用——

李白“孤帆远影碧空尽，唯见长江天际流。”

苏轼“登高回首坡陇隔，惟见乌帽出复没。”

张先“一帆秋色共云遥；眼力不知人远，上江桥。”等等。

所以我一直说，《诗经》是文学的瑰宝和字、词、典的母库！

7、“差池”，通参差，形容燕子翅膀舒展的样子。“差”也要读ci阴平。

8、“颉”，读音jie 阳平。上飞、往上飞。

9、“颃”，读音hang 阳平。下飞、往下飞。

10、“将”，送。

11、“仲”，排行第二。针对本诗，被送的是位女性，可以解为“二妹”。“仲”的其他解释参见上面骏注第4条。

12、“只”，语助词。无实意。之前其他诗篇中我们已经介绍过。

13、“任”，信任、诚信。闻一多先生认为是任国国君送妹妹嫁卫国。但，彼时任国在卫国之南，从任国到卫国，应该是往北，但诗中说的是“远送于南”，那正好反方向。所以闻一多先生之说也未必妥。还有说“任”是卫君的姓的，则显然不妥。卫君姬姓。周公平定三监叛乱后，将邶鄘卫三国合并成新的卫国，封其弟康叔做新卫国的首任卫君，后世卫君应该姓姬。所以我以为“任”，就解为“信任、诚信”。或者卫君送妹妹嫁到任国，倒说得通。

14、“塞”，秉性诚实。

15、“勖”，读音xu去声。勉励。
