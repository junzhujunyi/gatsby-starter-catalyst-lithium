孤臣远迈，怅望何之？游子无家，去将焉往？山有蕨薇，隰有杞桋。
君子作歌，维以告哀。



## 🌼 《小雅∘四月》原文：

```
四月维夏（hea），
六月徂暑（sjia）。
先祖匪人，胡宁忍予（jia）？
```

*书香君👉🏻：*

> “夏，暑，予”，鱼部a，押韵

```
秋日凄凄（tsyei），
百卉具腓（biuəi）。
乱离瘼矣，爰其适归（kiuəi）？
```

*书香君👉🏻：*

> “凄，腓，归”，脂（ei）微（əi）合韵，押韵

```
冬日烈烈（liat），
飘风发发（piuat）。
民莫不穀，我独何害（hat）？
```

*书香君👉🏻：*

> “烈，发，害”，月部at，押韵

```
山有嘉卉，侯栗侯梅（muə）。
废为残贼，莫知其尤（hiuə）。
```

*书香君👉🏻：*

> “梅，尤”，之部ə，押韵

```
相彼泉水，载清载浊（deok）。
我日构祸，曷云能穀（kok）？
```

*书香君👉🏻：*

> “浊，穀”，屋部ok，押韵

```
滔滔江汉，南国之纪（kiə）。
尽瘁以仕（dzhiə），
宁莫我有（hiuə）？
```

*书香君👉🏻：*

> “纪，仕，有”，之部ə，押韵

```
匪鹑匪鸢，翰飞戾天（tyen）。
匪鳣匪鲔，潜逃于渊（yuen）。
```

*书香君👉🏻：*

> “天，渊”，真部en，押韵

```
山有蕨薇（miuəi），
隰有杞桋（jiei）。
君子作歌，维以告哀（əi）。
【“薇，桋，哀”，微（əi）脂（ei）合韵，押韵）
```

## 🍁️ 骏注：

1、这是一首周时逐臣南迁之词。当为后世“迁谪诗”之鼻祖。屈原也深受此诗的影响，我们在他的作品《离骚》、《渔父》中都能明显感受到。

2、《四月》八章，以遭祸被逐之臣的口吻，诉说自己作为名门之后、可为“南国之纪”的“滔滔江汉”，却被放逐江南，受难无归。（“乱离瘼矣，爰其适归”）。描述了自四月初夏离京去国，颠沛流离，经六月酷暑再到凄凄秋时、烈烈冬日，历三季方抵贬谪之所。自诩独清于浊流（“相彼泉水，载清载浊”），自嘲既非雕鹰，高翔于天可避弓箭；也非鳣鲔，深潜于渊可躲罹网（“匪鹑匪鸢，翰飞戾天；匪鳣匪鲔，潜逃于渊”）。无可奈何之下，唯有以诗诉说自己的悲哀（“君子作歌，维以告哀）。

3、历代解读。

A、《毛诗序》：“《四月》，大夫刺幽王也。在位贪残，下国构祸，怨乱并兴焉。”

B、《郑笺》：“人为恶亦有渐，非一朝一夕。贪残之政行而万民困病。尤病之祸，必自之归为乱。王为酷烈惨毒之政，如冬日之烈；其亟急行于天下，如飘风之疾。上多赋敛，富人财尽，而弱民与受困穷，诸侯并为恶，曾无一善。王尽病其封畿之内以兵役之事，使群臣有土地曾无自保有者，皆惧于危亡。民性安土重迁，今而逃走，亦畏乱政。草木生各得其所，人反不得其所，劳病而愬之也。”

C、《诗集传》：“此亦遭乱自伤之诗。言四月维夏，则六月徂暑矣。我先祖岂非人乎，何忍使我遭此祸也？无所归咎之词也。”

D、明代学人朱善《诗经解颐》：“此诗专以为行役，则’先祖匪人’之怨，其辞过于深；专以为忧乱，则’滔滔江汉’之詠，其辞过于远。然则是诗也，盖大夫行役而忧时之乱惧其祸之辞也。”

E、姚际恒《诗经通论》：“此疑大夫之后为仕者遭小人构祸，身历南国，而叹其无所容身也。”

F、方玉润批驳毛序“割裂诗体，杂凑成言，前后文意，竟不能通。”

批评朱熹的“遭乱”说也离谱，“因遭乱而怨及祖宗非人，天下岂有如是子孙？”

认为朱善和姚际恒的说法接近诗旨但不够全面，“盖庶几焉，而未尽也。”

G、其《诗经原始》洋洋洒洒解道：“《四月》，逐臣南迁也。此诗明明逐臣南迁之词，而诸家所解，或主遭乱，或主行役，或主构祸，或主思祭，皆未尝即全诗而一诵之也。头绪既纷，不知所从。…愚谓当时大夫，必有功臣后裔，遭害被逐，远谪江滨者，故于去国之日作诗以志哀云。冒暑远征，人情所难；今遭放废，适当其厄，岂得已哉！然予虽获罪，而先人恒有功。论贵论功之典行，亦当宽宥而矜全之；何朝廷不齿我祖于人，而独忍加罪于予耶？故自夏徂秋，由秋而冬，历时三序，始抵南国。则见江汉交流，滔滔不断，有条有理；二渎纳川而注之海，民得赖以安居；是南国之有江汉，南国之事皆可坐而就理矣。独予尽瘁王室，而王终不我知，而我有者何哉？…孤臣远迈，怅望何之？游子无家，去将焉往？回忆来时，景象频更，…不觉怆然泣下。…夫是以作为歌诗，用诉衷情，亦聊以抒吾郁积之气而已矣。”

4、“四月”、“六月”，均指夏历（农历）。

5、“徂”，读音cu，阳平。往。

“徂暑”，到了盛夏酷暑天。

《郑笺》：“四月立夏矣，至六月乃始盛暑。”

6、“匪人”，非他人。

7、“先祖”，应该是指周朝开朝有功之臣，而不仅仅是说自己的先祖（这只是书香君个人的理解，与其他版本都不同，这样才能解释得通。仅供参考）。

“先祖匪人”，意思是开朝先祖不是旁人，而是自己的血亲。与下文“胡宁忍予”一起，点出自己乃名门之后，却被放逐，表达朝廷对自己不公的悲愤之情。

8、“胡宁”，为什么。

9、“忍予”，忍心让我（受难）。

10、“腓”，读音fei，阳平。“痱”的借字，病，枯萎。

“秋日凄凄，百卉具腓”两句，是说秋风使众草皆病，以此比兴贪残之政使万民困病。

11、“瘼”，读音mo，去声。疾苦，病、痛。

12、爰，句首语助词，无实意。可联系下文灵活翻译。

13、“适”，往，去到。“归”：归宿。回家。

14、“烈烈”，“冽冽”。

15、“飘风”，飙风，疾风。

16、“发发”，象声词，风呼啸声。发，要读作bo，阴平。

17、“榖”，读音gu，上声。善、好。

18、“何”，通“荷”，承受。要读去声。

19、“侯”，句首语助词，相当于“惟，惟有”。

20、“废” ，大。一说“习惯”。这个字是本诗难点之一。

《尔雅·释诂》解：“废，大也”。举的例子就是《小雅.四月》。

而《毛传》说：“废，忲也”。

“忲”也写作“忕”，读作tai去声。查词义是“奢侈”、“恶习”。（“忲”的“恶习”这个词义我想可能是从“奢侈”转义、引申而来。）

21、“残贼”，残害。一说“害虫”。这个词是本诗的第二个难点。

结合我们上面介绍的“废”和“残贼”这两个词的不同解读，第四章“山有嘉卉，侯栗侯梅。废为残贼，莫知其尤”可以对应翻译。无论采信哪种解释，对整个诗旨影响不大。

22、“尤”，过错，罪过。

23、“相”，看。要读阴平而非去声。

24、“构”，“遘”的借字，遇，遇到。

25、“江汉”，长江和汉水。

26、“南国”，指南方各河流。也暗指作者被贬谪至南方地区。

27、“纪”，纲纪。《诗集传》：“纪，纲纪也，谓经带包络之也。”

28、“滔滔江汉，南国之纪”，是说南方地区的大小河流最终都汇入长江和汉水而受江汉的把控制约，江汉就是这些大小河流的“纲”。

29、“有”，通“友”，友爱。此处应作动词用，且是意动用法，“我有”就是“以我为友”。

30、“鹑”，读作tuan，阳平。

31、雕。“鸢”，鹰。

32、“翰飞”，高飞。

33、“戾”，至也。

34、“桋”，读音yi，阳平。赤梀树。梀，读作su，去声。

35、“山有蕨薇，隰有杞桋”两句，说的是，连草木都各得其所，而人反而不得其所。伤之矣。

36、“君子”，作者自谓也。

37、“维”，是，只是。


## 🍁️ 骏译：


> 四月已是夏季天，
> 
> 六月酷暑又重现。
> 
> 先祖并非别家人，
> 
> 何忍使我受熬煎？
> 

> 秋日寒凉风凄冷，
> 
> 百花杀来百草零。
> 
> 忧伤离乱苦难诉，
> 
> 何时才能回家中？
> 

> 冬季到来寒气烈，
> 
> 狂风呼啸声不歇。
> 
> 众人生活都安康，
> 
> 唯我背祸独悲切。
> 

> 好树好花满山隈，
> 
> 既有栗树还有梅。
> 
> 无端受害遭蹂躏，
> 
> 草木究竟有何罪？
> 

> 泉水潺潺流不停，
> 
> 有时混浊有时清。
> 
> 我却天天遇灾祸，
> 
> 何时才能福上身？
> 

> 长江汉水波浪宽，
> 
> 南国百流汇大川。
> 
> 鞠躬尽瘁为朝廷，
> 
> 无人把我来夸奖。
> 

> 不是雕来不如鹰，
> 
> 避险高飞在天空。
> 
> 不如鲔鱼不是鳣，
> 
> 逃命可以潜深渊。
> 

> 山有蕨菜和薇菜，
> 
> 杞树桋树洼地栽。
> 
> 我今作诗把歌唱，
> 
> 满腔愁哀诉悲怀。
>
