何草不黄，何人不将。
哀我征夫，漫漫周行。

旷野之间，无非虎兕，幽草以内，尽是芃狐。

此何如荒凉景象乎？周衰至此，其亡岂能久待？所谓“亡国之音哀以思”也。



## 🌼 《何草不黄》原文：

```
何草不黄（huang）？
何日不行（heang）？
何人不将（tziang），
经营四方（piuang）？
```

*书香君👉🏻：*

> “黄，行，将，方”，阳部ang，押韵

```
何草不玄（hyuen）？
何人不矜（kien）？
哀我征夫，独为匪民（mien）。
```

*书香君👉🏻：*

> “玄，矜，民”，真部en，押韵

```
匪兕匪虎（xa），
率彼旷野（jya）。
哀我征夫（piua），
朝夕不暇（hea）！
```

*书香君👉🏻：*

> “虎，野，夫，暇”，鱼部a，押韵

```
有芃者狐（hua），
率彼幽草（tsu）；
有栈之车（kia），
行彼周道（du）。
```

*书香君👉🏻：*

> 注意，本章一三句“狐，车”，鱼部a，互押；二四句“草，道”，幽部u，互押。

```
按诗意一二连句、三四连句是合适的。按韵，我们似乎可以把本章排列成如下的样子：
有芃者狐，
有栈之车。
率彼幽草，
行彼周道。
这样就是一二句连句互押、三四句连句互押。完全可以，意思不变。为何《诗经》没有按韵脚连排而是韵脚交错排列呢？我还没看到有人分析。我猜当时是男女对歌或者二部轮唱？一方唱第一句，另一方接唱第二句相和。一方再唱第三句，另一方紧接着第四句相和。每一方所唱的两句都是自我押韵的，而整章的节奏既参差又和谐。手法高妙。
```

## 🍁️ 骏注：

1、《何草不黄》列在《小雅》最后一篇。方玉润认为这是编者的特意安排（事实上风篇亦然，如《周南》以《关雎》始而以《麟之趾》作结；召南以《鹊巢》始而以《驺虞》作结，都是有讲究的。这个话题以后有空再谈）。他说：“（《何草不黄》）纯是一种阴幽荒凉景象，写来可畏。所谓亡国之音哀以思，诗境至此，穷仄极矣，怨之至也！周衰至此，其亡岂能久待？编《诗》者以此奠《小雅》之终，亦《易》卦纯阴之象。”

本诗产生于周幽王时代，西周亡于猃狁，正是在幽王手上断送的。卦不卦的两说，起码已有末世之象。紧挨着本诗排在前面的《苕之华》已经出现“人相食”（参见我们昨天的分享），本诗也控诉了征夫被视为草芥、禽兽，过着非民生活的现实。人已不复为人，这难道还不是末世吗？

2、本篇是一首刺诗。它以征夫口吻倾诉役人劳苦无休，就像草木注定要枯萎腐烂，象野兽注定要出没于草丛，奔波于四方。

3、历代评说：

A、《毛诗序》：“《何草不黄》，下国刺幽王也。四夷交侵，中国背叛，用兵不息，视民如禽兽，君子忧之，故作是诗也。”

B、《郑笺》续解：“用兵不息，军旅自岁始草生而出，至岁晚矣，何草而不黄乎？言草皆黄也。从役者皆过时不得归，此岂非民乎？”

朱熹、方玉润及当代学者亦然序笺之说。

C、《诗集传》：“周室将亡，征役不息，行者苦之，故作此诗。言何草而不黄？何日而不行？何人而不将？以经营于四方也哉！”

D、《诗经原始》：“《何草不黄》，征夫恨也。此征伐不息，行者愁怨之诗，人皆知之矣。”

但方玉润认为，序笺及集传，对“匪兕匪虎，率彼旷野”的解读值得商榷——“唯兕虎二句，迄无定解”。方氏认为全诗都是“赋”体，而历代儒生却错以为是“兴”了。朱熹虽也说这段是赋，但又说前后几章是兴，诗这样的写法和朱熹这样的说法他觉得奇怪。

方玉润认为姚际恒是看清楚了这两句实际上是倒装句的——“匪兕匪虎，率彼旷野”，就是“率彼旷野，匪兕匪虎”。但姚际恒仍以为是兴，是“兴征夫朝夕在途”的，还是没有理解对。

方玉润指出，诸儒“皆泥两匪字而不得其解耳”。

他坚持认为：全诗皆赋，征役不息，终岁往来，以至“何草不黄”矣，然征夫犹未有已时，即至草色皆枯由黄而玄，征役仍如故也。且也旷野之间，无非虎兕，幽草以内，尽是芃狐，此何如荒凉景象乎？哀我征夫，朝夕不暇，乘此栈车，行彼周道，是虎兕芃狐相率而为群也，其幸而不至为恶兽所噬者，亦几希矣！嗟嗟，我征夫也，独非民哉？胡为遭此乱离，弃其室家，几至无人不鳏也哉？盖怨之至也。周衰至此，其亡岂能久待？编《诗》者以此殿《小雅》之终，亦《易》卦纯阴之象。《坤》上六曰：“龙战于野，其血玄黄。”其是之谓欤？观于《诗》，而世运之升降，人事之盛衰，可一览而识其故矣！纯是一种阴幽荒凉景象，写来可畏，所谓“亡国之音哀以思”也。诗境至此，穷仄极矣！

书香君：相较而言，方玉润的分析丝丝入微。

4、“行”，出行，行军，出征。读xing。与下文的“将”，义近。

“经营”，规划治理、当差办事。此处指行役、征战、操劳。《小雅.北山》有“旅力方刚，经营四方”句，也是类似意思。

5、“玄”，黑。发黑。指草腐烂发黑。

6、“矜”，通“鳏”，也宜读guan阴平。无妻者。喻指征夫长期离家，就如无妻之人。

“何草不黄”、“何草不玄”，说的是所有的草，最后都会枯黄腐烂变黄变黑。比兴征人无法自我把控命运，日日行役、“经营四方”。最终会象草一样枯萎腐烂。暗含着控诉：人非草木，我们生来也不是为了征役到死的！“何人不矜”，说的是所有的征夫因为长年行役有家难回，即便有家有妻子，也同鳏夫一样。以上种种，使诗者哀叹“独为匪民”。也暗含控诉。

7、“匪”，通“非”，宜读阴平而非上声。

8、“民”，要多解释一下。西周时所谓“民”，跟今天我们说的“人民”、“人”，意思还不完全相同。那时的“民”，比如士农工商，还是有一定地位的，民之下还有流民、奴隶。“匪民”跟我们现在说的“非人”意思不完全相同。现在的“民”基本可以泛指每个自然人。古时不是每个自然人都可以称“民”的。

9、“兕” ，读音si去声。野牛。我们在《豳风.七月》中介绍过“称彼兕觥”句。“匪兕匪虎，率彼旷野”的意思（之一）就是：我们又不是野牛老虎，为何却象这些野兽一样长年出没于旷野，朝夕不停？所以哀叹“独为匪民”——只有我们征夫不能象普通居民一样正常生活。

意思之二，就是骏注3，“D”项下面方玉润所作的分析。

10、“率”，循，沿，沿着。

11、“芃 ”，原意是草木茂盛。此代指兽毛蓬松。“有芃”即“芃芃”。有，起加强它后面的字的作用。

12、“幽”，深也。

13、“栈”，通“嶘”，高也。“有栈”就是“栈栈”、“高高的样子”。与上文介绍的“有芃”用法一样。

14、“车”，指役车。

“有栈之车”就是“高高的役车”。

15、“周道”，大路。
