崔述《读风偶识》评曰：卫之重封，由於齐桓。齐桓所封者，邢与卫也。然邢仅二十馀年而遂亡，而卫历春秋及战国秦又数百年而始亡，何哉？吾读《干旄》之篇而知卫之所以久存良有由也。

## 🌼 《鄘风∘干旄》原文：
```

孑孑干旄（mo），
在浚之郊（keo）。
素丝纰（biei）之，
良马四（siet）之。
彼姝者子，何以畀（piet）之？

```
*书香君👉🏻：* 

> 首章两韵。“旄、郊”，宵部，押韵；“纰、四、畀”，脂质通韵

```
孑孑干旟（jia），
在浚之都（ta）。
素丝组（tza）之，
良马五（nga）之。
彼姝者子，何以予（jia）之？

```
*书香君👉🏻：* 

> 二章一韵。“旟、都、组、五、予”，鱼部，押韵

```
孑孑干旌（tzieng），
在浚之城（zjieng）。
素丝祝（tjiuk）之，
良马六（liuk）之。
彼姝者子，何以告（kuk）之？

```
*书香君👉🏻：* 

> 末章两韵。“旌、城”，耕部，押韵；“祝、六、告”，觉部，押韵

## 🍁️ 骏注：

1、本诗的诗旨，自《毛诗序》至当代，有多达十几种不同解读，但主要的说法可归纳为三大类。

一、认为是赞美卫文公和他的大臣们礼遇贤才以期为卫所用的。

《毛诗序》：“《干旄》，美好善也。卫文公臣子多好善，贤者乐告以善道也。”就是说，卫文公的大臣们大多喜好听取那些对国家有益的好主张、好建议，所以民间的贤者们都乐意把自己好的想法反馈给当政者。《毛序》明确指出是卫文公时期的事。

朱熹认为是卫臣到民间访求贤才的。他和清代学者姚际恒都认为本诗是卫文公年代作品的说法，无考据证实。

《诗集传》：“卫本以淫乱无礼、不乐善道而亡其国，今破灭之余，人心危惧，正其有以惩创往事，而兴起善端之时也，故其为诗如此。盖所谓生于忧患死于安乐者。”

方玉润认为卫国曾因国君淫乱无礼、不乐善道而遭灭国之灾。故，卫人痛定思痛、汲取教训而作此诗，“美好善也”。他认为史称卫文公敬教勤学、授方任能，即以此诗属之亦无不宜，属合理推断。

朱熹的“卫臣求贤说”现在看是很符合诗意的，似可与《毛序》的说法归为同一大类，即“求贤说”。这也是历代大多数解诗者的看法。

二、有些学者比如魏源（《诗古微》作者）认为本诗说的是公子伋和公子寿的事。

三、还有学者认为本诗是说一个贵族男子探望美女并向其献殷勤的。

2、崔述《读风偶识》评论道：同样是齐桓公重新封立的国家，邢国仅撑了二十几年就亡国了，而卫国历春秋而战国而秦，再享国祚数百年，何故？人君对贤才不惟举之用之，且鼓之舞之。读了《干旄》就知道卫国之所以能存在很久是有理由的。

3、孑（jie阳平)，特立，出众。孑孑，特出之貌。指旗帜很显眼，高挂于旗杆上。《说文》：“孑，无又（右）臂也”。就是没有右胳膊，少了一条手臂。所以，“孑”可引申为“特立”。

4、旄，读音mao，阳平。通“牦”，指牦牛尾。以牦牛尾饰旗杆，树于车后，以状威仪。这种旗子称“旄”。

“干”，三家诗作“竿”，旗杆。

下文的“旟”、“旌”与“旄”同，都是指招贤才时用的旗子。

5、浚，鄘国的地名。 《邶风.凯风》，曾经提到“浚”（“爰有寒泉，在浚之下”），因邶鄘卫后来合并为卫，书香君以为可能是同一地。

6、郊，邑外较远的地方。相当于现在说的远郊。古时，郊、牧、野、林、垌，分别指由近到远的郊外。

7、都，近城，主城的下邑，类似卫城。

8、城，城区，类似现在所说的市区。

9、纰（pi阳平），连缀。在衣冠或旗帜上镶边。与下文的“组”、“祝”同义。皆束丝之法。

10、姝，美好貌。一说顺从貌。胡承珙认为“姝”是“孎”的借字，“谨慎”的意思。本诗中，“姝”，代指贤士。这个“姝”，学者们也有不同的理解。姚际恒认为《邶风.静女》之“静女其姝”和《郑风.东方之日》之“彼姝者子”的“姝”都是指女子，如果将《干旄》中的“姝”解为“贤士”似觉不妥。方玉润则认为，既然《诗经》里可以用“西方美人”代指男舞师（请参我2021年1月21日朋友圈《邶风.简兮》篇），屈原也常用“香草美人”自喻、以“美人”代指楚怀王，那么，用“姝”代指贤士也无不可。

11、畀，bi去声，给予。 与下章“何以予之”的“予”同义。

12、旟，yu阳平，画有鸟隼的旗。

13、旌，旗的一种。挂牦牛尾于竿头，下有五彩鸟羽。

14、祝，“属”的借字，编连，缝合。与上文的“纰、组”意近。

16、告，作名词用，指良策、建议。一说通“造”，赠予。

17、我想，刘备应该看过《干旄》吧，三顾茅庐或许是受此诗的触动？

18、季札至卫时曾说：“卫多君子，未有患也。”

崔述感叹：君子之所以多，正由其君好贤，因而其卿大夫咸知下士，躬访贤才於畎亩中，以故人皆竞於贤耳。是知立国之规模未有不在於好贤者。读《诗》者能以此篇例之，则授之以政而无不达者矣。

骏译:

牦尾饰杆旗高扬，
浚城郊外车翱翔。
素色丝线旗边镶，
四匹好马相跟上。
那位俊才多贤良，
献何良策报君王？

高扬旗上绣鸟隼，
华车驰至浚卫城。
白色丝线旗边缝，
良马五匹后面跟。
那位贤人有才能，
何计报国作回赠？

旗杆高高垂鸟羽，
华车已到浚城区。
白色丝线缝旗上，
六匹骏马同驰驱。
那位良士好忠顺，
有何高见与君语？
