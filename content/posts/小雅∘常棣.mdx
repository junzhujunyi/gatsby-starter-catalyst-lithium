有诗评曰：笔意抑扬曲折。前五章繁弦促节，多慷慨激昂之音；后三章轻拢慢捻，有洋洋盈耳之趣。

## 🌼 《小雅∘常棣》原文：

```
常棣之华，鄂不韡韡（hiuəi）；
凡今之人，莫如兄弟（dyei）。
```

*书香君👉🏻：*

> “韡、弟”，微脂合韵

```
死丧之威（iuəi），
兄弟孔怀（hoəi）。
原隰裒（bu）矣，
兄弟求（giu）矣。
```

*书香君👉🏻：*

> “威、怀”，微部əi，押韵；“裒、求”，幽部u，押韵

```
脊令在原（ngiuan），兄弟急难（nan）。
每有良朋，况也永叹（than）。
```

*书香君👉🏻：*

> “原、难、叹”，元部an，押韵

```
兄弟阋于墙，外御其务（miu）。
每有良朋，烝也无戎（njiuəm）。
```

*书香君👉🏻：*

> “务、戎”，幽侵合韵

```
丧乱既平（bieng），
既安且宁（nyeng）。
虽有兄弟，不如友生（sheng）。
```

*书香君👉🏻：*

> “平、宁、生”，耕部eng，押韵

```
傧尔笾豆（do），
饮酒之饫（o）。
兄弟既具（gio），
和乐且孺（njio）。
```

*书香君👉🏻：*

> “豆、饫、具、孺”，侯部o，押韵

```
妻子好合（həp），
如鼓瑟琴（giəm）。
兄弟既翕（xiəp），
和乐且湛（təm）。
```

*书香君👉🏻：*

> 一三句“合、翕”，缉部əp，押韵；二四句“琴、湛”，侵部əm，押韵

```
宜尔室家（kea），
乐尔妻帑（na）。
是究是图（da），
亶其然乎（ha）！
```

*书香君👉🏻：*

> “家、帑、图、乎”，鱼部a，押韵


## 🍁️ 骏注：

1、这首《常棣》，是宴会上歌唱兄弟亲情的诗。全诗八章，每章四句，但各章韵脚数不同，有一韵有两韵。

“常棣”，就是棠棣。

我觉得，诗旨体现在“凡今之人，莫如兄弟”这句。

诗中既有对“莫如兄弟”的赞美肯定，也有对“不如友生”的感叹，更有对“和乐且湛”的憧憬。前人有云：前五章繁弦促节，多慷慨激昂之音，后三章轻拢慢捻，有洋洋盈耳之趣。

本诗是《诗经》中的名篇之一。

2、《毛诗序》：“《常棣》，燕兄弟也。闵管蔡之失道，故作《常棣》焉。”

《毛传》：“周公吊二叔之不咸，而使兄弟之恩疏，召公为作此诗而歌之以亲之。”

《郑笺》：“喻弟以敬事兄，兄以荣履弟，恩义之显，亦韡韡然。人之恩亲，无如兄弟之最厚。”

书香君：序传笺什么意思呢？周公吊二叔，召公作是诗，说的是兄弟情义。二叔，指的是管叔姬鲜。我们知道，周文王长子伯邑考死于殷纣王之手，那是周灭商立朝之前的事了，本来武王姬发排行第二，管叔排行第三。长兄姬伯邑考死后，活着的众多兄弟中，姬发就成为大哥，管叔就成了“二哥”，所以称“二叔”。管、蔡、霍这三位叔（武王和姬旦的兄弟）与武庚一起叛乱。后周公平叛，杀管叔。周公宴请其他兄弟时，提起这事，感叹管叔不够意思，因作此诗（有人认为是周公异母弟召公所作）。

3、《诗集传》：“此诗盖周公既诛管、蔡而作。…《序》以为闵管蔡之失道者，得之。而又以为文武之诗，则误矣。大抵旧说《诗》之时世，皆不足信。”

4、《诗经原始》：“《常棣》，周公燕兄弟也。…是诗为周公作，穆公（穆公，指召穆公，是首任召公姬奭次子的后代，史又称召公虎、召虎。——书香君注）特重歌之耳。”

书香君：方玉润指出：《左传》说本诗是召穆公所作，《国语》又说是周公的诗。而韦昭说，当初是周公所作，后来周室衰弱、周厉王无道，骨肉恩缺、亲亲礼废，宴兄弟之乐绝。召穆公怀念周室初时的美好品德，复作《常棣》之歌以亲之。也就是：周成王时期周公所作，后世周厉王时期召穆公“重歌之”。方玉润所说的韦昭，又叫韦曜，三国时吴国史学家，著有《吴书》、《国语注》等。

5、但细品诗意，管、蔡、霍叔，周公的这三位兄弟与殷纣王之子武庚这个“外人”一起叛周（侄儿周成王之朝），显然是亲其所疏者而疏其所亲者，他们的叛乱行为，与诗中“脊令在原，兄弟急难”、“兄弟阋于墙，外御其务”、“每有良朋，况也永叹、烝也无戎”等等语意不洽，且诗中颇多衰世之语，如“死丧之威”、“丧乱既平”等等，不大像是周初作品。

故，现、当代学者也有不少倾向于是召穆公虎所作。

6、常棣，亦作棠棣、唐棣，即郁李，蔷薇科落叶灌木，花粉红色或白色，果实比李小，可食。

7、 华，花。

8、鄂，通“萼”，花萼、花托。

9、不，“丕”的借字。一说花托，读音fu阴平。一说当作语助词。

10、韡韡，读音wei上声。花儿鲜明茂盛貌。

11、威，通“畏”，畏惧，可怕。

12、孔怀，甚为思念、极其思念。孔，副词，表程度，很，最。

13、裒，读音pou阳平，聚集。

14、脊令，通作“鹡鸰”，一种水鸟。通常生活于水中，而在原野上，则比喻失其所。

15、每，虽然。

16、况，增加、更加。

17、永，长。

18、阋，读音xi去声，争吵、争斗、裂痕、分歧。

19、 御，抵御，抵抗。

20、务，通“侮”。读音也宜读wu上声。

21、烝，长久。一说为发语词。

22、戎，帮助。

23、 友生，友人。生，语气词，无实义。

24、傧（bīn）：陈列。

25、笾（biān）、豆：祭祀或燕享时用来盛食物的器具。笾用竹制，盛水果。豆用木制，盛肉类。我们之前介绍过“笾豆有践”。

26、饫，读音yu去声，宴饮同姓的私宴。一说酒足饭饱。

27、具，通“俱”，俱全，完备，聚集。

28、 孺，相亲。

29、翕，读音xi阴平，聚合，和好。

30、湛，读音dan阴平，通“耽”，喜乐。

31、帑，读音nu阳平，通“孥”，儿女。

32、究，深思。图，思虑。

33、亶，读音dan上声，信，确实。如我们介绍过的“古公亶父”。

34、有诗评曰：笔意抑扬曲折。前五章繁弦促节，多慷慨激昂之音；后三章轻拢慢捻，有洋洋盈耳之趣。


## 🍁️ 骏译：


> 棠棣丛丛花绚烂，
> 
> 花萼润泽色鲜艳。
> 
> 当今天下人攘攘，
> 
> 就数兄弟情深长。
> 

> 生老病死突然间，
> 
> 唯有兄弟最挂牵。
> 
> 即便异乡抛荒野，
> 
> 兄弟遍寻要找见。
> 

> 鹡鸰原野漫扑颠，
> 
> 兄弟急急来救援。
> 
> 平常虽有好朋友，
> 
> 危难不过长叹嗟。
> 

> 兄弟在家会龃龉，
> 
> 外侮当前鼎力助。
> 
> 倒是那些好朋友，
> 
> 关键时刻事无补。
> 

> 死丧急难诸事定，
> 
> 生活有序人安宁。
> 
> 此时此刻亲兄弟，
> 
> 不如朋友更贴心。
> 

> 摆盘布盏开盛宴，
> 
> 酒美肴佳果品鲜。
> 
> 兄弟亲热来团聚，
> 
> 血脉相连骨肉牵。
> 

> 夫妇姻缘上天许，
> 
> 琴瑟和谐好情趣。
> 
> 兄弟亲密聚一起，
> 
> 祥和欢乐永相续。
> 

> 家里家外若有序，
> 
> 老婆孩子就欢喜。
> 
> 凡事深思且熟虑，
> 
> 所言是否有道理？
>
