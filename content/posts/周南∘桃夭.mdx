灼灼，刘勰赞为千年佳词佳构。
桃夭，开千秋辞赋咏美人之祖。

流连万象之际，
沉吟视听之区。
写气图貌，
既随物以宛转；
属采附声，
亦与心而徘徊。

姚际恒赞《桃夭》：桃花色最艳，故以取喻女子，开千古辞赋咏美人之祖。

## 🌼 《周南∘桃夭》原文：
```

桃之夭夭，灼灼其华（hoa）；
之子于归，宜其室家（kea）。

```
*书香君👉🏻：* 

> 华、家，鱼部，押韵

```
桃之夭夭，有蕡其实（djiet）；
之子于归，宜其家室（sjiet）。

```
*书香君👉🏻：* 

> 实、室，质部，押韵

```
桃之夭夭，其叶蓁蓁（tzhen）；
之子于归，宜其家人（njien）。

```
*书香君👉🏻：* 

> 蓁、人，真部，押韵

## 🍁️ 骏注：

1、这首《桃夭》既著名又浅显且脍炙人口。

2、今人多认为这是一首送嫁或赞美新娘、祝福夫家的诗。

3、《毛诗序》有点迂腐、附会，认为是美后妃之德的诗：“《桃夭》后妃之所致也。不妒忌则男女以正，婚姻以时，国无鳏民也”。后世学者认为，诗中既以“之子”谓女子，该女子当不会是后妃，因为指后妃用“之子”就显得太过随便了。

4、《诗集传》：“文王之化，自家而国，男女以正，婚姻以时。故诗人因所见以起兴，而叹其女子之贤，知其必有以宜其室家也。”
朱子虽然承毛序之“教化”说，有过于附会之嫌，但有一点非常值得肯定——“自家而国”，这是根本，没有家何来国？人类历史，都是先有家后有国，如果没有老百姓的幸福，所有的口号都是空洞的。那种所谓“有国才有家”的忽悠百姓的谬论可以休矣！

5、《诗经原始》：“《桃夭》，喜之子能宜室家也。”

对于毛序评《桃夭》“不妒忌，则男女以正，昏姻以时，国无鳏民也。”之说，方玉润认为可能是从《孟子》那里来的：此亦本《孟子》“太王好色，内无怨女，外无旷夫”为言。

（书香君：《孟子.梁惠王下》其中一章有以下文字：（齐宣）王曰：“寡人有疾，寡人好色。”（孟子）对曰：“昔者大王好色，爱厥妃。当是时也，内无怨女，外无旷夫。王如好色，与百姓同之，于王何有？”

上文大意是：齐宣王说，他自己有一个毛病，那就是爱好女色。孟子回答说，爱好女色并非什么过错，以往周文王的祖父古公亶父（就是周太王，也写作周大王——书香君注）就很好色，十分宠爱其后妃。但他非常勤政，那个时候，男女都能适时婚配，内无大龄未嫁的怨女，外无大龄未婚的旷夫。您齐王自己好色没关系，只要能够让老百姓跟您
一样也能满足色欲需要，又有什么内疚的呢？ ）

方玉润进一步阐述：“此亦咏新昏诗，与《关雎》同为房中乐，如后世催妆坐筵等词。《关雎》从男求女一面说，此从女归男一面说，互相掩映，同为美俗。以如花胜玉之子，而宜室宜家，可谓德色双美，艳称一时。…一章艳绝，开千古词赋香奁之祖。”

6、“夭夭”，花朵怒放茂盛之貌。

5、“华”，花也。如早生华发（赤壁怀古），邺水朱华（滕王阁序），如果诵读，也要读阴平。

6、“灼灼”，花儿明艳、鲜亮之貌。“灼灼”二字被刘勰（《文心雕龙》著者）盛赞为千年佳词佳构。

我还记得《谁是最可爱的人》，魏巍描写一个年轻战士用了这样的好句子——“他站在那里，就象秋天田野里的一株红高粱那样淳朴可爱”，也许他也从《桃夭》中汲取过养分吧。

7、“之子”，这个人，这个女子。子，在诗经时代指男女皆可，似乎多指女子。

8、“于归”，女子出嫁。于，去也，往也。归，归宿。古时，丈夫、夫家为女子归宿。于归，指去到夫家，去到归宿之地，当然就是出嫁。

9、“宜”，相宜、相适也。使…和顺、使…和睦也。

10、“室家”，以及后文的“家室”、“家人”，均指家庭或夫妇，换字是为了押韵。无论室或家，皆指夫家的家庭，要说有什么细微区别，我理解，室，更多地指夫妇的小家庭；家，更多地指与公婆或夫家多代长辈同住的大家庭。

11、“蕡”，读音fen，阳平。原指果实成熟时红白相间的纹理，盛赞果实长得好，可喻指果实累累、丰收等等。

12、“有”，我们多次解释过，诗经时代，用于加强其后面的字的程度，可以简单理解成重复后面的那个字，“有蕡”，就是“蕡蕡”。

13、“实”，果实。“蓁”，树叶茂盛之态。

14、清代诗经大家姚际恒高度评价《桃夭》以桃花比美人之妙：“桃花色最艳，故以取喻女子，开千古辞赋咏美人之祖。”我们知道此后历代文人以花，特别是桃花咏美女始无穷尽。我猜想，崔护的“人面桃花相映红”应该也有《桃夭》的影响吧。至于后世描摹美女的常用词：杏眼、桃腮、樱唇、梨窝、葱指、柳腰等等，抑或发端于此呢。

15、本诗以“灼灼其华”比兴出嫁少女之青春美艳、面容娇羞；以“蕡蕡其实”比兴多子多孙；以“其叶蓁蓁”比兴家族兴旺、基业常青。既鲜活又贴切！正如《文心雕龙》所言：“诗人感物，联类不穷。流连万象之际，沉吟视听之区。写气图貌，既随物以宛转；属采附声，亦与心而徘徊。”
