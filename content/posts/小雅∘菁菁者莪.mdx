莪草青青
我俩相遇在那山陵
碧水湖心
荡漾我俩泛舟倒影

《菁菁者莪》当作情歌未尝不可



## 🌼 《菁菁者莪》原文：

```
菁菁者莪（ngai），
在彼中阿（ai）；
既见君子，乐且有仪（ngiai）。
```

*书香君👉🏻：*

> “莪、阿、仪”，歌部ai，押韵

```
菁菁者莪，在彼中沚（tjiə）；
既见君子，我心则喜（xiə）。
```

*书香君👉🏻：*

> “沚、喜”，之部ə，押韵

```
菁菁者莪，在彼中陵（liəng）；
既见君子，锡我百朋（bəng）。
```

*书香君👉🏻：*

> “陵、朋”，蒸部əng，押韵

```
泛泛杨舟（tjiu），
载沉载浮（biu）；
既见君子，我心则休（xiu）。
```

*书香君👉🏻：*

> “舟、浮、休”，幽部u，押韵

> 本诗四章，每章一韵。


## 🍁️ 骏注：

1、此诗的主旨，《毛诗序》说是“乐育材”。此一观点延绵两千年，影响深远。后世也约定俗成地以“菁莪”比喻“乐育人才”，或者代指“贤才”。就连反对《毛诗序》这一观点、认为此观点“全失诗意”的朱熹（他认为本诗是“燕饮之歌”），也以“菁莪”喻学子，其《白鹿洞赋》即有“广青衿之疑问，乐菁莪之长育”句。

2、现在我们仅从字面看，分明是一首述说女子因与所爱之人多次相逢而欣喜欢愉之歌。也许是互相钟情的两人心有灵犀而多次不期而遇；也许就是男女有约——徜徉于丘陵山坳、水中泛舟沙洲奔跑。诗文虽没有明示是邂逅抑或约会，但读者自可想象，画面不要太美喔。所以，我认为“情诗说”更有道理些。我们之后将要分享的《小雅.隰桑》与本诗的句式、章法都有相似之处。而《隰桑》更公认是爱情诗。两诗可比对欣赏。

3、“菁”，读音jing，阴平。菁菁，草木茂盛貌。本诗以“莪”之菁菁起兴。《诗经》时代，往往以植物起兴，尤以常见的可食用的为多。这也说明诗歌来自于日常生活和劳作。目之所见，心之所想，口之所唱也。起兴的载体有时意有所喻，有时又很随意，只是看到了就唱出来了，未必有关联，这才应该是最早的诗歌之态。（但仍有学者坚持认为，《诗经》中所有的托“兴”之物与诗旨都有深层次的关联。书香君不认同）。如果一定要说“莪”与这首爱情诗的诗意有什么关联，那就是它的繁茂蓬勃（“菁菁”）和女子的喜悦心情是相谐的，如此而已。假如认可诗旨是“乐育才”的话，那么，“莪之菁菁”确实可以比兴“人才的茁壮成长”。

4、先看看旧说如何说。

A、《毛诗序》：“《菁菁者莪》，乐育材也。君子能长育人材，则天下喜乐之矣。”

B、《毛传》：“乐育材者，歌乐人君教学国人，秀士、选士、俊士、造士、进士，养之以渐，至于官之。君子能长育人材，如阿之长莪菁菁然。”

C、《郑笺》：“长育之者，既教学之，又不征役也。舟者沉物亦载，浮物亦载，喻人君用士，文亦用，武亦用，于人之材无所废。”

D、《诗集传》：“君子，指宾客也。此亦燕饮宾客之诗。言菁菁者莪，则在彼中阿矣。既见君子，则我心喜乐而有礼仪矣。或曰：以菁菁者莪比君子容貌威仪之盛也。”

E、姚际恒《诗经通论》批驳朱熹之释，认为诗中无“燕饮”字，当为“人君喜得见贤之诗，其余则不可以臆断。”

F、《诗经原始》：“此诗当是君临敝廱，见学校人材之盛，喜而作此。或即以燕飨群材，亦未可知。总之，不离育材者近是。”

G、王先谦

《诗三家义集疏》：“古者育材之法备于此矣。”

H、程俊英/蒋见元《诗经注析》认为该诗“作者深受贵族的培植与赏赐，写这首诗来表示学有榜样和喜悦的心情”，并否定是恋爱诗，认为：“以此诗为恋爱诗歌者，则相去远也。”

书香君认为，作为恋诗未尝不可。

5、“莪”，读音e，阳平。莪蒿，草本植物，俗称萝蒿、山姜黄。其根状茎可入药，中医称“郁金”。

6、“阿”，读音e，阴平。大陵也。山阿也。  “中阿”，即阿中，山坳。

7、“仪”，仪容、仪表、气度。

8、“沚”，读音zhi，上声。水中小洲。可参见《秦风.蒹葭》。“中沚”，沚中也。

9、“锡”，通“赐”，也要读作ci。

10、“朋”，上古以贝为一般等价物（也即货币）。五个贝壳串为“一系”，两系为“一朋”。百朋，非实指也，言财多矣。“锡我百朋”  ，就是“赠送给我百朋（钱）”，也许确实是男子赠送给女子钱财了，她很开心。也许女子只是以此比喻她“既见君子”的喜悦，这份快乐就好像，甚至，胜过“受了百朋”。（ 认同“乐育材”观点者则认为君王考察时对培育人才的事很满意而“赐百朋”。）

11、“泛泛”，有些版本作“汎汎”。义同。可参我们分享过的《邶风.二子乘舟》（2021年1月27号朋友圈）。

12、“载沉载浮”，指船在水中忽上忽下、悠悠荡荡。载，则，又，也。

13、“休”，安也，安定。一说欣欣然，欢喜。结合诗文，“安宁、欣喜”都是应有之义。上文的“载沉载浮”与“我心则休”一对照，其中的韵味就出来了——两人在湖中泛舟，倾谈甚欢。湖水荡漾也切合了心情的激动、愉快。“我心则休”，也许是得到了爱的承诺、爱的表白，所以笃定了、心安了。

14、这篇《菁菁者莪》虽在《小雅》，看上去跟《风》也没什么区别。这是《小雅》的特色之一。我们前已介绍，《小雅》所收的民歌是产生于“王畿之地”而非诸侯国的，故不能收入《国风》，但为什么没有放在二南里？原因可能还在于“宴饮之歌”的性质。所以朱熹认为“此亦燕饮宾客之诗”，也算合理。（风和雅的区别及各自定义，请参我2020年12月3日朋友圈《关雎》和2021年6月6日朋友圈《鹿鸣》）

那么，我们当它是“宴饮之歌”中的爱情篇章就好。在宴饮之堂，官方可以宏大叙事，民间也可以儿女情长。往往民间作品生命力更强呢。
