一咏不已，再三反覆而咏叹之，则其情亦可见矣。三章只换六字，而一往情深，低徊无限。此专以描摹虚神擅长，凭吊诗中绝唱也。

## 🌼 《王风∘黍离》原文：
```

彼黍离离（liai），
彼稷之苗（mio）。
行迈靡靡（miai），
中心摇摇（jio）。
知我者谓我心忧（iu），
不知我者谓我何求（giu）。
悠悠苍天（thyen），
此何人（njien）哉？

```
*书香君👉🏻：* 

> 首章四韵。一三句“离、靡”，歌部；二四句“苗、摇”，宵部；五六句“忧、求”，幽部；七八句“天、人”，真部

```
彼黍离离（liai），
彼稷之穗（ziuet）。
行迈靡靡（miai），
中心如醉（tziut）。
知我者谓我心忧（iu），
不知我者谓我何求（giu）。
悠悠苍天（thyen），
此何人（njien）哉？

```
*书香君👉🏻：* 

> 二章四韵。一三句同首章一三句，歌部；二四句“穗、醉”，质物合韵；五六句同首章五六句，幽部；七八句同首章七八句，真部

```
彼黍离离（liai），
彼稷之实（djiet）。
行迈靡靡（miai），
中心如噎（yet）。
知我者谓我心忧（iu），
不知我者谓我何求（giu）。
悠悠苍天（thyen），
此何人（njien）哉！

```
*书香君👉🏻：* 

> 末章四韵。一三句同首章一三句，歌部；二四句“实、噎”，质部；五六句同首章五六句，幽部；七八句同首章七八句，真部

```
本诗三章，每章的第一、三、五、六、七、八，六句完全相同，即叠句。第二句只有一字之换，就是“苗、穗、实”，递进地描述从禾苗到抽穗到结籽。第四句只有一到两个字的变化，就是“摇摇、如醉、如噎”。三章基本可看成重章，复沓但有递进。

```
## 🍁️ 骏注：

1、这首《黍离》，是《王风》（共10篇）的第一篇。它的“骏注骏译”，三年前首发于我的个人公众号“骏语君谈”。这个公众号已无法更新，所以这次发朋友圈再作补充和修订。

2、《黍离》，是一首怀古凭吊诗，被誉为“凭吊诗之绝唱”（清人、《诗经》大家方玉润语）。

周幽王“烽火戏诸侯”大家都耳熟能详。西周随后就被猃狁（猃狁是匈奴旧称，我之前有一篇小文，大致梳理了先秦“蛮夷狄戎”的前世今生。）所灭而平王只得东迁，周王室由此式微。当然，西周被猃狁灭，有内外两方面原因，老丈人申侯因为女儿（申后）和外孙（宜臼，后来的周平王）被周幽王废掉王后和太子之位，遂联合猃狁来讨罪。此处不细表。东周臣子出差到西周都城故地，眼见得昔日宗庙宫殿已成农田，黍稷生长其间。于是“闵周室之颠覆，彷徨不忍去，而作是诗”。

《毛诗序》的这个判断被后世认可。此外也仍然有不同解读。比如宋代大儒程颐就将“彼稷之苗”解为“后稷的苗裔”，诗的意味就不一样了。余冠英认为是“流浪者的叙说”，程俊英则说是“诗人难舍家园之作”。之所以有这样那样的其他解读，源自于韩诗和鲁诗有一些流传下来的说法与《毛诗序》不同。当然，所有解读也都有一个共同点：因时世变迁而忧思神伤。今天来读，弥漫于其间的凄怆、沧桑仍能击中我们。本诗对后世的感怀诗影响很大，而“黍离”一词也因此约定俗成地有了“故国之思”、“对国家由盛而衰的感怀”、“因时事变迁而忧思”的意象。

曹植《情诗》有“游者叹黍离，处者歌式微。慷慨对嘉宾，凄怆内伤悲”。（《式微》，出自《诗经.邶风》，可参我2021年1月19日朋友圈）。

西晋向子期《思旧赋》有“叹《黍离》之愍周兮，悲《麦秀》于殷墟。惟古昔以怀今，心徘徊以踌躇。”

南宋姜夔在其《扬州慢》小序里也说“夜雪初霁，荠麦弥望……予怀怆然，感慨今夕，……千岩老人以为有黍离之悲也”。

同时，诗中“悠悠苍天”也成为经典词。

初唐诗人陈子昂那首著名的《登幽州台歌》就有“念天地之悠悠，独怆然而涕下”。

收在《诗经.唐风》中的《鸨羽》也有“悠悠苍天，曷其有所（所，常）”句。

我们罗列一下历代主要的解说做参考。

《毛诗序》：“《黍离》，闵宗周也。周大夫行役，至于宗周，过故宗庙宫室，尽为禾黍，闵周室之颠覆，彷徨不忍去，而作是诗也。”

《毛传》：“宗周，镐京也，谓之西周。周，王城也，谓之东周。幽王之乱，而宗周灭。平王东迁政遂微弱，下列于诸侯，其诗不能复《雅》，而同于《国风》焉。”

*书香君👉🏻：* 

> 《毛传》什么意思呢？关于《王风》的概念，之前我也介绍过，这里再说一下。就是说，西周都于镐京（即丰镐，丰邑为宗庙区，镐京为行政区。丰镐也称作“西都”），又叫宗周，平王迁都到洛邑后叫东周，也叫成周，洛邑也称作“东都”，洛邑有内城叫王城，方圆六百里。当代考古发现确实有王城遗址，是洛邑的内城、中心区。本来东周王室区域收集的诗章应该属于《雅》的范畴，《国风》，除二南之外，应该都是诸侯国之诗，为啥出来一个不伦不类的《王风》？“王”，不是诸侯国名，东周王室不是诸侯。因为从平王东迁开始周室就逐渐走向衰败、微弱，对诸侯也失去了控制和调度能力，其地位遂卑，而与诸侯无异。故，其诗的教化能力、风格、内容、意象、气度、格局、音律等等都不复能“雅”，不复为《雅》，而混同于《国风》，但它的王号并没有更替，所以不称“周”而称“王”，称《王风》。注意，这只是《王风》解释中的一种，朱熹持此观点。自古至今，学人尚有各种其他观点，此处不细说，以后会随篇章介绍。

《诗集传》：“周既东迁，大夫行役至于宗周，…闵周室之颠覆，故赋其所见黍之离离，与稷之苗，以兴行之靡靡，心之摇摇。既叹时人莫识己意，又伤所以致此者，果何人哉？追怨之深也。”

朱熹和序笺都认为诗中“此何人哉”的“何人”，是作者追怨的对象，是造成西周灭亡的责任者，就是那个亡国之君，当然暗指周幽王。

当代有把“何人”当做“泛指”的，泛指害人者。书香君倾向于是诗人自我解嘲的，“何人”就是诗人自己——别人都能安于现状，独我心忧，我究竟是怎样一个人啊这么不合世道？暗喻自己不愿同流合污、执着于理想的高洁情怀。参见后文“骏注9”。

《诗经原始》：“《黍离》，闵宗周也。周撤既东，无复西幸。文武成康之旧，一旦灰烬，荡然无存。有心斯世者，所为目击心伤，不能无慨于其际焉。…一咏不已，再三反覆而咏叹之，则其情亦可见矣。三章只换六字，而一往情深，低徊无限。此专以描摹虚神擅长，凭吊诗中绝唱也。”

陆侃如/冯沅君夫妇的《中国诗史》认为，诗中并不能看出凭吊故国之意，是写迁都时心中难受的感觉。程俊英从其说。

3、黍，读音shu，上声。黄米，小米。稷，谷类中最早种植的一种谷物，历代有多种称呼。有红，白，黄，黑多种，为五谷之长，所以古时称农官为“稷”，谷神也为“稷”、“稷正”。史上最著名的农官，就是周的先祖后稷。

4、离离，茂密的样子，一说庄稼成行成列的样子。从本诗看，两说皆可。我采前说（“离离原上草”中的离离，就是茂密）。“彼黍离离，彼稷之苗”，事实上是“彼黍稷离离，彼黍稷之苗”，这种用法，黍、稷虽分在两句，实则黍、稷兼写。汉语之妙存乎我心矣。

5、行迈，行走，迈步。

6、靡靡，走路迟缓的样子。

7、中心，就是心中，这是古时的用法（倒装）。我们前面在《周南.兔罝》中也介绍过，比如“中林”就是“林中”，“中逵”就是“逵中”。

8、摇摇，心神不安的样子。“行迈靡靡，中心摇摇”，走路迟缓心神不宁东张西望的，表面上看好像在路上寻找什么东西，所以下文有“不知我者谓我何求”，“不了解的人还以为我在寻找什么呢”，深层的意思是“我心中有什么思虑、追求、追问”

8、噎，气逆噎着，喻指郁闷至极到了郁结堵塞的程度了。

9、“此何人哉”，这句中的“何人”，到底怎么解？很多版本把“悠悠苍天，此何人哉”译成：“老天啊，是什么人害得我离家？”我认为不妥。我的理解是：“我究竟是个什么样的人？为何跟别人如此不同？”。别人都岁月静好，你为什么这样忧思操心啊？你到底有何求？你到底想怎样？这既是自问也是反问，有自我解嘲的意味，但同时又坚定地表达自己的独立思考和不懈追求。个人观点。

10、本诗三章仅换了六个字，由“苗”而“穗”而“实”是写黍稷的成长；由“摇摇”而“如醉”而“如噎”，是情绪的递进。高超的手法简直了！

11、后世受《黍离》影响的诗词作品很多，仅举数例。

如唐代诗人刘沧写唐太宗陵墓的诗《秋日过昭陵》：

寝庙徒悲剑与冠，
翠华龙驭杳漫漫。
原分山势入空塞，
地匝松阴出晚寒。
上界鼎成云缥缈，
西陵舞罢泪阑干。
那堪独立斜阳里，
碧落秋光烟树残。

晚唐诗人许浑《姑苏怀古》：

宫馆余基辍棹过，
黍苗无限独悲歌。
荒台麋鹿争新草，
空苑岛凫占浅莎。
吴岫雨来虚槛冷，
楚江风急远帆多。
可怜国破忠臣死，
日月东流生白波。

杜甫《哀江头》、刘禹锡《金陵五题.石头城》中都有类似的情绪，大家有空可对照品读。
