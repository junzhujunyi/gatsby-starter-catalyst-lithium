一句“仲可怀也”，道出女子的心声！不是不爱，是不敢啊。
“人言可畏”、“邻里”，源于此诗。

## 🌼 《将仲子》原文：
```

将仲子（tziə）兮，
无逾我里（liə），
无折我树杞（khiə）。
岂敢爱之？畏我父母（mə）。
仲可怀（hoəi）也；
父母之言，亦可畏（iuəi）也。

```
*书香君👉🏻：* 

> “子、里、杞、母”，之部，押韵；“怀、畏”，微部，押韵

```
将仲子兮，无逾我墙（dziang），
无折我树桑（sang）。
岂敢爱之？畏我诸兄（xyuang）。
仲可怀（hoəi）也；
诸兄之言，亦可畏（iuəi）也。

```
*书香君👉🏻：* 

> “墙、桑、兄”，阳部，押韵；“怀、畏”，微部押韵

```
将仲子兮，无逾我园（hiuan），
无折我树檀（dan）。
岂敢爱之？畏人之多言（ngian）。
仲可怀（hoəi）也；
人之多言，亦可畏（iuəi）也。

```
*书香君👉🏻：* 

> “园、檀、言”，元部，押韵；“怀、畏”，微部，押韵

```
本诗三章，每章两韵。“逾”，有些版本作“窬”。

```
## 🍁️ 骏注：

1、旧说是刺郑庄公的。当代则基本认为是女子表达犹豫心理的爱情诗。女子担心父母兄长邻里的责怪和耻笑，不敢私下与男友约会，但自己又爱着男友。本篇把这种矛盾犹豫心理描写得细致入微。

2、我们来看看古人具体是怎样说的。

《毛诗序》：“《将仲子》，刺庄公也。不胜其母，以害其弟，弟叔失道而公弗制，祭仲（祭仲，郑国大夫，祭读zhai，曾为共叔段的事谏庄公——书香君注）谏而公弗听，小不忍以致大乱焉。”
《毛诗序》的这个说法依据的是史上一个著名的故事（“郑伯克段于鄢”），把诗中的“仲子”解为祭仲，那诗就是郑庄公的口吻了。

《郑笺》释曰：“祭仲骤谏，庄公不能用其言，故言请，固拒之。无逾我里，喻言无干我亲戚也。无折我树杞，喻言无伤害我兄弟也。段将为害，我岂敢爱之而不诛与？以父母之故，故不为也。仲子之言可私怀也，我迫于父母有言，不得从也。”

朱熹从郑樵（宋代史学家郑渔仲——书香君注）说，认为本诗跟郑庄公、共叔段没有关系，而是一首淫奔诗。《诗集传》解读诗旨只有下面这区区一句：莆田郑氏（指郑樵，郑樵，宋代史学家，兴化军莆田县人，著有200卷700万字的文史类百科全书《通志》——书香君注）曰：“此淫奔者之辞”。

朱子将本诗当做男女情爱诗，比之于序笺无疑是鲜活的；但其“淫奔之辞”的说教，今天看来，又是多么陈腐！

方玉润《诗经原始》认为，《毛诗序》之所以说刺庄公不胜其母以害其弟云云，乃诗中有“父母、兄弟、仲子”等字词。方氏既否定毛序说，也不认同朱熹淫奔说——“盖女心既有所畏而不从，则不得谓之为奔，亦不得谓之为淫”。但方氏仍未逃脱教化之窠臼——“其义有合于圣贤守身大道，故太史录之，以为涉世法。惕以人言可畏，即父母兄弟有所不敢欺，则欲念顿消而天理自在，是善于守身法也。此非淫词，断可知也。”

姚际恒与方玉润有相似看法，其《诗经通论》言：“此虽属淫，然女子为此婉转之辞以谢男子，而以父母、诸兄及人言为可畏，大有廉耻，又岂得为淫者哉！”

*书香君👉🏻：* 

> 总之，方、姚皆肯定女子守身、有廉耻，故谢绝男子的约会。难道他们没有看出诗中女子是爱着男子的吗？——一句“仲可怀也”，道出女子的心声！不是不爱，是不敢啊！与其说女子“善守身、有廉耻”，不如说世俗礼教对自由爱情的干预使女子很无奈、很烦恼。

今人程俊英《诗经注析》：“这是女子拒绝情人的诗，她怕家庭反对、舆论批评。短短八句一波三折，曲尽女子爱和畏的矛盾心理。”

3、将，读音qiang，阴平。请、愿。如《将进酒》之“将”。

4、仲子，男子之字。应该是诗中女子恋人的代指。春秋时代男女皆可称“子”，是敬称、美称。“仲子”，可以简单理解为“二哥”（邻里某家的老二）。《毛诗序》认为诗中“仲子”是郑国大夫祭仲。

5、逾，有些版本作“窬”，词义同（读音yu阳平），翻越、跨越。

6、里，古时，“里”最早是长度单位，成人步行多少步的长度为一里，之后也把方圆一里的面积称为“里”，“里”就又成了一个“面积”或“区域”的概念。周时，五家为一“邻”，五邻即二十五家为一“里”。每“里”的居住区域是有围墙围起来的，故有“里墙”。我们今天的“邻里”一词就是这么来的。从诗意分析，诗中的“里”，当为“里墙”。

7、爱，爱惜、吝惜，怜惜。我们不要理解成不敢“爱”仲子，而是该女子表白自己不是“爱惜”杞树桑树檀树，而是害怕爸妈、哥哥们和邻里的责骂和闲话（万一树枝被攀爬压折了，父母、诸兄、邻居就会发现有人翻墙进来）。后文“仲可怀也”的“怀”，才是“惦记、思念、爱”。

8、成语“人言可畏”应该出于此诗。

9、本诗，层层递进，由逾里而逾墙而逾园，仲子之来也渐而迫；由父母而诸兄而邻人，女子之畏也渐而广。

> 
> 可不要压折杞树翻进我家院落！
> 
> 我不是怜惜杞树啊，
> 
> 我是怕我爸妈说。
> 
> 其实我也好想你啊，
> 
> 可又担心爸妈责苛！

> 
> 可不敢攀折桑树越过我家墙垛！
> 
> 我哪是怜惜桑树啊，
> 
> 我是怕我兄长啰嗦。
> 
> 其实我也好想你呀，
> 
> 可又害怕兄长数落！

> 
> 可不能攀折檀树翻进我家花园！
> 
> 我哪是怜惜檀树啊，
> 
> 我担心邻里说长道短。
> 
> 其实我也好想你啊，
> 
> 人言可畏不得不防！
