恨嫁。
恨嫁诗也可以写得这么有情趣。

## 🌼 《召南∘摽有梅》原文：
```

摽有梅，其实七（tsiet）兮。
求我庶士，迨其吉（kiet）兮。

```
*书香君👉🏻：* 

> 首章一韵。“七、吉”，质部，押韵。

```
摽有梅，其实三（səm）兮。
求我庶士，迨其今（kiəm）兮。

```
*书香君👉🏻：* 

> 二章一韵。“三、今”，侵部，押韵。

```
摽有梅，顷筐塈（xiət）之。
求我庶士，迨其谓（jiuət）之。

```
*书香君👉🏻：* 

> 尾章一韵。“塈、谓”，物部，押韵。

## 🍁️ 骏注：

1、先说下标题。

“摽”，读音biao去声。坠落、打落。一说抛掷。“有”，在本诗中只是虚词。“摽有梅”，就是“摽梅”，理解成“梅子坠落”就可以了。如果把“摽”解读成“抛掷”，也未尝不可。那就是另一种情趣了——杨梅树上采摘梅子的姑娘，拿杨梅果扔掷树下心仪的小伙子，一帮青年男女采摘、嬉戏、示爱，美美的画面。

西周时，三十之男，二十之女，可不待礼备而婚，以蕃育人口。时礼虽不备，相奔不禁（后面还会提及）。

什么意思呢？就是西周时，为鼓励生育，男子年过三十，女子年过二十，就可以不受婚姻的各种程序礼节仪式的约束，也不必非得经父母之命媒妁之言，就可以同居婚配，所谓私奔。“私奔”，并不是现在人们通常理解的“相爱男女悄悄地、私下地跑到别的地方”，而是“没有父母之命、媒妁之言就相爱同居”。

2、七，七成。当然“七”也并不一定就是实数，可以理解成“较多”，就像下文的“三”，你可以解为“三成”或“较少”。古人一至三为数小，七至九为数大。

3、庶，众多。士，本诗当指未婚男子。庶士，就是“诸位帅哥”、“各位才俊”、“先生们”之类。

4、迨，读音dai去声，趁着、等着。

5、顷筐，斜口浅筐（我们在《卷耳》篇已做过介绍），犹今之撮箕、簸箕。

6、谓，“说”、“对（某某）说”。朱子释为“谓之，则但相告语而约可定矣。”也有解读成“相会”或“出嫁”的，窃以为稍显牵强。

7、 本诗写梅子季节、女子恨嫁。是一首女子既委婉又大胆的求爱诗。摽有梅，就是摽梅、梅摽，梅子坠落。梅子成熟就会坠落（或被采摘者打落），喻指姑娘大了就要出嫁。“梅”、“媒”同音，诗人见梅起兴。我们知道，周时，“仲春之月，令会男女。于是时也，奔者不禁”，以这样的政策，爱情是及时的，也是质朴而奔放的，女子晚嫁者应该很少。且“若无故而不用令者，罚之。司男女之无夫家者而会之。”所以，未嫁的女子看到成熟的杨梅不断坠落，从果满枝头到“其实七兮”、“其实三兮”，再到“顷筐塈之”，梅子是越来越不新鲜、越不精贵了，恨嫁之情跃然纸上——从“迨其吉”到“迨其今”再到“迨其谓之”，质朴自然、生动传神，诚不谬也。

毛序将诗中的“今”字，解为“急辞也”；将“谓”字解为“勤也”，颇得诗意。

8、需要说明的是，多种解读是《诗经》的一大特色，早期也曾把《摽有梅》当做一首求贤诗呢。

我们来看看古人是怎么解读这首《摽有梅》的。

9、《毛诗序》：“《摽有梅》，男女及时也。召南之国，被文王之化，男女得以及时也。”

10、《诗集传》：“南国被文王之化，女子知以贞信自守，惧其嫁不及时，而有强暴之辱也。故言梅落而在树者少，以见时过而太晚矣 求我之众士，其必有及此吉日而来者乎？”

*书香君👉🏻：* 

> 朱子怎么得出“而有强暴之辱”的呢？即便嫁不及时，又何至于有强暴之辱？

11、清初疑谷派代表姚际恒（著有《诗经通论》《古今伪书考》）、明代学者章潢（著有《诗经原体》）、清代诗经大家方玉润，均认为是求贤诗。

姚际恒说：“此篇乃卿大夫为君求庶士之诗”。

章潢说：“或者诗人伤贤哲之凋谢，故寓言摽梅，使求贤者及时延访之耳。”

方玉润则说：“讽君相求贤也。”方氏的根据是：《尚书》曾以“盐梅和羹”喻贤者，不就跟摽梅的说法一致吗？他并且指出本诗不像是男女词的三个理由：一，咏婚姻不说桃而说梅、不说花而说果；二，求婿不说“吉士”而说“我庶士”；三，亟亟难待，情近私奔。

以上“求贤”说，我们有所了解即可。

古今主流观点是“恨嫁”。

12、龚橙《诗本义》直谓：“《摽有梅》，急婿也。”

13、清代毛诗研究大家陈奂（著有《毛诗传疏》、《毛诗说》）评本诗：“梅由盛而衰，犹男女之年齿也。梅、媒声同，故诗人见梅而起兴。”

> 趁着仲春选良辰。

> 
> 如今枝头仅三成。

> 小伙有心要追我，
> 
> 即日就是好时辰。

> 
> 树下多得拿筐盛。
> 
> 妾本有意郎若允，
