秋日，草木萧瑟、山川寂寥，最易伤情感怀，动离人之思。
一个“秋”字，虽或妙手偶得，确乎入神造微。
雅韵欲流，遂成千秋佳语。

## 🌼 《王风∘采葛》原文：
```

彼采葛（kat）兮；
一日不见，如三月（ngiuat）兮。

```
*书香君👉🏻：* 

> “葛、月”，月部，押韵

```
彼采萧（syu）兮；
一日不见，如三秋（tsiu）兮。

```
*书香君👉🏻：* 

> “萧、秋”，幽部，押韵

```
彼采艾（ngat）兮；
一日不见，如三岁（siuat）兮。

```
*书香君👉🏻：* 

> “艾、岁”，月部，押韵

```
本诗三章，每章一韵。

```
## 🍁️ 骏注：

1、虽然《王风》多悲怨之音，但仍有一首非常著名且缠绵唯美、悱恻感人的爱情诗，那就是今天要介绍的《采葛》。当然《王风》里还有其他几首涉及爱情的诗作，比如我们前已介绍的《君子于役》和《扬之水》，以及之后将要分享的《大车》和《丘中有麻》（后两篇早前也曾发在我的个人公众号“骏语君谈”里，微信查询输入“骏语君谈”仍可以看到）。

但，若论唯美的爱情诗，《王风》中则仅此一首。

2、这首《采葛》，三章仅换四个字，就完成了重章复沓、层层递进的情绪渲染，高效至极。由采葛而采萧而采艾；由三月而三秋而三岁。换字既为押韵也递进了情感的抒发。成语“一日三秋”、“一日不见，如隔三秋”就脱胎于此诗。

3、当代多认为这是一首思念情人的诗。这位被思念的情人可能是一位采摘姑娘。春秋时代，采葛以织布、采萧以祭祀、采艾以疗疾，基本是女子的活计。

4、但，古时可不是这样解读的。今天看旧说，我们也许会觉得有些奇异，有点匪夷所思。让我来介绍一下。

A、《毛诗序》：“《采葛》，惧谗也。桓王之时，政事不明，臣无大小，使出者则为馋人所毁，故惧之。”

*书香君👉🏻：* 

> 《毛诗序》什么意思呢？就是说，在周桓王时代，周室威信已经大降，诸侯争霸开始公开化。周朝政事混乱，出使在外的周王室的臣子们，离开朝廷后都会被人进谗言而遭毁谤，是以臣子们都很害怕出使，一日不在周王身边、一日见不到周王就担心别人在周王面前说自己的坏话。此诗是周桓王时期的作品，作者可能就是周朝的臣。

周桓王是周平王的孙子、周宣王的四世孙，是东周第二任君王。其时郑国国君是郑庄公（郑国第三任国君，就是《郑伯克段于鄢》的郑伯，与母亲黄泉相见的典故说的就是他）。郑庄公是周宣王的侄孙，按辈分，比周桓王还长两辈。周桓王与郑庄公其时交恶，亲率大军伐郑而败，周桓王受了箭伤还差点被俘，周王室自此威望不再，崛起的几个强国已不把周室放在眼里，诸侯争霸也逐渐肆无忌惮了。

B、《郑笺》：“以采葛喻臣以小事使出，以采萧喻臣以大事使出，以采艾喻臣以急事使出，一日不见于君，尤惧于馋矣。”

*书香君👉🏻：* 

> 郑玄把采葛、采萧、采艾分别比作小事、大事和急事。因为葛是做衣服用的，故日常小事也；萧，古时祭祀用，故大事也（古云：国之大事唯祀与戎也）；艾，疗疾之用，疾，急也，故喻急事。

C、朱熹脱离序笺所谓“惧馋”之说，解为情诗（当然，他一以贯之地认为是“淫奔之词”）。是更加贴近诗旨、生活化、鲜活化的一大进步。

其《诗集传》谓：“采葛，所以为絺綌，盖淫奔者托以行也。故因以指其人，而言思念之深，未久而似久也。曰三秋，则不止三月矣；曰三岁，则不止三秋矣。”

D、有意思的是方玉润。他既不同意序笺之解诗（序谓“为惧馋”，尤不足与辩”），更驳斥朱子之言，认为本诗是一首“怀友诗”。

其《诗经原始》直怼朱熹：“《采葛》，怀友也。此诗明明千古怀友佳章，自《集传》以为淫奔者所托，遂使天下后世士夫君子皆不敢有寄怀作也。不知此老何以好为刻薄之言若是！”

他且高度评价本诗：“雅韵欲流，遂成千秋佳语。”

3、首章，“葛”与“月”押韵；二章，“萧”与“秋”押韵；尾章，“艾”与“岁”押韵。一定会有朋友问，这三对字读起来似乎不押韵呀。是的，除了二章的粤语读音还算押韵（粤语“萧”读音如xiu，“秋”读音如cou，韵母iu和ou算得上合韵）之外，首尾两章以今之汉语拼音的读音来看，则完全不押韵。不过，古音押韵是没问题的。按王力《诗经韵读》，“葛”、“月”；“萧”、“秋”；“艾”、“岁”；都是两两押韵的。其中首尾两章的四个韵脚字“葛、月、艾、岁”同属月部，二章的两个韵脚字“萧、秋”则属幽部。我们略微了解就好。

4、葛，一种藤蔓植物，其纤维可织布。
我们之前介绍《周南.葛覃》时曾有解释（可参我2020年12月5日朋友圈）。

萧，俗名香蒿，有香味，古人用以祭祀。

艾，俗名艾草、香艾。古人烧艾叶灸病或煮水洗浴防病。今天仍有此用法。

5、古时在田野、山野采摘之事向由女性任之。少女出嫁前须祭祀祖先以教妇徳。祭品的准备甚至祭祀程序的执掌也由女子承担（召南《采蘋》说的就是此类事，可参我2020年12月24日朋友圈）。故，诗作者当为男性，抒其极思某女子也。“彼”，那个（女子），那位（姑娘），就是男子心仪倾慕的女子。采葛采萧采艾，都是代指该女子在劳作中。男子应该是远远地欣赏着她，并吟唱出心中爱慕的歌。

6、“秋”字尤其用得妙。用“三秋”代指三年（当然也未必确指三年，可以是“很多年”），而不是用“三春、三夏、三冬”，在这里就更见其渲染烘托离情愁绪之功效。秋日，草木萧瑟、山川寂寥（可参欧阳修《秋声赋》），最易伤情感怀，动离人之思。一个“秋”字，虽或妙手偶得，确乎入神造微。

## 🍁️ 骏译：

> 有位俊美好姑娘，
> 
> 看她采葛灵动样。
> 
> 如果一天不见她，
> 
> 好似隔了三月长。

> 那位秀丽好姑娘，
> 
> 看她采蒿俏模样。
> 
> 假如一天不见面，
> 
> 就像已隔三秋荒。

> 有位柔美好姑娘，
> 
> 看她采艾玉手扬。
> 
> 要是一天没遇见，
> 
> 竟似离别三年长。

> （原诗每章三句，我译成每章四句，跟各版本句数都不同。当然，“骏译”只是完成之前设定的“骏注骏译”程序中的“译”，对于这么好的一首诗，我认为没必要也不适合翻成白话，译成白话文反而破坏了意境。抱歉。）
