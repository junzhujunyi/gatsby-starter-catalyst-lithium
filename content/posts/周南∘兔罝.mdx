本诗以“干城”代指智勇的武士，以“腹心”代指忠诚的部下，或开创“借代”这一文学修辞手法的先河。

## 🌼 《周南∘兔罝》原文：
```

肃肃兔罝（tzia），
椓之丁丁（teng）；
赳赳武夫（piua），
公侯干城（zjieng）。

```
*书香君👉🏻：* 

> 首章两韵。一三句“罝、夫”鱼部，押韵；二四句“丁、城”耕部，押韵。

```
肃肃兔罝（tzia），
施于中逵（giu）；
赳赳武夫（piua），
公侯好仇（giu）。

```
*书香君👉🏻：* 

> 二章两韵。一三句同首章；二四句“逵、仇”，幽部押韵。

```
肃肃兔罝（tzia），
施于中林（liəm）；
赳赳武夫（piua），
公侯腹心（siəm）。

```
*书香君👉🏻：* 

> 尾章两韵。一三句同首章；二四句“林、心”，侵部押韵。

## 🍁️ 骏注：

1、今人多认为这是一首礼赞猎人或武士的诗作。诗以“干城”代指智勇的武士，以“腹心”代指忠诚的部下，或开创“借代”这一文学修辞手法的先河。

2、序笺及集传之旧说种种，或失之于曲衍附会。书香君分述如左：

3、《毛诗序》：“《兔罝》，后妃之化也。《关雎》之化行，则莫不好德，贤人众多也。”

*书香君👉🏻：* 

> 自《关雎》始，毛序每篇皆言及后妃（下篇《罘苢》亦然，序谓：言后妃之美也），真不知猎人、武夫又与后妃何涉？愚以为毛公实以教化而曲加附会也。

4、《郑笺》进一步分章阐述：“罝兔之人，鄙贱之事，犹能恭敬，则是贤者众多也。干也，城也，皆以御难也。此罝兔之人，贤者也，有武力，可任为将帅之德，诸侯可任以国守，扞城其民，折冲御难于未然。怨耦曰仇。此罝兔之人，敌国有来侵伐者，可使和好之。于行攻伐，可用为策谋之臣，使之虑无，亦言贤者。”

5、朱子步《毛序》以言。其《诗集传》曰：“化行俗美，贤才众多。虽罝兔之野人，而其才之可用犹如此，故诗人因其所事以起兴而美之。而文王德化之盛，因可见矣。”

6、当然，有清一代之学人，不乏跳出旧说窠臼者。如方玉润《诗经原始》评《兔罝》：毛序、集传所言，皆属虚衍附会、毫无征实。此必羽林卫士，扈跸游猎，英姿伟抱，奇杰魁梧，遥而望之，无非公侯妙选。

崔述《读风偶识》：世胄常蹑高位，而寒畯苦无进身之阶。文士或间一遇时，而武夫犹难以逢世。是故，诗人惜之曰：此林中之施兔罝者，其才智皆公侯之干城、腹心也。惋惜之情，显然意外。

崔述读出了诗中的惋惜之意。

7、肃，读音suo阴平，通“梭”。肃肃，严密貌、整饬貌。

罝，读音ju阴平，猎网，捕猎的设备、装置、机关。

肃肃兔罝：严密地布置、捯饬猎网。或者：（多么）密实的猎网（啊）！

8、兔，有些版本解释成“兔子”，有些释为“老虎”，我认为“老虎”似更合理。我们知道周南的核心区域在江汉一带，楚地古时称老虎为“於菟”。结合下文判断，应该是捕捉猛兽。

9、椓，读音zhuo阳平。敲击、敲打、击打（桩、柱）。

10、丁丁，象声词。伐木声、打桩声。丁，古读teng（王力《诗经韵读》），有些版本解作：读zheng阴平。

11、公侯，指国君。西周以公、侯、伯、子、男爵位封列国国君（是不是跟古代欧洲相似啊？）。

12、干城。干，读阴平，垣。可释为盾牌、屏障、城防。城，城墙。干城，代指守卫者。

13、施，读yi去声。布置、设置。又如《周南.葛覃》（请参我12月4号朋友圈）有“葛之覃兮，施于中谷”句，其中的“施”也读yi，词义引申为“分布”、“蔓延”。

14、中逵，逵中也。大路的分岔口。逵，泛指大道，九达之道曰“逵”。顺便提一下，逵，古读如giu，跟下文的“逑”押韵的。不过，现代汉语没这种发音，我们了解就好。

15、仇，读qiu上声。通“逑qiu阳平”，配对的人、配偶（如“君子好逑”）。可释为同道、密友，也可引申为助手、帮手。

16、中林，林中也。密林深处。诗经时代，这种现在看来倒装的用法很多，如上文“中逵”；如《邶风.式微》“胡为乎中露”句，“中露”就是“露中”；又如前“骏注7”所说的《周南.葛覃》“施于中古”句，“中谷”就是“谷中”。

17、腹心，心腹、亲信也。不用“心腹”而用“腹心”，一是古时用词习惯，一是因为“心”在句尾、与“林”押韵。另外，《诗经》的重章叠唱为了押韵也经常换字。

18、本诗每章都是一三句押韵，二四句押韵。这也许就是男女声或者两个声部对唱、轮唱的方式，每个声部自己押韵。韵脚：每章的一、三句尾“罝”和“夫”押韵；每章的二、四句尾分别是“丁”与“城”、“逵”和“仇”、“林”与“心”押韵。
