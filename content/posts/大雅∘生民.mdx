踩脚印这事，要是弄好了，可以繁衍一个伟大民族。记住，要踩在大脚趾头位置喔，呵呵。

本篇太长。不过书友们如果愿意把它读完，哪怕快速浏览一遍，也许多少有点收益吧。



## 🌼 《大雅∘生民》原文：

```
厥初生民，
时维姜嫄。
生民如何？克禋克祀，以弗无子，
履帝武敏，
歆攸介攸止。
载震载夙，
载生载育。
时维后稷。
```

*书香君👉🏻：*

> “民，嫄”，真元合韵，押韵；“祀，子，敏，止”，之部，押韵；“夙，育，稷”，觉职合韵，押韵。

```
诞弥厥月，
先生如达。
不坼不副，无菑无害。以赫厥灵，
上帝不宁。
不康禋祀，
居然生子。
```

*书香君👉🏻：*

> “月，达，害”，月部，押韵；“灵，宁”，耕部，押韵；“祀，子”，之部，押韵。

```
诞寘之隘巷，牛羊腓字之；
诞寘之平林。
会伐平林，
诞寘之寒冰。
鸟覆翼之。
鸟乃去矣，
后稷呱矣。
实覃实訏，
厥声载路。
```

*书香君👉🏻：*

> 注意，第一句“字”与第五句“翼”，之职通韵，押韵；“林，林，冰”，侵蒸合韵，押韵；“去，呱，訏，路”，鱼铎通韵，押韵。

```
诞实匍匐，
克岐克嶷，
以就口食。
蓺之荏菽，荏菽旆旆，禾役穟穟。
麻麦幪幪，
瓜瓞唪唪。
```

*书香君👉🏻：*

> “匐，嶷，食”，职部，押韵；“旆，穟”，月物合韵，押韵；“幪，唪”，东部，押韵。

```
诞后稷之穑，有相之道。
茀厥丰草，
种之黄茂。
实方实苞，
实种实褎，
实发实秀，
实坚实好，
实颖实栗，
即有邰家室。
```

*书香君👉🏻：*

> “道，草，茂，苞，褎，秀，好”，幽部，押韵；“栗，室”，质部押韵。

```
诞降嘉种，维秬维秠，维穈维芑。
恒之秬秠，
是获是亩；
恒之穈芑，
是任是负。
以归肇祀。
```

卬盛于豆，于豆于登，其香始升，

*书香君👉🏻：*

## 🍁️ 骏注：

1、这首《生民》，是《大雅.生民之什》的第一篇。与我们之前介绍的《绵》一样，也是周民族的史诗之一。只不过《绵》是歌颂古公亶父的，而《生民》所歌颂的人物则要早得多，早得不能再早，早到源头了。这位被歌颂的人，是周氏族的真正始祖——后稷。他是帝喾和姜嫄的儿子，名“弃”，也是“挚”、“尧”和“契”的兄弟。

（帝喾又是什么人呢？简单说一下，大家略有了解就好。按夏代及夏之前的传说——这些传说只是传说，且是后世记载，多为编造附会——帝喾是黄帝的曾孙。中华始祖母华胥氏生伏羲、女娲；伏羲女娲兄妹婚，生少典；少典生黄帝；黄帝生长子玄嚣、次子昌意；玄嚣生蟜极；蟜极生帝喾。我另有一篇小文《致敬诗经之史前神话和夏商周人物关系管窥》，发于2020年11月17日朋友圈，可参）。

这四兄弟同父但都不同母。挚是帝喾的首个接班人，华夏民族的首领，后传位给尧。“契”这个人则是尧舜时代的“火正”、殷商的始祖。“弃”呢，是尧舜时的农官（稷），被舜封在邰地（“弃”的母亲姜嫄乃有邰氏之女，娘家氏族在邰地）。“后”，王也；“稷”，农官也，善稼穑也，“弃”因而有“后稷”之名。稷，事实上是一种谷物，据考证，就是高粱或小米。应该是当时最好的谷物（黄色谷物是嘉谷——本诗曰“黄茂”）。以嘉谷称谓农官再合理不过了。我在《浅谈诗经之古远丰美》里曾经介绍过，这里就不再细说。

2、所谓“生民”，就是“民之生”，周先民、周氏族始祖之降生也。可译成“后稷的降生（诞生）”。后稷善稼穑，因以为稷。而“周”字，在甲骨文中“口像四周田界，其中阡陌纵横，像田中所植。田，言种植之地，周字直作田。”也就是说，最早的“周”字是指有边界且满种庄稼的农田，田里种满了庄稼。后稷善稼穑，其后人以农耕为生，是谓周人（当然，周人之“周”，尚有其他解释，此处不表）。

3、商朝始祖“契”和周朝始祖“弃”（后稷）是兄弟，契和弃的出生都被附会了神话色彩，以阐释其大命天降的“合法性”，其实都是编造的，其来源是古时的祭祀礼和求子风俗，也一定程度反映了母系社会的传承遗迹。

4、本篇描述了后稷的出生异象（母亲如何怀孕，出生如何不同常人，如何被弃而天佑之）、如何富有稼穑天赋以及周氏族开初时的农耕生活——除草、播种、禾谷生长、成熟、收割、运输、打谷、炊饮、祭祀。是最早的后稷赞美诗。叙述生动形象，语汇丰富优美，极受后世称道。

历代评点：

《毛诗序》：“《生民》，尊祖也。后稷生于姜嫄，文、武之功起于后稷，故推以配天焉。”

《毛传》：“生民，本后稷也。姜，姓也。后稷之母，配高辛氏帝焉。”

书香君：高辛氏，即帝喾。帝喾封于高辛，故名。

《郑笺》：“厥，其。初，始。时，是也。言周之始祖，其生之者，是姜嫄也。姜姓者，炎帝之后，有女名嫄，当尧之时，为高辛氏之世妃。本后稷之初生，故谓之生民。”

《诗集传》：“周公制礼，尊后稷以配天，故作此诗，以推本其始生之祥，明其受命于天，固有以异于常人也。然巨迹之说，先儒或颇疑之。”

书香君：朱熹本来已经质疑“巨迹之说”（所谓踩了天帝脚印而孕生后稷）了，但是他后面又引其他学者如苏辙“神人之生，而有以异于人，何足怪哉！”之说，认为苏辙“斯言得之矣”，还是没有逃出“君命神授”的谬论。

《诗经原始》：“《生民》，述后稷诞生之异，为周家农业之始也。一章受孕之奇；二章诞生之易；三章保护之异；四章嗜好天生；五章克勤人事，教种应封；六章播种肇祀；七章报赛祈年；八章尊祖无怠。通篇层次井然，不待深求而自了了。”

书香君：方玉润是按八章划分的，我们现在是划分为九章的（第七、第八两章朱熹、方玉润是并作一章的，当代也仍然有人分为八章）。

5、“厥初”，当初。

6、“时”，是，乃。

7、“厥初生民，时维姜嫄”，意为“当初生育周始祖的，就是姜嫄”。

8、“克”，能。

9、“禋”，读音yin，阴平。一种祭祀仪式。

10、“弗”，“祓”的借字，祈子除灾求福的祭祀典礼。禋，祀，弗，都是指的“郊禖”（也称“高禖”）求子仪式，是上古向主生殖的神祈祷的仪式。

但是方玉润质疑：既然求子，后又得子了，为何要数弃之？不通啊。对“以弗无子”如何解读很关键。他认为郑玄解为“祓除其无子之疾”、姚际恒解为“弗使其无子”都是不通的；邓潜谷、季明德解为“姜嫄未嫁而生子”是对的。“以弗”，以其弗嫁，未字于人也。“无子”，以其未字于人，故尚无子也。

我们可做参考。

11、“履”，践、踩、覆盖。“帝”，上帝。一说帝喾，就是后稷的父亲。“武”，足迹。“敏”，通“拇”，指脚的拇趾。

“履帝武敏”整句就是“踩到了帝的大脚印的拇趾处”。

12、“歆攸介攸止”，有些版本把“歆”这个字放在上句，断句为“履帝武敏歆，攸介攸止”，而我采信的是“履帝武敏，歆攸介攸止”这种断句方式。“歆”，愉悦，心有所感的样子，此指姜嫄受孕的心理生理感受。“介” 通“愒”，息也。“止”，止息。

“歆攸介攸止”这句，按林义光、闻一多的解读，是“舞毕而相携止息于幽闭之处，因而有孕也。”是求子者与主生殖的神交合而受孕（主祭祀者以生殖神的名义）。但后世大多传说姜嫄在郊禖时踩到上帝（或帝喾）的大脚印而受孕生后稷。后世的这些解释，其实都是附会。我之前说了，这是周氏族后人为了宣扬祖先的“天降大任”和执政合法性乃“天授”，有意编造的。类似的编造同样发生在殷商始祖“契”的身上（他是母亲简狄吞食玄鸟蛋怀孕而生的）。闻一多先生认为古时求子仪式，主持祭祀者以“帝（主生殖的神）”附身（曰“尸”），跳一种舞，求子者跟跳（当然会履踩尸之脚印），然后与“帝”交合得孕。我倾向于认同此说。我们之前分享《螽斯》时也解释过远古祭祀的“禹跳”，一种舞步，说明祭祀时是经常会跳舞的。原始的舞蹈来源于先民们劳作时和祭祀时的动作。人类是有性繁殖，怎可能踩脚印就怀孕呢？事实上一定是男女交合而有孕，为了某种忌讳而编造神话又不断发展这个神话，说穿了都是一种愚民手段，是握有权力者乐于做的，其精髓传之弥远以至于今。

13、当然，如果按常规解释，“介”就解为通“祄”，神明保佑也。“止”就解为通“祉”，神明降福也。

14、“载震载夙”，“震”，指姜嫄怀孕时的生理异样感受，一说通“娠”，怀孕。还有人解为“长子”。“肃”，严肃、端庄。指怀孕时谨慎启居。一说是怀孕的时间概念。

15、“诞”，发语词，有叹美之味。一说，通“迨”，到了。“弥”，满。指怀孕足月。

16、“先生”，头生，第一胎。这个“头生”又引致后人对后稷为何生下来遭“弃”很多解释，如头生子必须遗弃等怪论。

“如”，而，古时“如、而”互通。

“达”，通“羍”，初生的羊羔、羊胎、小肉蛋之类。喻指后稷出生时的外形与众不同。一说“滑利”，喻指生产很顺利。

17、“坼”，读音che，去声。裂开。

“副”，读音是pi，去声。破裂。

18、“菑”，“灾”的古写，读音也是zai。

“不坼不副，无菑无害”指生产顺利。

19、“不”，丕，大也。

“不宁”，丕宁，大宁。同样地，“不康”，丕康，大康也。喻示后稷出生，周的伟业会大康大宁。但，另有一解，“不”就当作否定词，以喻后稷出生之不祥，故弃之。两说都有道理，就看怎么理解后稷被“弃”这件事。如果是姜嫄因为生了怪胎害怕而弃之（有这种说法），那就是后一种解读合理。如果弃只是一种考验仪式，况且数弃皆有上天庇护，分明是神佑之人，那又何来不祥？所以具体怎么翻译就看个人理解了。存疑。

相应地，“居然”一词，配合“不”字的前解，就理解成“果然”；配合后解，就理解成“徒然”。

20、“寘”，解释过，置也。此处应为“弃置”。后稷之名曰“弃”，就是生而被数弃之。

21、“腓”，解释过，通“庇”，庇护。

“字”，哺育。

22、“实”，是，如此。

“覃”，长。

“訏”，大。

“实覃实訏”指后稷哭声真是悠长且响亮。

“实”，也含叹美之味。

23、“载”，充满。

24、“克”，能。

“岐”，知意。

“嶷”，通“逆”，逆略，识别。

“克岐克嶷”指后稷既聪明又有认知。

“以就口食”，“以”，通“已”；“就”，求。“口食”，指食物。

“诞实匍匐，克岐克嶷，以就口食”这三句说的是，后稷很小只能爬的时候就会自己找到吃的了。喻示其聪颖非常。

25、“旆旆”，草术茂盛。如我们之前说过的“肺肺”、“芾芾”。

26、“役”，通“颖”，禾苗之末。

“穟穟”，禾穗饱满下垂貌。

27、“幪幪”，茂密貌。

28、“唪唪”，果实累累貌。“唪”，读音feng上声；一说读beng上声。

29、“有相之道”，有判断土地是否适合种植和作物好坏的技能、方法。

30、“茀”，通“拂”，拔除。

31、“黄茂”，嘉谷，优良品种，即黍、稷类黄色谷物。

《孔疏》：“谷之黄色者，惟黍、稷耳。黍、稷，谷之善者，故云嘉谷也。”

32、“实”，是，乃。

“方”同“放”，萌芽出土。

33、“种”，禾芽初长，一说肥盛、矮壮。

34、“褎”，读音you，去声。禾苗长高。

35、“发”，禾苗抽茎、拔节。

“秀”，结穗。（我们在《豳风.七月》里也解释过“秀葽”）。

36、“坚”，谷粒灌浆饱满。

37、“栗栗”，谷物众多貌。

“即”，到。当指后稷因善于农事而被舜封于邰地。

“即有邰家室”可以理解为“后稷封于邰地成家立业”。

我们可以看到，本诗用“方，苞，种，褎，发，秀，坚，好，颖，栗”描述了庄稼从破土出秧到嫩苗初长，到长高长壮，到拔节结穗，到灌浆饱满，到谷穗低垂，到谷物丰收的全过程，用字简洁而内涵丰满。

38、“降”，赐与。

39、“秬”，读音ju，去声。黑黍。

“秠”，读音pi，阴平。一种黍。

40、“穈”，读音men，阳平。红米。

“芑” ，白米。

我们还可以看到，本诗涉及到的谷物种类已经很多，有“荏菽（大豆）、麦、麻、瓜、秬、秠、穈、芑”。说明周人很早就有多种农作物的种植实践。

41、“恒”，通“亘”，遍。

42、“亩”，按今天的说法就是名词动化，意为“堆在田亩”。

43、“任”，挑起。

“负”，背起。

44、“揄”，读音you，阳平。

舀，从臼中取出舂好的谷子。

45、“簸”，扬谷。

“蹂”，搓谷壳。

46、“释”，淘米。

“叟叟”，指淘米的声音。

47、“烝”，同“蒸”。

“浮浮”，热气升腾貌。

48、“惟”，考虑，思忖。

49、“萧”，艾蒿，香蒿。

“脂”，牛油，祭祀时燃柴用。

50、“羝”，小公羊，祭祀之牺牲。

“軷”，通“拔”，剥去羊皮。

51、“燔”，读音fan，阳平。烤制。

“烈”，串烤。

我们又可以看到，本诗用“舂、揄、簸、蹂、释、烝、軷、燔、烈”来描述处理、加工谷物和肉类的动作、程序。词汇丰富。

52、“嗣岁”，来年。

53、“卬”，通“仰”，举。

一说，我。“卬”本就可以通“我”和“仰”，《诗经》里都出现过，如《邶风.匏有苦叶》有“人涉卬否，卬须我友”句，“卬”就是“我”；《大雅.瞻卬》有“瞻卬昊天”句，“卬”就是“仰”的借字，“瞻卬”就是“瞻仰”。本诗“卬”两解都通。

54、“豆”，木制的容器，主盛肉食。“笾”，竹制的容器，主盛果品。笾、豆一般连用。比如我们之前介绍过的“笾豆有践”。

55、“登”，瓦制容器。

56、居歆（xīn）：为歆，来享用，安享。

57、“胡臭亶时”，正当香气浓烈之时。

“胡”，大也。

“臭”，读xiu，去声，香气；

“时”，得其时。

58、“庶”，幸，幸亏。
