本诗为我们描摹了《诗经》时代先民们质朴美妙的爱情画卷。
风神摇曳，姿态横生。

## 🌼 《野有死麕》原文：
```

野有死麕（kyuən），
白茅包（peu）之；
有女怀春（thjiuən），
吉士诱（jiu）之。

```
*书香君👉🏻：* 

> 本章两韵。一三句“麕、春”文部，押韵；二四句“包、诱”幽部，押韵

```
林有朴樕（sok），
野有死鹿（lok）；
白茅纯束（sjiok），
有女如玉（ngiok）。

```
*书香君👉🏻：* 

> 二章一韵。“樕、鹿、束、玉”，屋部，押韵

```
舒而脱脱（thuat）兮，
无感我帨（sjiuat）兮，
无使尨也吠（biuat）。

```
*书香君👉🏻：* 

> 尾章一韵。“脱、帨、吠”，月部，押韵

## 🍁️ 骏注：

1、“麕”，有些版本作“麇”。意思一样。关于这个字，我们多说几句。“麕”，通“麇”。先秦时期常用，一般用作鹿类动物的总称。也指獐子。獐，古时写作“麞”（读音zhang阴平）。鹿而美者曰麞。章，就是华美。文章的章也是这个意思。麕（麇）的读音有两个，分别是jun阴平和qun阴平。读qun时，又可通“群”，故有“麇集”、“麇居”之类的雅词。《尔雅·释兽》：“麕牡麌，牝麜，其子麆”。什么意思呢？就是说，雄性的麕称麌（读音yu），雌性的称麜（读音li），小麕称麆（读音zhu）。《埤雅》：“麕性胆尤怯，饮水，见影辄奔”。麕，胆子小，喝水时都能被自己的影子吓到而惊跑。麕的这个特性引申出“麇惊”这个词以及类似的词汇，好美呀，是不是联想到“翩若惊鸿”？大雁（天鹅）受惊吓猝然起飞翅膀张开而盘旋，美极了喔。此外，先秦时期表示鹿类动物的词汇还有很多，就不一一列举。其中有个字“麈”（读音zhu），很有名，我在这里说两句。麈，指大鹿。文人雅士常拿麈尾毛做“拂尘”（即如今之鸡毛掸子之类），故有雅词“麈尾”，指的就是“拂尘”。又因为群鹿同行时，麈尾摇动指示方向，故“麈尾”又可引申为“领袖群伦”。东晋名士王导还专门作过一篇《麈尾铭》。白居易“老翁持麈尾，坐拂半张床”、欧阳修“抱琴不暇抚，挥麈无由停。”都说的是麈尾。古代文豪们的词汇量由此可见一斑。

2、本诗是《诗经》里著名的爱情诗。男子是一位青年猎人，女子是山林人家的姑娘。诗情活泼、奔放，为我们描摹了诗经时代先民们自由热烈又美妙的爱情生活画卷。

3、这样一首我们今天看来比较明确的爱情诗，在古时却有各种解读，有些太过牵强，有些则因历史局限所致，现在读来不禁哑然。你品——

《毛诗序》：“野有死麕，恶无礼也。天下大乱，强暴相陵，遂成淫风。被文王之化，虽当乱世，犹恶无礼也。”

《郑笺》：“乱世之民贫，而彊暴之男，多行无礼，故贞女之情，欲令人以白茅裹束野中田者所分麕肉，为礼而来。…吉士使媒人道成之，疾时无礼而言然。贞女欲吉士以礼来，脱脱然舒也。又疾时无礼，彊暴之男相劫胁。”

朱熹《诗集传》：“南国被文王之化，女子有贞洁自守，不为强暴所污者。故诗人因所见以兴其事而美之。…乃述女子拒之之辞，言姑徐徐而来，毋动我之帨，毋惊我之犬，以甚言其不能相及也。其凛然不可犯之意，盖可见矣。”

王先谦《诗三家义集疏》：“韩说曰：‘平王东迁，诸侯侮法男女失官昏之礼，《野麕》之刺兴焉。”

姚际恒评曰：“山野为婚”。

欧阳修认为是“刺淫诗”。

章潢（其人上文有介绍）：“诗人托言怀春之女，以讽士之炫才求用，而又欲人勿迫于己。”

范处义（南宋诗经大家，著有《诗学》、《诗补传》）：“女子之德洁白如玉，不可犯以非礼。白茅纯束，亦以比德。”

《诗经原始》：“《也有死麕》，拒招隐也。此诗，意极深而词又甚婉。此必高人逸士抱璞怀贞，不肯出而用世，故托言以谢当世求才之贤也。”

4、“白茅”，草名。属禾本科，在阴历三四月间开白花。有韧劲，可捆扎。初生的白茅曰荑，白且柔，也用于祭祀、庆典等场合。老化后耐腐，可铺屋顶，“茅屋”是也。“包”，古读如pou或bou上声。男子猎取野麕并以白茅包裹好，当做礼物送给女子表明心迹。也是一种尊重、一种郑重其事的行为。我们可以设想，该猎人经常到这一带山林打猎，多次遇到居住此地的一家人，这家的女儿与猎人互生好感乃至爱慕。终于青年忍不住要表白了，拿什么做表白甚至提亲的礼物呢？打一只麕吧。古时，鹿或者类似的动物，可是正经百八的求婚、结婚礼品呢。用白茅细心捆扎，很正式。开头一句“野有死麕”，先声夺人，暗示猎人就在附近、即将出现。因为，总不至于无缘无故地，野地里会有一只死麕吧？应该有人在捕猎。诗篇一开头就很抓人，好手法。

5、“吉士”，未婚男子的美称。相当于今之“好小伙、好青年、帅哥”之类。

6、“诱”，很多版本解为“挑逗”，我觉得更象是互有好感状态下的搭讪、调情。

7、“朴樕”，丛生小树，灌木。亦可写成“朴遬”。樕，读音su去声。朴樕，叠韵词，多说几句。扶苏、扶疏、扶胥等词也由“朴樕”分化而来。灌木丛生貌曰“朴樕”，花枝丛生貌曰“扶苏”（诗经郑风就有一篇《山有扶苏》以后要讲，嬴政的长子就因此诗取名扶苏），鸟兽羽毛丛生貌曰扑朔。比如木兰辞：“雄兔脚扑朔雌兔眼迷离”，扑朔最早也是雄兔脚扑腾而兽毛乱杂杂的，后面才有今意。朴樕可为柴薪，也暗喻情事婚事。

8、纯（tun阳平）束：捆扎，包裹。三家诗作“屯”。“纯”、“屯”，都是“稇”（读音kun上声）的借字。

9、“脱”，此处应读（tui去声。此作形容词，舒缓貌、舒迟貌，指动作文雅舒缓。

10、“感”，通“撼” ，也要读han去声。动，动摇，掀开。

11、“帨”，读音shui去声。又名袆（hui去声），佩巾。或者就是女子的一种服装，系在女子腹前。有人解为蔽膝或围裙都是不准确的 。

12、“尨”，读音mang阳平。多毛的狗，当指女子家的看家护院的小狗。有解为猛狗的，未必妥。

13、你侬我侬。女孩既高兴又紧张，既欢娱又害羞。多美的画面！
