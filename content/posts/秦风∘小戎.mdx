方玉润盛赞本诗：“刻划典奥瑰丽已极，西京诸赋迥不能及，况下此者乎！”。




## 🌼 《秦风∘小戎》原文：

```
小戎俴收（sjiu），
五楘梁辀（tjiu），
游环胁驱（khio）。
阴靷鋈续（ziok），
文茵畅毂（kok），
驾我骐馵（tjiok）。
言念君子，温其如玉（ngiok）。
在其板屋（ok），
乱我心曲（khiok）。
```

*书香君👉🏻：*

> 首章两韵。“收、辀、驱”，幽侯合韵；“续、毂、馵、玉、屋、曲”，屋部，押韵

```
四牡孔阜（biu），
六辔在手（sjiu）。
骐駵是中（tiuəm），
騧骊是骖（tsəm）。
龙盾之合（həp），
鋈以觼軜（nəp）。
言念君子，温其在邑（iəp）。
方何为期（giə），
胡然我念之（tjiə）？
```

*书香君👉🏻：*

> 二章四韵。“阜、手”，幽部；“中、骖”，侵部；“合、軜、邑”，缉部；“期、之”，之部

```
俴驷孔群（giuən），
厹矛鋈錞（zjiuən），
蒙伐有苑（iuan）。
虎韔镂膺（iəng），
交韔二弓（kiuəng），
竹闭绲滕（dəng）。
言念君子，载寝载兴（xiəng）。
厌厌良人，秩秩德音（iəm）。
```

*书香君👉🏻：*

> 尾章两韵。“群、錞、苑”，文元合韵；“膺、弓、縢、兴、音”，蒸侵合韵


## 🍁️ 骏注：

1、今天这篇《小戎》，是风诗里生僻字较多的一首。我先集中介绍下：

A、俴，jian去声，意思是“浅”。

B、楘，mu去声，皮革或金属（多为铜，古时铜较早被冶炼出来）制成的环形圈、环形箍。

C、辀，zhou阴平，弓形的车辕。象架屋的木梁，故称梁辀。

D、靷，yin上声，马拉车的牵引绳或皮带。一头系于马颈脖，一头系于车轴或车轼横板。

E、鋈，wu去声，白铜，可能是古时一种铜合金。

F、馵，zhu去声，左后足白色的马。

G、骐，毛色青黑色块相间的马，状如棋格，故名。

H、駵，liu阳平，通“骝”，黑红色的马。有些版本就作“骝”。

I、騧，gua阴平，体黄嘴黑的马。

J、骊，黑马。

K、觼，jue阳平，带舌的环。

L、軜，na去声，骖马的内辔。上篇和本篇都有“六辔在手”这句，指驭者手执驷马的六条缰绳。四马，每马两辔，四马本是八辔。两匹骖马的各一条内辔，叫“軜”，其尾端有个环，连接车上的固定环。这两条内辔（軜）是不用驭者手握的。所以驾驭四马，驭者“六辔在手”。

M、厹，qiu阳平，三棱矛，长丈八。

N、錞，dui去声，矛底端的装饰，多为白铜（鋈）制成。

O、韔，chang去声，弓套。

P、緄，gun上声，绳。

Q、縢，teng阳平，捆扎。

2、本诗，当代多解为妻子思念远征西戎的丈夫的诗作。还有传是尹吉甫前妻（抑或女友）仲姬所作呢。

3、旧说是赞美秦襄公，或是怀西征将士之诗。

4、《毛诗序》：“《小戎》，美襄公也。备其兵甲，以讨西戎。西戎方彊，而征伐不休，国人则矜其车甲，妇人能闵其君子焉。”

《毛传》：“矜，夸大也。国人夸大其车甲之盛，有乐之意也。妇人闵其君子，恩义之至也。作者叙外内之志，所以美君政教之功。”

5、虽然秦襄公及其历代前任均以与西戎作战为主要工作，但本诗并未出现“西戎”或之类的词，为何说是讨伐西戎的作品呢？朱熹给出了解释——诗中的“板屋”：“板屋者，西戎之俗，以板为屋”。就是说西戎是住板屋的。可能是游牧民族为迁徙方便而采用的临时居住方式。

《诗集传》解本诗：“西戎者，秦之臣子所与不共戴天之雠也。襄公上承天子之命，率其国人往而征之，故其从役者之家人，先夸车甲之盛如此，而后及其私情。盖以义兴师，则虽妇人亦知勇于赴敌，而无所怨矣。”

6、方玉润认为本诗出自襄公之手笔。谓襄公“怀西征将士也”，乃劳大夫征伐之诗。

他批评序笺解诗既谓“国人矜其车甲”，又云“妇人闵其君子”，是“一诗两义”。“中间并无递换，上下语气全不相贯，天下岂有此文义？”

方玉润引明代学者邹肇敏之言：凡劳诗，或代为其人言，或代为其室家言。而此诗“言念君子”（三章皆有“言念君子”句——书香君注），则襄公自念其臣子也。

方氏并举了宋代开国大将王全斌伐蜀时受宋太祖赐裘帽的例子——王全斌伐蜀正值寒冬，太祖在汴梁，衣紫貂裘帽坐于毡帷，谓左右曰：我被服如此尚觉寒冷，西征将士何以堪处？即解裘帽令中黄门速送千里之外全斌以示慰劳。

按方氏之见，本诗“在其板屋，乱我心曲”，是襄公自语，念如玉之君子身处板屋而归期不确，其何以堪？与宋太祖赐裘帽与全斌同理心也。

方玉润并嘲讽道：后儒不察，以为从役者之家人所言，则襄公劳士一片苦衷，不几为其所没？千载下谁复能谅之耶？

方玉润端的犀利不饶人。

7、今人程俊英、蒋见元同意方玉润对序笺的批评，但是不同意方玉润“襄公怀西征将士”之说，指出“言念君子”、“厌厌良人”等语，均为女子口吻，与襄公扯不上，还是应该解为“妇人思念西征的丈夫”。

他俩的这个分析书香君认同。

8、戎，兵车。小戎，小兵车。因车厢较小，故称小戎，兵士所乘。古时作战，大战车先行，谓之“大戎”（或“元戎”），小战车随后，谓之“小戎”。

《毛传》：“小戎，兵车也。”

《孔疏》：“元戎十乘，先以启行。”

9、收，轸。围成车厢的四周横木。四面束舆之木谓之轸，轸分四边。古人登车多从车后，车后之轸较低，以利人员登车，故本诗之“收”当为车后之轸，故言“俴收”，也就是“浅收”、较低的轸。

《毛传》：“收，軫也”

《诗集传》：“收，軫也，谓车前后两端横木，所以收敛所载者也。凡车之制，广皆六尺六寸。其平地任载者为大车，则軫深八尺。兵车则軫深四尺四寸。故曰小戎俴收也。”

10、五，用皮革交叉缠在车辕成“X”形，起加固和修饰作用。五，古文作X。

《毛传》：“五，五束也。”

11、协驱，皮带，上系于车衡木，后系于轸木，以控制骖马向内偏。

12、续，接续

13、文，斑斓、花纹。茵，坐垫，车凳上的褥子。文茵，就是有花纹的坐垫，多为虎皮豹皮制成（也可能正是这些兽皮有花纹，故曰“文”）。

14、畅，长。

15、言，乃。

16、君子，指从军的丈夫。

17、温其如玉，女子夸美自己丈夫性情温润如玉。

18、心曲，心灵深处。

19、牡：公兽，此处指公马。

20、孔，很、甚、非常，表程度的副词。

21、阜，肥大、壮硕。

22、龙盾，画有龙图像的盾牌。

23、合，两只盾合挂于车上。

24、邑，秦国的属邑。

25、方，将。

26、期，指归期。

27、胡然，为什么。

28、俴驷，披浅薄金甲的驷马。一说不挂甲的驷马。

29、孔群，很合群。指群马很协调。

30、蒙，盾上刻画杂乱的羽纹。

31、伐，通“瞂”，盾。

32、苑，此处指花纹。

33、镂膺：在弓囊前刻花纹。

34、交韔二弓，两张弓，一弓向左，一弓向右，交错放在袋中。交：互相交错；韔，此处名词动化，用作动词，“放入（弓袋）”，“藏于（弓袋）”。

35、闭，弓檠（qing阳平）。竹制的校正弓箭的用具。

36、载寝载兴，又寝又兴，起卧不宁。

37、厌厌，安静柔和貌。

38、良人，指女子的丈夫。《唐风.绸缪》里介绍过“良人”。一般是妇人美称自己的丈夫。

39、秩秩，有礼节，一说聪明多智貌。

40、德音，好的声誉。

41、前已述及，本诗今人多认为是一首妻子怀念征夫的诗。

首章回忆丈夫出征时的壮观场面；次章尾章想象丈夫出征后的情景，回味丈夫的温暖，期盼丈夫建功凯旋。字里行间充满着对丈夫的仰慕和思念。

42、这首诗体现了《秦风》尚武的特点，并且看不到后世作品中常有的哀怨情绪。

43、每章的前六句赞美秦师壮观阵容，后四句抒发女子思君情意。其状物言情，少有雷同；其抒怀喻人，一气贯通。

方玉润也盛赞本诗：“刻划典奥瑰丽已极，西京诸赋迥不能及，况下此者乎！”。

书香君：方氏所言之“西京诸赋”，当指东汉张衡《西京赋》、《东京赋》和班固《西都赋》。

44、本诗的诗旨，除了前面已经介绍的几种之外，也还有以下几种认同者不太多的解读：

A、赞美秦庄公（《诗古微》）；

B、慰劳征戎大夫（《诗传》）；

C、伤王政衰微（《诗故》）；

D、是一首军歌（《毛诗复古录》）；

当然，现代学者多认为是“女子怀念征夫”之作。
