脚印不好乱踩的，
鸟蛋就能乱吞吗？

《生民》说周民族的先祖是其母踩了巨人脚印而孕生的；本篇又附会了殷商先祖是其母误呑鸟蛋而孕生。


## 🌼 《商颂∘玄鸟》原文：

```
天命玄鸟，降而生商（sjiang）。
宅殷土芒芒（mang）。古帝命武汤（thang），正域彼四方（piuang）。
```

*书香君👉🏻：*

> “商，芒，汤，方”，阳部ang，押韵

```
方命厥后，奄有九有（hiuə）。
商之先后，受命不殆（də），
在武丁孙子（tziə）。
```

*书香君👉🏻：*

> “有，殆，子”，之部ə，押韵

```
武丁孙子，武王靡不胜（sjiəng）。
龙旂十乘（djiəng），大糦是承（zjiəng）
```

*书香君👉🏻：*

> “胜，乘，承”，蒸部əng，押韵

```
邦畿千里（liə），
维民所止（tjiə）。
肇域彼四海（xə）。
```

*书香君👉🏻：*

> “里，止，海”，之部ə，押韵

```
四海来假，来假祁祁（giei）。
景员维河（hai）。
殷受命咸宜（ngiai），百禄是何（hai）。
```

*书香君👉🏻：*

> “祁，河，宜，何”，脂（ei）歌（ai）合韵

> 说明一下：本诗二十二句，只是单章。我为划分韵部的方便，把它按韵断开了，不是分章。


## 🍁️ 骏注：

1、《商颂》出于宋，我们之前也介绍过，宋、商二字古音同，周封微子启于商丘，建立宋国，以宋之名代商之实，继其文化，祀其先祖。周公平定三监叛乱后处死武庚，纣王之子武庚既死，那么商的传人就非微子启莫属了。微子启，子姓（商始祖“契”是子姓），名启，纣的庶兄。纣王时代，因被封于微国而得名微子启，史上又称宋微子、商微子。周朝特许宋国继嗣殷商，故有《商颂》五篇，皆祭祀殷商先祖的乐歌。

2、本篇《玄鸟》，歌颂商的先祖契、商王成汤和商王武丁（殷高宗）。我们之前曾说过，中国历史上，很多当权者都被后人附会某种天命神授的传说（这种附会对他们的好处很多，比如权力来源的合法性就不容易受到挑战，直到有人喊出“王侯将相宁有种乎？”。但悲哀的是，历代占得江山者，依然炮制不同牌号的“合法性大补丸”喂给民众）。从华胥氏生伏羲女娲始，一直不断。当然这跟远古人民对自然的认知很浅有关，也与群婚制的母系氏族社会“只知其母，不知其父”的习俗有关。所以，“圣人无父，感天而生”的神话故事也多少反映了母系社会的这种遗迹。传说商的始祖“契”就是其母“简狄”误吞鸟蛋怀孕所生。本诗即以这个神话开篇，句曰：天命玄鸟，降而生商。

3、契，帝喾的儿子，挚、尧、后稷的兄弟。尧时代的火正（保管火种的官）、司徒。被封于“商”，死后也葬于“商”（现在的商丘）。商丘现存“阏伯台”就是纪念他的。契的后人“汤”灭夏建立商朝（国），所以称契为商祖。

4、历代有代表性的点评：

《毛诗序》：“《玄鸟》，祀高宗也。高宗，殷王武丁。中宗玄孙之孙也。有雊雉之异，又惧而修德，殷道复兴，故亦表显之，号为高宗云。崩而始合祭于契之庙，歌是诗焉。”

《毛传》：“玄鸟，鳦也。春分玄鸟降，汤之先祖，有娀氏女简狄，配高辛氏帝，帝率与之祈于郊禖而生契，故本其为天所命，以玄鸟至而生焉。”

书香君：鳦，读音yi上声。古人谓燕子。清人程先贞诗《西亭闲坐独酌有述》有句：

柳色绿初成，

絮语来新鳦。

《郑笺》：“降，下也。天使鳦下而生商者，谓鳦遗卵，娀氏之女简狄呑之而生契，为尧司徒有功，封商，尧知其后将兴，又锡其姓焉。自契至汤八迁，始居亳之殷地而受命，国日以广大芒芒然。汤之受命，由契之功，故本其天意。”

《诗集传》：“高辛氏之妃，有娀氏女简狄，祈于郊禖，鳦遗卵，简狄呑之而生契。其后世遂为有商氏，以有天下。事见《史记》。”

《诗经原始》：《玄鸟》，祀高宗也。所谓“孙子”者，武丁也，对汤言，故曰“孙子”。自《吕览》，好为附会，遂以为呑鳦卵而生，《史记》从之，郑氏（指郑玄——书香君注）更本以说经。盖因姜源有履迹生稷之异，故并此亦以为鳦卵所生，是何天之好异，而古圣之生如出一辙哉？

书香君：方玉润虽批评后儒好附会，但对本篇的手法颇为嘉许：诗骨奇秀，神气浑穆，而意亦复隽永，实为三《颂》（指周、鲁、宋颂）压卷。其眉评又曰：“意本寻常，造语特奇。遂使小儒咋舌，惊为怪事，创为无稽妄谈。皆不知诗人语不惊人死不休之过也。”

我个人真的喜欢方玉润的风格。

5、“玄鸟”，黑色鸟。故人认为就是燕子，所谓“鳦”也。传说有娀氏之女简狄（帝喾的妻子之一）与姐妹沐浴湖中，有玄鸟飞来，遗下五彩斑斓的鸟蛋，简狄甚爱，以手玩之，以口含之，误吞而孕，遂生契也。

6、“商”，指商的始祖契。

7、“宅”，居住。“芒芒”，即“茫茫”，广大无边。

《毛传》：芒芒，大貌。

8、“武汤”，成汤，名汤号武。也称武王。

《史记》：“汤曰：吾甚武，号曰武王。”

9、“正”，同“征”，宜读阴平。

10、“方”，广，遍。“方命”，遍告也。

11、“厥”，代词，其。

12、“后”，王也。上古时代称部落首领为“后”。如后羿、后稷。

13、“奄”，拥有。

14、“九有”，九域，九州。有，“域”的借字。传说禹划天下为九州。

15、“先后”，先君，先王。“商之先后，受命不殆，在武丁孙子”这句，应该理解成：商的先君们受天命而不危殆，这样好的国运，到子孙辈的武丁依然如此，依然被上天庇佑。这大概也是“江山万年”的原始版本之一吧，把百姓忽悠得一愣一愣的，这种忽悠术一直流传喔。

16、“殆”，通“怠”，懈怠。

17、“武丁”，殷高宗，汤的九世孙，为商代明君之一。也有人认为，本诗“武王”指武丁。

18、“武王”，即武汤，成汤。不是周武王喔。

19、“胜”，胜任。

《毛传》：“胜，任也”。

20、“旂”，读音qi，阳平。古时一种龙旗。

21、“乘”，应读sheng去声。四马拉车为乘。

22、“糦”，读音chi，去声。同“饎”，酒食。一说通“祭”。

《豳风.七月》里就有“天畯至饎”句，也宜读作chi而非xi。

23、“承”，捧，进献。

24、“止”，停留，居住。

25、“肇域四海”，“肇”，始也。“域”，名词动用，且是意动用法。域四海，就是以四海为域。整句意思就是“开始拥有四海之疆域”。四海，四边。指中国周边蛮夷狄戎之地。跟海洋没有一毛钱关系。

《尔雅》：“九夷、八狄、七戎、六蛮”为“四海”。

我这里提一下，对古文中“四海”的解读很多人是不妥的。

“肇”一说通“兆”，肇域就是兆域、疆域。结合上下文，应该是描述商的发展过程，逐步开疆拓土，从没有，到开始拥有四海之地。应该采信前说。

26、“假”，读音ge，阳平。通“格”，到达。

27、“景”：广大，一说山名，商都所在地，代指殷商土地。“员”，幅员，方圆。“河”，黄河。“景员维河”可以解为：（商地）幅员广大，直到黄河边。或者：黄河流经之地皆在景山周边，都属于殷商国土，殷商广阔的疆域被黄河围绕。

28、“何”，通“荷”，承受，承担。宜读去声。

29、本篇的手法，朱子说是“赋”，其曰：“赋也。此亦祭祀宗庙之乐，而追叙商人之所由生，以及其有天下之初也。”

30、玄鸟生商的传说还被屈子用在《离骚》和《天问》里，但有他自己的用意。

我们看看《离骚》这段：

望瑶台之偃蹇兮，

见有娀之佚女。

吾令鸩为媒兮，

鸩告余以不好。

雄鸠之鸣逝兮，

余犹恶其佻巧。

心犹豫而狐疑兮，

欲自适而不可。

凤皇既受诒兮，

恐高辛之先我。

（诗中“有娀之佚女”就指契的母亲简狄，简狄属有娀氏部落。“高辛”指的是简狄的丈夫帝喾——高辛氏）。

再看看《天问》这两句：

简狄在台，喾何宜？

玄鸟致诒，如何喜？

（这两句我译下：

简狄瑶台去嬉戏，

帝喾对她怎中意？

玄鸟飞来送嘉礼，

简狄为何心欢喜？）

所以，有人说《楚辞》可算是后《诗经》。中国文学样式，紧接《诗经》之后确实就是《楚辞》了。

> 
> 简狄生契为建商，

> 住在殷地广无边。

> 
> 征伐天下安四方。

> 
> 九州疆域尽归商。

>
