情韵欲绝仙骨姗姗

## 🌼 《卫风∘竹竿》原文：
```

籊籊竹竿，以钓于淇（giə）。
岂不尔思（siə），
远莫致之（tjiə）。

```
*书香君👉🏻：* 

> “淇、思、之”，之部，押韵。“之”，此处不是虚词，入韵

```
泉源在左，淇水在右（hiuə）；
女子有行，远兄弟父母（mə）。

```
*书香君👉🏻：* 

> “右、母”，之部，押韵

```
淇水在右，泉源在左（tzai）；
巧笑之瑳（tsai），
佩玉之傩（nai）。

```
*书香君👉🏻：* 

> “左、磋、傩”，歌部，押韵

```
淇水滺滺（jiu），
桧楫松舟（tjiu）。
驾言出游（jiu），
以写我忧（iu）。

```
*书香君👉🏻：* 

> “滺、舟、游、忧”，幽部，押韵

```
本诗四章，每章一韵。

```
## 🍁️ 骏注：

1、这是一首远嫁女的思乡诗。语言凝炼清新，情感真挚深沉。

2、《毛诗序》：“《竹竿》，卫女思归也。适异国而不见答，思而以礼者也。”说的是某女思归源于受冷遇“不见答”。三家诗无异议。

《郑笺》：“小水有流入大水之道，犹妇人有嫁于君子之礼。今水相与为左右而已，亦以喻己不见答。”

书香君归纳序笺及三家诗，其意谓“卫女适异国不见答而犹不恶君子、不违妇礼。伤君子疏远己、己无由致此道，尤蕴于心。其除此尤，维有归耳。”这种解读，老实说，我们今天看不出来也不大认可。

3、朱熹《诗集传》言：卫女嫁于诸侯，思归宁而不可得，故作此诗。首章言思以竹竿钓于淇水，而远不可至也；二章思二水（泉源、淇水——书香君注）之在卫，而自叹其不如也；三章言二水在卫而自恨其不得笑语游戏于其间也；末章与《泉水》末章意同。”

朱熹的解读虽客观些，但其言“嫁于诸侯”，诗中并不能看出。

4、何楷认为《邶风.泉水》（请参我2021年1月22日朋友圈）和本篇都是许穆夫人所作（还有人认为《载驰》也是）。

5、姚际恒认为本篇《竹竿》和《泉水》两诗语多重复非一人笔，疑为媵妾和唱许穆夫人之词。

6、方玉润批评以上二位“均未尝细咏诗辞”。方氏认为《载驰》、《泉水》与《竹竿》，虽皆思卫之作，前者遭乱以思归，后者无端而念旧，词意迥乎不同。不仅不是许穆夫人所作，而且也看不出《毛诗序》所说的“不见答”的意味。似也不必深究作者究为何人，若务求确解则三百篇诗不过一本记事簿，怎能有陶情寄兴之作呢？他说的有一定道理，我们读《诗经》最要紧是欣赏它的文学和史学价值。

今人多认为本诗是卫女嫁于别国、思归不得而深吟惆怅之作。

方玉润高度称美本诗：局度雍容、音节圆畅、风致嫣然，足以擅美一时，是无心求工而妙手天成之作。

7、王先谦特别解释为何本篇四章，每章皆出现“淇水”？女子怀念出嫁前在母国卫国的淇水岸边垂钓的时光；在夫国又驾船游于淇水。何故？盖古之小国数十百里，虽云异国，不离淇水流域也。前三章，淇水是卫国境内之淇水，末章则是异国（丈夫的国）境内之淇水也。

8、籊（tì）籊，竹长而锐貌，一说竹长而细貌。

9、不尔思，不思尔。
尔，应指淇水（卫国境内的那一段，即女子垂钓于斯的河岸），或代指卫国。

10、远，远离；致，到，到达。

11、泉，水名。卫国河流名。即百泉，在卫之西北，东南流入淇水。《毛传》：“泉源，小水之源。淇水，大水也。”《郑笺》：“小水有流入大水之道，犹妇人有嫁于君子之礼。今水相与左右而已，亦以喻己不见答。”

12、“泉源在左，淇水在右”，泉和淇，皆水名。但，为何泉水称泉源而淇水不称淇源？又为何一左一右？胡承珙释：卫都朝歌，淇自其城北屈而西转，亦在卫之西北，其下流乃在西南。而泉有二源，曰马沟曰美沟，皆出朝歌西北。诗人在泉河的源头而言，故曰“在左”，而淇水在诗人此时的眼里已经向下流至卫国西南，故曰“在右”。胡氏称赞诗人：“一字分别，不苟如此”。古人用字之讲究、后世学人解读之深研，值得赞美。但，另外一方面，我们似也不必过于拘泥，比如《小雅》中的《裳裳者华》就有“左之左之，君子宜之。右之右之，君子有之。”这个左和右就并非确指。再如今之歌词：“左手一指太行山，右手一指是吕梁”，左右当为随机运用，人转个方向，左右又变了。还比如朱明瑛唱的《回娘家》“左手一只鸡，右手一只鸭”，难不成右手鸡左手鸭就不行吗？文学作品这样的叙述方式未必就是确指哎。

13、有行，当代多解为“出嫁、远嫁”。但，单独一个字“行”，旧说多解为“道”，原指“官道、大路”，“有行”可引申为“有道”，遵守妇道之谓也。后来也解为“出嫁”。《郑笺》：“行，道也。女子有道当嫁耳，不以不答而违妇礼。”前已述及，清人方玉润就已经分析诗中并不能看出“不见答”之意，我们现在研读也看不出（我个人是看不出的），那么“有行”就未必解为“有道”，还是简单理解为“出嫁”为宜。

14、瑳（cuō），《毛传》：“瑳，巧笑貌”。马瑞辰：“瑳，当为“龇”的借字，笑而见齿，故以龇状之。龇借作瑳，犹玼或作瑳也。”这里指露齿巧笑状。

15、傩（nuó），通“娜”，婀娜。一说行动有节奏的样子。

16、滺（yōu），流貌。“滺滺”河水荡漾之状。鲁诗作“油油”。古本作“攸攸”。《说文》：“攸，水行也”。

17、楫（jí），船桨。

18、桧、松，树名，均可作船木。桧（guì），柏叶松身。暗喻舟楫相配，得水而行；男女相配，得礼而备。

19、驾言：本意是驾车，这里是操舟。言，语助词，无实意。或可依文意解为“而”。

20、写（xiè），通“泻”，消除、宣泄、排解。

21、本诗前三章通过回忆或想象，把女子少时在家乡的垂钓之乐、嬉戏之景以及出嫁途中所见所闻描绘得极其鲜活灵动、如在眼前。拿方玉润的话说，真乃“仙骨姗姗，风韵欲绝。”

## 🍁️ 骏译：

> 青青竹竿细又长，
> 
> 我曾垂钓淇河岸。
> 
> 河水清澈常入梦，
> 
> 路途遥遥难还乡。

> 泉源流淌在南边，
> 
> 淇水荡漾于北端。
> 
> 姑娘长大要出嫁，
> 
> 远离父母别兄长。

> 淇河淙淙右边淌，
> 
> 泉源涓涓流左边。
> 
> 巧笑嫣然齿晶亮，
> 
> 步履摇曳玉叮当。

> 淇河悠悠不尽流，
> 
> 桧桨松舟人依旧。
> 
> 故国难回且乘舟，
> 
> 出游聊解思乡愁。
