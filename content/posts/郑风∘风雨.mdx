风雨凄凄，风雨潇潇
风雨如晦，鸡鸣不已
等成语、习语皆出于此诗。

凄风苦雨，本是阴暗的悲景；久别重逢，又是欢快的喜景。一悲一喜、情境互反的映衬，强化了本诗的感染力。

所谓“以乐景写哀，以哀景写乐，一倍增其哀乐。”

## 🌼 《郑风∘风雨》原文：
```

风雨凄凄（tsyei），
鸡鸣喈喈（kei）。
既见君子，云胡不夷（jiei）？

```
*书香君👉🏻：* 

> “凄、喈、夷”，脂部，押韵

```
风雨潇潇（syu），
鸡鸣胶胶（keu）。
既见君子，云胡不瘳（thiu）？

```
*书香君👉🏻：* 

> “潇、胶、瘳”，幽部，押韵

```
风雨如晦（xuə），
鸡鸣不已（jiə）。
既见君子（tziə），
云胡不喜（xiə）？

```
*书香君👉🏻：* 

> “晦、已、子、喜”，之部，押韵

```
本诗三章，每章一韵。

```
## 🍁️ 骏注：

1、当代多认为这是一首描写夫妻久别重逢而喜出望外之情境的诗。也有解为感慨乱世之中有德君子给人希望和慰藉的诗作。

2、序笺为代表的旧说，认为是思君子诗，把“风雨”之时解为“乱世” 并且加上了“不改其度”这样的“政治正确”话语。

《毛诗序》：“《风雨》，思君子也。乱世则思君子不改其度焉。”

《郑笺》：“鸡犹守时而鸣，喈喈然。不为如晦而止不鸣。喻君子虽居乱世，不变改其节度。思而见之，云何而心不说（悦）？”

传笺将本也可以是妻子敬称、美称其丈夫的“君子”一词，解为品德高尚的君子，以风雨喻乱世，对后世影响深远，以致“风雨如晦”成为经典成语，而“风雨凄凄”、“风雨潇潇”除了描述自然天气之外，也都被赋予了与时局、环境、命运相关的深层意境，“鸡鸣不已”也成为自我勉励或勉励他人的句子。客观地说，序笺对本篇略有附会的解读也并非完全没有道理，并且为汉语增添了几个成语和文学意象呢。

3、朱熹独具慧眼，还是看出了男女之情。不过，他一以贯之地认为是淫奔之诗。《诗集传》：“风雨晦冥，盖淫奔之时。君子，指所期之男子也。淫奔之女当此之时见其所期之人而心悦、积思之病至此而愈也。”

朱老夫子，就不能是夫妻重逢吗？

4、方玉润另辟蹊径，解此诗为“怀友”诗。他认为传序笺所谓风雨喻乱世之说使诗味索然，他认为风雨晦冥独处无聊时最易怀人。书香君认为他的如下解读也有几分道理：

故友良朋，一朝聚会，则尤可以促膝谈心。虽有无限愁怀，郁结莫解，亦皆化尽，如险初夷（方玉润解诗中“云胡不夷”的“夷”为“化险为夷”的“夷”，而《郑笺》解为“悦也”、朱熹解为“平也”，与今人多解为“平复、平静下来”相近——书香君注），如病初瘳，何乐如之！此诗人善于言情，又善于即景以抒怀，故为千秋绝调也。若必以风雨喻乱世，则必待乱世而始思君子，不遇乱世则不足以见君子，义旨非不正大，意趣反觉索然。故此诗不必定指忽、突乱世，凡属怀友，皆可以咏，则意味无穷矣。

他并且批驳朱子之“淫诗说”：深宵风雨，联床话旧，不觉情亲，晓犹未已。此何如友谊耶？而乃以为淫也，岂不怨哉？《集传》指为淫诗，则无良甚矣。使真其淫，似不必待晦翁而始知其为淫矣。

方氏骂朱子无良，并且说，如果真的是淫奔诗，还要等到宋代你老先生手上才看得出来？

*书香君👉🏻：* 

> 客观说，方玉润以为本诗“怀友”是有一定道理，但是，当作爱情诗（即朱子所谓“淫奔诗”）则更加合适，方玉润对朱熹的抨击有点偏激了。

5、要而言之，本诗历代有代表性的解读有三：其一夫妻重逢（今人多采此说）；其二，乱世思明君。其三，怀友。而本诗也产生了象征险恶处境或动荡环境的成语“风雨如晦”以及表述有识之士虽风雨如晦仍坚定不移、奋斗不止的习语“鸡鸣不已”。

6、喈喈，喈读音jie阴平，以前的篇目中也出现过。此处指鸡鸣声。

7、胡，何。

8、潇潇，风雨迅疾貌。《毛传》释：“潇潇，暴疾也。”

9、胶胶，鸡鸣声，与“喈喈”义同。胶，是“嘐”的借字。三家诗正作“嘐”。《毛传》：“胶胶犹喈喈也。”

10、夷，平。指心中平静。《毛传》：“夷，说也”，“说”，就是“悦”。方玉润解读为“化险为夷”之“夷”（可参骏注4）。

11、瘳，读音chou阴平。病愈，此处指心病化解。《毛传》：“瘳，愈也。”

12、晦，夜晚、昏暗。《毛传》：“晦，昏也。”
《说文》释：“晦，月尽也。”
还有说“晦”的读音为yin的，我尚未找到出处。

13、凄风苦雨，本是阴暗的悲景；而久别重逢，又是欢快的喜景。一悲一喜、情境互反的映衬，加强了诗的艺术感染力。所谓“以乐景写哀，以哀景写乐，一倍增其哀乐。”

## 🍁️ 骏译：

> 风声凄凄雨不停，
> 
> 檐下轻轻闻鸡鸣，
> 
> 夫君终于回到家，
> 
> 妾心欢喜得安宁。

> 狂风阵阵雨相交，
> 
> 窗外鸡鸣声声高，
> 
> 风雨之时见到你，
> 
> 我的心病全已消。

> 风大雨急天地昏，
> 
> 鸡鸣之声不停顿，
> 
> 风雨之时见到你，
> 
> 叫我如何不开心。
