勤劳苦熬者位卑而饥寒，
尸位素飧者赤芾而乘轩。



## 🌼 《曹风∘候人》原文：

```
彼候人兮，何戈与祋（tuat）。
彼其之子，三百赤芾（piuat）。
```

*书香君👉🏻：*

> “祋、芾”，月部at，押韵

```
维鹈在梁，不濡其翼（jiək）；
彼其之子，不称其服（biuək）。
```

*书香君👉🏻：*

> “翼、服”，职部ək，押韵

```
维鹈在梁，不濡其咮（tio），
彼其之子，不遂其媾（ko）。
```

*书香君👉🏻：*

> “咮、媾”，侯部o，押韵

```
荟兮蔚兮，南山朝隮（tzyei）；
婉兮娈兮，季女斯饥（kiei）。
```

*书香君👉🏻：*

> “隮、饥”，脂部ei，押韵


## 🍁️ 骏注：

1、《候人》的诗旨，历代分歧不大。基本认为是刺诗。具体是刺“曹共公任用小人”的，还是刺“尸位素飧者不劳而获”的，有些争议。

2、旧说回顾：

《毛诗序》：“《候人》，刺近小人也。共公远君子，而好近小人焉”。

书香君：毛序之所以如是说，可能源于《史记》的如下记载：“余寻曹共公之不用僖负羁，乃乘轩者三百人，知唯德之不建。”

僖负羁是曹国大夫。《史记》讲的是晋文公为与楚国争霸，攻打楚国的同盟者曹国，晋军攻入曹国后晋文公曾指责曹共公不重用贤臣僖负羁——“三月丙午，入曹。数之，以其不用僖负羁而乘轩者三百人也”。文献既有“乘轩者三百人也”，而《候人》又有“彼其之子，三百赤芾”句，似乎坐实了《候人》是刺曹共公的。

《郑笺》进一步解读：“鹈在梁，当濡其翼而不濡者，非其常也。以喻小人在朝，亦非其常。德薄而服尊，虽见任于君，终将薄于君而不能成其德教也。国之无政令，则下民困病。”

《诗集传》：“候人，道路迎送宾客之官。此刺其君远君子而近小人之词。言彼候人而何戈与祋者，宜也。彼其之子，而三百赤芾，何哉？晋文公入曹，数其不用僖负羁，而乘轩者三百人，其谓是欤？荟蔚朝隮，言小人众多而气焰盛也。季女婉鸾自保，不妄从人，而反饥困。言贤者守道而反贫贱也。”

《诗经原始》：“刺曹君远君子而近小人也。”方玉润基本认同朱子之解。

现代学者多认为这是一首对“好人在下层、庸才居高位”的现实进行讥刺的歌诗。全诗四章，每章四句。赋比兴手法皆有。

3、“候人”：春秋时官名。是看守边境、迎送宾客和治理道路、掌管禁令的小官。

《毛传》：“候人，道路送宾客者。”

《诗集传》：“候人，道路迎送宾客之官。”

4、“何”，读去声，通“荷”。扛着。

《毛传》：“何，揭。”

5、“祋”读音dui去声。殳的一种，竹制的长殳，有棱无刃。

《毛传》：“祋，殳也”

6、“其”，读ji去声。语气词。“之子”：那个人，那些人。“彼其之子”，在本诗中有嘲讽、不屑的语气。

《毛传》：”彼，彼曹朝也。”

7、“三百”：指穿“芾”的人有三百个，极言其多，未必确数；一说指“芾”的件数，即有三百件芾服。“芾”，有两个读音fei和fu。目前的词典里作名词用时，多读fu，阳平。指“蔽膝”，一种革制的系于腰间而垂于膝盖的祭祀服饰。作形容词时，多读fei，去声。指“树枝、树叶小而密”，引申义为“繁茂貌”。“芾”在本诗的读音我有自己的看法，此处不表，后文讲本诗的韵脚时会再提一下。本诗中，“芾”作名词用。“赤芾”，赤色的芾。官位高低所穿的“芾”的颜色也不同。着赤芾、乘轩是大夫以上官员的待遇。“赤芾”、“朱芾”逐渐就代指王室、高官、贵族了。如《小雅.车攻》有“赤芾金舄，会同有绎”；《小雅.斯干》有“朱芾斯黄，室家君王”。

《毛传》：“芾，韠（bi去声）也”

8、“维”，发语词，虚词，无实意。

9、“鹈”，读音ti，阳平。即鹈鹕，一种水鸟，食鱼为生。

《毛传》“鹈，洿（读音wu阴平——书香君注）泽鸟也。”

《诗集传》：“鹈，鴮（读音wu阴平——书香君注）泽，水鸟也，俗所谓淘河也。”

书香君：鹈鹕看到有鱼的小泽塘，会群聚，然后用其巨大的嗉囊吸水吐到别处，使水干而捕食鱼，故名淘河。

10、“梁”，鱼梁。捕鱼的堤坝。

《毛传》：“梁，水中之梁。”

11、“濡”，湿润。名词动化，沾湿。“维鹈在梁，不濡其翼”，就是守在鱼梁上的鹈鹕，因为所处的位置好，紧靠水面，都不需要打湿翅膀就能就近叨鱼吃。下文的“不濡其咮”，就是连嘴喙都不用打湿就可以吃到鱼（离水面近，蹦跳的鱼有时会直接跳到鹈鹕的嘴里，形容鹈鹕捕鱼太容易了）。喻示有些人辛劳孩子却吃不饱，而有些人就凭占据的好位置不劳而获。

12、“称”，相称，匹配。

13、“咮”，读音zhou，去声。禽鸟的喙。

《毛传》：“咮，喙也”

14、“遂”：遂意。

《郑笺》：“遂，犹久也。”

《诗集传》：“遂，称。遂之为称，犹今人谓遂意曰称意。”

15，“媾”，交媾。引申指婚姻。“不遂其媾”，指不尽对婚姻的责任、义务。“媾”的字义有很多，既有“婚姻”，也有“宠爱、受宠”的义项。奇怪的是，《说文》解“婚姻”，《康熙字典》解“宠爱”，但在这两个义项下都以《候人》“不遂其媾”句举例。事实上，“媾”，最早的字义是“重婚”，这个“重婚”不是现在的词义，而是“亲上加亲”、重叠交互为婚。引申为“宠”、“厚”，是有道理的。《孔疏》即言：“重昏媾者，情必深厚”。

另外，诗文中“鹈鹕不濡其翼、不濡其咮”说的是捕鱼、食鱼的事，而“鱼”在上古含有与“性”、“婚姻”有关的意象。

《毛传》：“媾，厚也”

《诗集传》：“媾，宠也。”

《毛诗传笺通释》：“不遂其媾”，与“不称其服”同类意思，媾，是“韝”的借字，《说文》：韝，臂衣也。是射弦所佩戴的装备。遂，是拉弓所用的指佩。不遂其媾，就是佩戴了射箭的相关用具却不去射箭（是不合适的），跟“不称其服”同义。

而《郑笺》对“不遂其媾”的解释是：遂，犹“久”也；媾，犹“厚”也。不遂其媾，即不久其厚，言小人对君王不可能一直很忠厚，终将薄于君也。

16、“荟”、“蔚”：云雾聚集迷漫。如“云蒸霞蔚”

《毛传》：“荟蔚，云兴貌。

17、“隮”，读音ji，阴平。通“跻”，升，登。

《毛传》：“隮，升云也。”

《郑笺》：“荟蔚之小云，朝升于南山，不能为大雨。以喻小人虽见任于君，终不能成其德教。”

18、“婉”、“娈”，柔顺美好貌。

《毛传》：“婉，少貌。娈，好貌。”

19、“季女”，少女。本诗指“候人”的幼女。

《毛传》：“季，人之少子也。女，民之弱者。”

《郑笺》：“天无大雨，则岁不熟而幼弱者饥，犹国之无政令，则下民困病。”

20、“斯”：虚词、语助词。

21、本诗的韵脚：

首章“祋、芾”押韵。“祋”古读tuat，“芾”古读piuat，皆月部，押腹韵uat。那么，问题来了，读今音如何处理才押韵？“祋”今音dui，前面我已说了，“芾”今音有fu和fei，按词性，本诗的“芾”是名词，应读fu。但，为了与“祋dui”押韵，我建议“芾”读fei。

二章，“翼，服”，古读分别是jiək，biuək。同属职部，押韵。而今音则不押韵。

三章，“咮，媾”，押韵。“咮”古读tio，“媾”古读ko。皆侯部，押韵。今音（zhou，gou）也押韵。

四章，“隮，饥”押韵。“隮”古读tzyei，“饥”古读kiei。皆脂部，押韵。今音两字皆读ji，也押韵。
